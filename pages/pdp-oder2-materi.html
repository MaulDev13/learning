Persamaan Diferensial Parsial Orde 2

PDP orde 2 adalah persamaan yang melibatkan turunan parsial hingga orde dua. Bentuk umumnya: 

A u_xx + B u_xy + C u_yy = 0

Dimana:
A, B, C = konstanta
u_xx = turunan kedua terhadap x
u_yy = turunan kedua terhadap y
u_xy = turunan campuran

Klasifikasi PDP Orde 2

Klasifikasi ditentukan dari diskriminan: 

D = B² - 4AC

1. Elliptic

D < 0

Contoh:

u_xx + u_yy = 0

Menggambarkan kondisi keseimbangan (steady state), seperti distribusi suhu stabil.

2. Parabolic

D = 0

Contoh:

u_t = α u_xx

Menggambarkan proses difusi atau penyebaran, seperti panas.

3. Hyperbolic

D > 0

Contoh:

u_tt = c² u_xx

Menggambarkan gelombang atau propagasi sinyal.

Kesimpulan:
Elliptic → statis (keseimbangan)
Parabolic → difusi (menyebar)
Hyperbolic → gelombang (bergerak)

Klasifikasi PDP Orde 2 + Intuisi + Contoh

1. Bentuk Umum

A u_xx + B u_xy + C u_yy = 0

Dimana:

  • A → koefisien u_xx
  • B → koefisien u_xy
  • C → koefisien u_yy

2. Diskriminan

D = B² - 4AC
        
D < 0 → Elliptic
D = 0 → Parabolic
D > 0 → Hyperbolic

Penting: nilai ini hanya untuk klasifikasi jenis persamaan, bukan hasil solusi.

3. Pendekatan Intuisi

Elliptic → Keseimbangan

u_xx + u_yy = 0

Tidak ada waktu → sistem sudah stabil.

Contoh: distribusi suhu yang sudah tidak berubah.

Ciri: nilai di suatu titik dipengaruhi lingkungan sekitar.

Parabolic → Difusi (Menyebar)

u_t = α u_xx

Ada waktu → perubahan perlahan menuju stabil.

Contoh: panas menyebar.

Hyperbolic → Gelombang

u_tt = c² u_xx

Ada percepatan → terjadi gerakan bolak-balik.

Contoh: getaran, gelombang.

4. Contoh + Hitung A, B, C, D

Elliptic

u_xx + u_yy = 0

A = 1
B = 0
C = 1

D = 0² - 4(1)(1) = -4 < 0 → Elliptic

Solusi sederhana:

u(x,y) = x² - y²

u_xx = 2
u_yy = -2

2 + (-2) = 0 ✔

Parabolic

u_xx + 2u_xy + u_yy = 0

A = 1
B = 2
C = 1

D = 2² - 4(1)(1) = 0 → Parabolic

Contoh solusi:

u(x,y) = f(x - y)

(bergantung bentuk fungsi)

Hyperbolic

u_xx - u_yy = 0

A = 1
B = 0
C = -1

D = 0² - 4(1)(-1) = 4 > 0 → Hyperbolic

Solusi sederhana:

u(x,t) = x + t

u_xx = 0
u_tt = 0

0 = 0 ✔

Contoh gelombang:

u(x,t) = sin(x - t)

5. Ringkasan Cepat

Elliptic   → diam / steady state
Parabolic  → menyebar / difusi
Hyperbolic → gelombang / propagasi
  • Tidak ada waktu → elliptic
  • Ada waktu & menyebar → parabolic
  • Ada percepatan → hyperbolic

Transformasi PDP Orde 2 (Versi Sederhana)

Ide utama dari bagian ini adalah: kita mengubah variabel x, y menjadi variabel baru ξ (xi) dan η (eta) supaya persamaan lebih sederhana. 

Awal:
A u_xx + B u_xy + C u_yy

Setelah transformasi (ganti koordinat): 

Menjadi:
A* w_ξξ + B* w_ξη + C* w_ηη

A*, B*, C* adalah versi “baru” dari A, B, C setelah sistem koordinat diganti.

Contoh Super Sederhana (Biar Kebayang)

Misalkan kita punya fungsi: 

u(x,y) = x² + y²

Kita ubah variabel menjadi: 

ξ = x + y
η = x - y

Sekarang kita lihat “arti perubahan arah”: 

Jika x naik → ξ naik & η naik
Jika y naik → ξ naik & η turun

Kenapa muncul A*, B*, C*?

Karena turunan juga ikut berubah saat variabel diganti.

Contoh ide sederhana:

u_x = perubahan u terhadap x
u_y = perubahan u terhadap y

Setelah diganti variabel: 

w_ξ = perubahan u terhadap ξ
w_η = perubahan u terhadap η

Contoh Intuisi (Tanpa Rumus Berat)

Bayangkan kamu berjalan di peta: 

x = arah timur
y = arah utara

Lalu kamu ganti cara lihat: 

ξ = arah diagonal kanan atas (↗)
η = arah diagonal kiri atas (↖)

Maka:

  • Persamaan yang awalnya rumit di x-y
  • Bisa jadi lebih sederhana di ξ-η
