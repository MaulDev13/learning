Sistem Persamaan Linear (SPL)

Sistem Persamaan Linear (SPL) dengan n persamaan dan n variabel dapat dinyatakan dalam bentuk umum, kemudian direpresentasikan ke dalam bentuk matriks untuk mempermudah analisis dan perhitungan.

Bentuk umum SPL:

a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ = b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ = b₂
...
aₙ₁x₁ + aₙ₂x₂ + ... + aₙₙxₙ = bₙ

Dapat ditulis dalam bentuk matriks: Ax = b

Topik: Metode Numerik

Fokus: Representasi Matriks & Metode Penyelesaian

Representasi Matriks

A adalah matriks koefisien berukuran n × n, x adalah vektor variabel, dan b adalah vektor konstanta.

Bentuk ini memungkinkan penggunaan metode numerik untuk menyelesaikan SPL secara sistematis.

Dekomposisi LU

Matriks A diuraikan menjadi hasil perkalian dua matriks: A = LU

L adalah matriks segitiga bawah, sedangkan U adalah matriks segitiga atas.

Penyelesaian dilakukan melalui dua tahap:

Ld = b → menggunakan substitusi maju
Ux = d → menggunakan substitusi mundur

Terdapat dua pendekatan umum:

• Crout: diagonal U bernilai 1
• Doolittle: diagonal L bernilai 1

Ilustrasi Penyelesaian LU

Misalkan diberikan sistem:

3x₁ + 4x₂ + 2x₃ = 16
x₁ + 3x₂ + x₃ = 8
2x₁ + 5x₂ + x₃ = 12

Langkah penyelesaian:

• Bentuk matriks A
• Lakukan dekomposisi menjadi L dan U
• Selesaikan Ld = b
• Selesaikan Ux = d

Hasil akhir diperoleh:

x₁ = 2, x₂ = -1, x₃ = 3

Metode Iteratif (Jacobi)

Metode Jacobi merupakan metode numerik berbasis iterasi, di mana nilai variabel diperbarui secara berulang hingga mencapai konvergensi.

Bentuk umum iterasi:

xᵢ(k+1) = (1 / aᵢᵢ) [ bᵢ − Σ aᵢⱼ xⱼ(k) ]

Proses dimulai dari tebakan awal, kemudian dihitung secara berulang sampai perubahan nilai sangat kecil (error minimum).

Contoh Iterasi Jacobi

8x₁ + x₂ − x₃ = 8
2x₁ + x₂ + 9x₃ = 12
x₁ − 7x₂ + 2x₃ = -4

Dengan tebakan awal: x⁰ = (0, 0, 0)

Setelah beberapa iterasi, solusi mendekati:

x₁ = 1, x₂ = 1, x₃ = 1

Hasil yang Mungkin dari Metode Jacobi

Metode Jacobi tidak selalu menghasilkan solusi yang sama dalam semua kasus. Hasil iterasi sangat bergantung pada sifat matriks koefisien dan nilai awal yang digunakan.

1. Konvergen ke Solusi Unik

Ini adalah kondisi ideal, di mana iterasi mendekati satu solusi tertentu.

Ciri:

• Nilai stabil menuju satu titik
• Error semakin kecil
• Solusi mendekati nilai eksak

Contoh sebelumnya menghasilkan: x = (1, 1, 1)

2. Divergen (Tidak Menuju Solusi)

Iterasi tidak mendekati solusi, bahkan bisa semakin besar atau tidak stabil.

Ciri:

• Nilai membesar (tak hingga)
• Berosilasi tanpa pola
• Tidak stabil

Biasanya terjadi jika matriks tidak dominan diagonal.

3. Konvergen Lambat

Metode tetap menuju solusi, tetapi membutuhkan banyak iterasi.

Ciri:

• Perubahan sangat kecil setiap iterasi
• Membutuhkan banyak langkah
• Kurang efisien

4. Sensitif terhadap Nilai Awal

Nilai awal dapat mempengaruhi kecepatan konvergensi.

• Tebakan awal dekat solusi → cepat konvergen
• Tebakan jauh → bisa lambat atau bahkan gagal

Kesimpulan

Metode Jacobi dapat menghasilkan:

• Solusi yang benar (konvergen)
• Tidak menghasilkan solusi (divergen)
• Solusi dengan proses lambat

Oleh karena itu, sebelum menggunakan metode ini, perlu diperiksa apakah sistem memenuhi syarat konvergensi.

Konsep Lanjutan yang Perlu Dipahami

Untuk memahami Sistem Persamaan Linear secara lebih mendalam, tidak cukup hanya mengetahui langkah penyelesaian. Diperlukan pemahaman terhadap sifat sistem dan stabilitas metode yang digunakan.

Kondisi Keunikan Solusi

SPL memiliki solusi unik jika matriks A bersifat invertible.

Hal ini terjadi jika:

• Determinan A ≠ 0
• Rank matriks penuh

Kondisi Singular

Jika determinan A = 0, maka sistem dapat:

• Tidak memiliki solusi
• Memiliki tak hingga solusi

Stabilitas Numerik

Dalam komputasi, kesalahan pembulatan dapat mempengaruhi hasil.

Sistem yang buruk dapat menyebabkan error membesar selama perhitungan.

Pivoting (pada LU)

Pivoting digunakan untuk meningkatkan stabilitas numerik.

• Partial pivoting: menukar baris
• Full pivoting: menukar baris dan kolom

Spectral Radius

Pada metode iteratif, konvergensi ditentukan oleh spectral radius (ρ).

Metode konvergen jika:

ρ < 1

Condition Number

Mengukur sensitivitas solusi terhadap perubahan kecil pada data.

• Nilai kecil → stabil
• Nilai besar → sensitif (ill-conditioned)

Perbandingan Metode Langsung vs Iteratif

• Metode langsung (LU): solusi dalam langkah terbatas
• Metode iteratif (Jacobi): pendekatan bertahap

Pemilihan metode bergantung pada ukuran sistem dan kebutuhan komputasi.

Kesimpulan

Pemahaman lanjutan mencakup:

• Keberadaan solusi
• Stabilitas perhitungan
• Perilaku metode iteratif
• Sensitivitas terhadap error

Pengembangan Materi Sistem Persamaan Linear

Selain metode penyelesaian, terdapat beberapa konsep penting yang perlu dipahami agar pemahaman terhadap Sistem Persamaan Linear menjadi lebih lengkap dan tidak hanya bersifat prosedural.

Syarat Konvergensi Metode Iteratif

Tidak semua sistem dapat diselesaikan dengan metode Jacobi atau metode iteratif lainnya.

Salah satu syarat penting adalah dominasi diagonal, yaitu:

|aᵢᵢ| ≥ jumlah |aᵢⱼ| untuk j ≠ i

Jika syarat ini tidak terpenuhi, metode bisa divergen.

Analisis Error dan Toleransi

Dalam metode numerik, solusi tidak selalu eksak sehingga diperlukan ukuran kesalahan (error).

Iterasi dihentikan jika:

• |xᵏ⁺¹ - xᵏ| kecil (toleransi terpenuhi)
• Jumlah iterasi maksimum tercapai

Perbandingan Metode

Setiap metode memiliki kelebihan dan kekurangan:

• LU: akurat, cocok untuk solusi langsung
• Jacobi: sederhana, tetapi bisa lambat
• Gauss-Seidel: lebih cepat dari Jacobi

Gauss-Seidel (Pelengkap Penting)

Mirip dengan Jacobi, tetapi menggunakan nilai terbaru langsung dalam iterasi.

Biasanya lebih cepat konvergen dibanding Jacobi.

Interpretasi Geometris

SPL dapat diinterpretasikan sebagai perpotongan garis (2D) atau bidang (3D).

• Satu titik → solusi unik
• Tidak berpotongan → tidak ada solusi
• Berimpit → tak hingga solusi

Implementasi Komputasi

Metode numerik biasanya digunakan dalam komputasi:

• Program (Python, MATLAB, dll)
• Simulasi teknik
• Machine learning dan optimasi

Kesimpulan

Pemahaman SPL tidak hanya berhenti pada perhitungan, tetapi juga mencakup:

• Kapan metode bisa digunakan
• Seberapa akurat hasilnya
• Bagaimana perilaku iterasinya
