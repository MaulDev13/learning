Solusi: Klasifikasi PDP Orde Dua

Identifikasi Koefisien

Diberikan persamaan:

2 ∂²u/∂x² + 4 ∂²u/∂x∂y + 3 ∂²u/∂y² = 0

Bandingkan dengan bentuk umum:

A ∂²u/∂x² + B ∂²u/∂x∂y + C ∂²u/∂y² = 0

Diperoleh nilai koefisien:

Koefisien Nilai
A 2
B 4
C 3

Menghitung Diskriminan

Gunakan rumus diskriminan:

D = B² - 4AC

Substitusi nilai koefisien:

D = (4)² - 4(2)(3) D = 16 - 24 D = -8

Menentukan Klasifikasi

Karena nilai diskriminan:

D < 0

Maka persamaan termasuk dalam kategori PDP elliptik.

Nilai Diskriminan Jenis PDP
D = -8 Elliptik

Kesimpulan

Persamaan 2 ∂²u/∂x² + 4 ∂²u/∂x∂y + 3 ∂²u/∂y² = 0 merupakan Persamaan Diferensial Parsial tipe elliptik karena memiliki diskriminan negatif.
