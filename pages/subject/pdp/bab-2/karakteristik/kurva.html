Kurva Karakteristik PDP Orde Satu

Dalam penyelesaian Persamaan Diferensial Parsial orde satu, terdapat konsep penting yang disebut kurva karakteristik.

Kurva karakteristik adalah kurva-kurva tertentu di dalam domain persamaan yang memiliki sifat khusus berkaitan dengan perilaku solusi PDP.

Ide Dasar

Ide dasar dari metode karakteristik adalah bahwa sepanjang kurva karakteristik, suatu PDP orde satu dapat direduksi menjadi persamaan diferensial biasa.

Dengan kata lain, masalah mencari solusi suatu PDP dalam domain dua dimensi disederhanakan menjadi masalah menyelesaikan persamaan diferensial biasa sepanjang kurva-kurva tertentu.

Penurunan Persamaan Karakteristik

Perhatikan PDP kuasi-linear orde satu:

a(x, y, u) ∂u/∂x + b(x, y, u) ∂u/∂y = c(x, y, u)

Misalkan u = u(x, y) adalah solusi PDP tersebut, dan terdapat kurva parametrik di bidang xy yang dinyatakan sebagai:

x = x(t), y = y(t)

Nilai u sepanjang kurva tersebut adalah:

u(t) = u(x(t), y(t))

Turunan u terhadap t sepanjang kurva adalah:

du/dt = ∂u/∂x · dx/dt + ∂u/∂y · dy/dt

Apabila dipilih dx/dt = a dan dy/dt = b, maka persamaan di atas menjadi:

du/dt = a · ∂u/∂x + b · ∂u/∂y = c

sehingga PDP tereduksi menjadi persamaan diferensial biasa du/dt = c.

Persamaan Karakteristik

Berdasarkan penurunan di atas, persamaan karakteristik dari PDP:

a(x, y, u) ∂u/∂x + b(x, y, u) ∂u/∂y = c(x, y, u)

adalah sistem persamaan diferensial biasa:

dx/dt = a(x, y, u)
dy/dt = b(x, y, u)
du/dt = c(x, y, u)

atau dalam bentuk simetris:

dx/a = dy/b = du/c

Kurva yang memenuhi sistem ini disebut kurva karakteristik dari PDP tersebut.

Sifat Kurva Karakteristik

Pertama, sepanjang kurva karakteristik, nilai u berubah mengikuti persamaan diferensial biasa du/dt = c, bukan lagi persamaan parsial.

Kedua, informasi atau kondisi awal merambat sepanjang kurva karakteristik. Nilai u di suatu titik ditentukan oleh nilai u di titik awal pada kurva karakteristik yang melewatinya.

Ketiga, apabila dua kurva karakteristik tidak berpotongan, maka solusi PDP umumnya terdefinisi dengan baik.

Proyeksi Karakteristik

Kurva karakteristik sejatinya adalah kurva dalam ruang tiga dimensi (x, y, u).

Proyeksi kurva karakteristik ke bidang xy disebut karakteristik dasar atau kurva basis karakteristik.

Proyeksi ini diperoleh dari dua persamaan pertama dalam sistem karakteristik:

dx/dt = a(x, y, u)
dy/dt = b(x, y, u)

atau ekuivalen:

dy/dx = b(x, y, u) / a(x, y, u)

Integral Pertama

Penyelesaian sistem persamaan karakteristik menghasilkan dua integral pertama, yaitu fungsi φ₁ dan φ₂ yang bernilai konstan sepanjang setiap kurva karakteristik.

φ₁(x, y, u) = C₁
φ₂(x, y, u) = C₂

Solusi umum PDP kemudian dinyatakan sebagai hubungan antara kedua integral pertama tersebut:

F(φ₁, φ₂) = 0

atau ekuivalen:

φ₁ = G(φ₂)

dengan F dan G adalah fungsi sembarang.

Kondisi Awal dan Kurva Karakteristik

Apabila diberikan kondisi awal sepanjang suatu kurva Γ di bidang xy, maka nilai u pada titik sembarang diperoleh dengan menelusuri kurva karakteristik yang berawal dari Γ hingga mencapai titik tersebut.

Syarat agar masalah nilai awal terdefinisi dengan baik adalah kurva Γ tidak boleh bersinggungan dengan kurva karakteristik.

Apabila Γ bersinggungan dengan kurva karakteristik, maka masalah mungkin tidak memiliki solusi unik atau tidak memiliki solusi sama sekali.

Penyelesaian — Contoh 1

Tentukan kurva karakteristik dan solusi umum dari persamaan berikut.

2 ∂u/∂x + 3 ∂u/∂y = 0

Langkah 1: Identifikasi koefisien.

a = 2, b = 3, c = 0.

Langkah 2: Tulis persamaan karakteristik.

dx/2 = dy/3 = du/0

Langkah 3: Selesaikan persamaan karakteristik.

Dari du/0, diperoleh du = 0, sehingga u = C₁.

Dari dx/2 = dy/3, diperoleh:

3 dx = 2 dy
3x − 2y = C₂

Langkah 4: Tentukan kurva karakteristik.

Kurva karakteristik di bidang xy adalah garis-garis lurus:

3x − 2y = C

Langkah 5: Tulis solusi umum.

Integral pertama yang diperoleh adalah u = C₁ dan 3x − 2y = C₂.

Kesimpulan: Solusi umum persamaan ini adalah:

u = f(3x − 2y)

dengan f adalah fungsi sembarang. Nilai u konstan sepanjang setiap garis 3x − 2y = C.

Penyelesaian — Contoh 2

Tentukan kurva karakteristik dan solusi umum dari persamaan berikut.

y ∂u/∂x − x ∂u/∂y = 0

Langkah 1: Identifikasi koefisien.

a = y, b = −x, c = 0.

Langkah 2: Tulis persamaan karakteristik.

dx/y = dy/(−x) = du/0

Langkah 3: Selesaikan persamaan karakteristik.

Dari du/0, diperoleh du = 0, sehingga u = C₁.

Dari dx/y = dy/(−x), diperoleh:

−x dx = y dy
−x²/2 = y²/2 + konstanta
x² + y² = C₂

Langkah 4: Tentukan kurva karakteristik.

Kurva karakteristik di bidang xy adalah lingkaran-lingkaran berpusat di origin:

x² + y² = C

Langkah 5: Tulis solusi umum.

Integral pertama yang diperoleh adalah u = C₁ dan x² + y² = C₂.

Kesimpulan: Solusi umum persamaan ini adalah:

u = f(x² + y²)

dengan f adalah fungsi sembarang. Nilai u konstan sepanjang setiap lingkaran x² + y² = C.

Penyelesaian — Contoh 3

Tentukan kurva karakteristik dan solusi dari masalah nilai awal berikut.

∂u/∂x + 2 ∂u/∂y = 0
u(x, 0) = sin(x)

Langkah 1: Identifikasi koefisien.

a = 1, b = 2, c = 0.

Langkah 2: Tulis persamaan karakteristik.

dx/1 = dy/2 = du/0

Langkah 3: Selesaikan persamaan karakteristik.

Dari du/0, diperoleh u = C₁.

Dari dx/1 = dy/2, diperoleh:

2 dx = dy
y − 2x = C₂

Langkah 4: Tentukan kurva karakteristik.

Kurva karakteristik di bidang xy adalah garis-garis lurus:

y − 2x = C

Langkah 5: Tulis solusi umum.

u = f(y − 2x)

Langkah 6: Terapkan kondisi awal u(x, 0) = sin(x).

Substitusi y = 0 ke solusi umum:

u(x, 0) = f(0 − 2x) = f(−2x) = sin(x)

Misalkan s = −2x, maka x = −s/2, sehingga:

f(s) = sin(−s/2) = −sin(s/2)

Langkah 7: Substitusi kembali ke solusi umum.

Kesimpulan: Solusi khusus persamaan ini adalah:

u(x, y) = −sin((y − 2x)/2)

Nilai u merambat konstan sepanjang setiap garis y − 2x = C, dengan nilai awal diberikan pada sumbu y = 0.

Kesimpulan

Kurva karakteristik adalah kurva-kurva di dalam domain PDP orde satu yang mereduksi PDP menjadi persamaan diferensial biasa.

Persamaan karakteristik diperoleh dari:

dx/a = dy/b = du/c

Penyelesaian sistem ini menghasilkan dua integral pertama yang menentukan solusi umum PDP.

Apabila kondisi awal diberikan, fungsi sembarang dalam solusi umum dapat ditentukan secara khusus sehingga diperoleh solusi tunggal.
