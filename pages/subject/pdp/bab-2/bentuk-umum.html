Persamaan Diferensial Parsial Orde Satu

Persamaan Diferensial Parsial (PDP) orde satu adalah PDP yang turunan parsial tertingginya berorde satu.

PDP orde satu merupakan bentuk PDP yang paling sederhana dan menjadi dasar dalam memahami PDP orde yang lebih tinggi.

Bentuk Umum

PDP orde satu dengan dua variabel bebas x dan y dapat ditulis dalam bentuk umum:

F(x, y, u, ∂u/∂x, ∂u/∂y) = 0

dengan u = u(x, y) adalah fungsi yang akan ditentukan, dan F adalah fungsi yang diketahui.

Bentuk Linear PDP Orde Satu

PDP orde satu dikatakan linear apabila u dan turunan parsialnya muncul secara linear dalam persamaan.

Bentuk umum PDP linear orde satu dengan dua variabel bebas adalah:

a(x, y) ∂u/∂x + b(x, y) ∂u/∂y = c(x, y) u + d(x, y)

dengan a, b, c, dan d adalah fungsi yang hanya bergantung pada x dan y.

Bentuk Kuasi-Linear PDP Orde Satu

PDP orde satu dikatakan kuasi-linear apabila linear terhadap turunan parsialnya, namun koefisiennya boleh bergantung pada u.

Bentuk umum PDP kuasi-linear orde satu adalah:

a(x, y, u) ∂u/∂x + b(x, y, u) ∂u/∂y = c(x, y, u)

dengan a, b, dan c adalah fungsi yang dapat bergantung pada x, y, dan u.

Contoh PDP Linear Orde Satu

∂u/∂x + ∂u/∂y = 0

Koefisien turunan parsial bernilai satu dan tidak bergantung pada u. Persamaan ini linear homogen.

2 ∂u/∂x − 3 ∂u/∂y = u

Koefisien turunan parsial adalah konstanta dan tidak bergantung pada u. Persamaan ini linear homogen.

x ∂u/∂x + y ∂u/∂y = x + y

Koefisien turunan parsial bergantung pada variabel bebas x dan y saja. Persamaan ini linear tak homogen.

Contoh PDP Kuasi-Linear Orde Satu

u ∂u/∂x + ∂u/∂y = 0

Koefisien ∂u/∂x bergantung pada u. Turunan parsial muncul secara linear, sehingga persamaan ini kuasi-linear.

(u + x) ∂u/∂x + y ∂u/∂y = u²

Koefisien ∂u/∂x bergantung pada u dan x. Persamaan ini kuasi-linear.

Contoh PDP Tak Linear Orde Satu

(∂u/∂x)² + (∂u/∂y)² = 1

Turunan parsial berpangkat dua. Persamaan ini tak linear. Persamaan ini dikenal sebagai persamaan Eikonal.

∂u/∂x · ∂u/∂y = u

Terdapat perkalian antara sesama turunan parsial. Persamaan ini tak linear.

Metode Karakteristik

Salah satu metode utama dalam menyelesaikan PDP orde satu adalah metode karakteristik.

Ide dasar metode ini adalah mengubah PDP menjadi sistem persamaan diferensial biasa sepanjang kurva-kurva tertentu yang disebut kurva karakteristik.

Untuk PDP kuasi-linear orde satu:

a(x, y, u) ∂u/∂x + b(x, y, u) ∂u/∂y = c(x, y, u)

Persamaan karakteristiknya adalah:

dx/a = dy/b = du/c

Langkah Penyelesaian Metode Karakteristik

Langkah pertama, tuliskan persamaan dalam bentuk kuasi-linear:

a ∂u/∂x + b ∂u/∂y = c

Langkah kedua, bentuk persamaan karakteristik:

dx/a = dy/b = du/c

Langkah ketiga, selesaikan sistem persamaan diferensial biasa tersebut untuk mendapatkan dua integral pertama, yaitu φ₁(x, y, u) = C₁ dan φ₂(x, y, u) = C₂.

Langkah keempat, solusi umum PDP dinyatakan sebagai:

F(φ₁, φ₂) = 0

atau ekuivalen:

φ₁ = G(φ₂)

dengan F dan G adalah fungsi sembarang.

Penyelesaian — Contoh 1

Tentukan solusi umum dari persamaan berikut.

∂u/∂x + ∂u/∂y = 0

Langkah 1: Identifikasi koefisien persamaan.

a = 1, b = 1, c = 0.

Langkah 2: Bentuk persamaan karakteristik.

dx/1 = dy/1 = du/0

Langkah 3: Selesaikan persamaan karakteristik.

Dari du/0, diperoleh du = 0, sehingga u = C₁.

Dari dx/1 = dy/1, diperoleh dy = dx, sehingga y − x = C₂.

Langkah 4: Tulis solusi umum.

Dua integral pertama yang diperoleh adalah u = C₁ dan y − x = C₂.

Kesimpulan: Solusi umum persamaan ini adalah:

u = f(y − x)

dengan f adalah fungsi sembarang.

Penyelesaian — Contoh 2

Tentukan solusi umum dari persamaan berikut.

x ∂u/∂x + y ∂u/∂y = u

Langkah 1: Identifikasi koefisien persamaan.

a = x, b = y, c = u.

Langkah 2: Bentuk persamaan karakteristik.

dx/x = dy/y = du/u

Langkah 3: Selesaikan persamaan karakteristik.

Dari dx/x = dy/y, diperoleh:

ln|x| = ln|y| + konstanta, sehingga y/x = C₁.

Dari dx/x = du/u, diperoleh:

ln|x| = ln|u| + konstanta, sehingga u/x = C₂.

Langkah 4: Tulis solusi umum.

Dua integral pertama yang diperoleh adalah y/x = C₁ dan u/x = C₂.

Kesimpulan: Solusi umum persamaan ini adalah:

u/x = f(y/x)

atau ekuivalen:

u = x f(y/x)

dengan f adalah fungsi sembarang.

Penyelesaian — Contoh 3

Tentukan solusi umum dari persamaan berikut.

u ∂u/∂x + ∂u/∂y = 0

Langkah 1: Identifikasi koefisien persamaan.

a = u, b = 1, c = 0.

Langkah 2: Bentuk persamaan karakteristik.

dx/u = dy/1 = du/0

Langkah 3: Selesaikan persamaan karakteristik.

Dari du/0, diperoleh du = 0, sehingga u = C₁.

Dari dx/u = dy/1 dan u = C₁, diperoleh:

dx = u dy = C₁ dy, sehingga x − u y = C₂, atau x − C₁ y = C₂.

Langkah 4: Tulis solusi umum.

Dua integral pertama yang diperoleh adalah u = C₁ dan x − u y = C₂.

Kesimpulan: Solusi umum persamaan ini adalah:

u = f(x − u y)

dengan f adalah fungsi sembarang. Persamaan ini merupakan persamaan implisit terhadap u.

Kesimpulan

PDP orde satu adalah PDP yang turunan parsial tertingginya berorde satu, dan dapat diklasifikasikan menjadi linear, kuasi-linear, dan tak linear.

Metode karakteristik merupakan metode utama dalam menyelesaikan PDP orde satu, dengan cara mengubah PDP menjadi sistem persamaan diferensial biasa sepanjang kurva karakteristik.

Solusi umum PDP orde satu memuat fungsi sembarang sebagai gantinya konstanta sembarang pada PDB, sehingga keluarga solusinya jauh lebih luas.
