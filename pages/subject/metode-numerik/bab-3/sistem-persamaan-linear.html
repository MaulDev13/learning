Pengenalan Sistem Persamaan Linear

Sistem Persamaan Linear (SPL) adalah kumpulan dua atau lebih persamaan linear yang melibatkan variabel yang sama.

Tujuan utama adalah mencari nilai variabel yang memenuhi semua persamaan secara bersamaan.

Bentuk Umum SPL

a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ = b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ = b₂
...
aₙ₁x₁ + aₙ₂x₂ + ... + aₙₙxₙ = bₙ

dimana:

a → koefisien
x → variabel
b → konstanta

Bentuk Matriks

SPL dapat ditulis lebih ringkas sebagai:

Ax = b

dimana:

A → matriks koefisien
x → vektor variabel
b → vektor konstanta

Contoh SPL

2x + y = 5
x − y = 1

Dalam bentuk matriks:

[ 2 1 ] [ x ] = [ 5 ]
[ 1 -1 ] [ y ] [ 1 ]

Jenis Solusi SPL

• Solusi tunggal → satu solusi unik
• Tak hingga solusi → banyak solusi
• Tidak ada solusi → sistem tidak konsisten

Interpretasi Geometri

SPL dapat diinterpretasikan sebagai perpotongan beberapa garis atau bidang.

• 2 variabel → garis di bidang
• 3 variabel → bidang di ruang

Solusi adalah titik perpotongan tersebut.

Permasalahan dalam SPL

Untuk sistem kecil, solusi dapat dicari secara langsung.

Namun, untuk sistem besar:

• Perhitungan menjadi kompleks
• Sulit dilakukan secara manual
• Membutuhkan metode numerik

Peran Metode Numerik

Metode numerik digunakan untuk menyelesaikan SPL secara efisien.

Terdapat dua pendekatan utama:

• Metode langsung (Direct Method)
• Metode iteratif (Iterative Method)

Hubungan dengan Bab Sebelumnya

Sama seperti pencarian akar, penyelesaian SPL juga melibatkan aproksimasi.

Konsep galat dan toleransi tetap digunakan dalam metode iteratif.

Intuisi

SPL adalah cara untuk menemukan titik yang memenuhi banyak kondisi sekaligus.

Semakin banyak persamaan, semakin ketat syarat yang harus dipenuhi.

Kesimpulan

Sistem Persamaan Linear adalah dasar dalam banyak permasalahan matematika dan teknik.

Untuk sistem besar, metode numerik menjadi alat utama dalam mencari solusi secara efisien.
