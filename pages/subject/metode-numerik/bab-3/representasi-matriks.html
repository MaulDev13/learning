Representasi Matriks dan Operasinya

Sistem Persamaan Linear (SPL) dapat ditulis dalam bentuk matriks untuk mempermudah manipulasi.

Bentuk umum: Ax = b

Dengan bentuk ini, kita dapat menggunakan berbagai operasi matriks untuk menyelesaikan sistem.

Contoh Representasi

2x₁ + x₂ = 5
x₁ − x₂ = 1

Matriks:

[ 2 1 ] [ x₁ ] = [ 5 ]
[ 1 -1 ] [ x₂ ] [ 1 ]

Matriks Augmented

Untuk mempermudah eliminasi, digunakan matriks gabungan:

[ 2 1 | 5 ]
[ 1 -1 | 1 ]

Penjumlahan Matriks

Dua matriks dapat dijumlahkan jika memiliki ukuran yang sama.

Operasi dilakukan elemen per elemen:

[ a b ] + [ e f ] = [ a+e b+f ]
[ c d ] [ g h ] [ c+g d+h ]

Perkalian dengan Skalar

Matriks dapat dikalikan dengan konstanta (skalar).

Contoh:

k [ a b ] = [ ka kb ]
[ c d ] [ kc kd ]

Perkalian Matriks

Matriks A dapat dikalikan dengan B jika jumlah kolom A = jumlah baris B.

Elemen hasil diperoleh dari perkalian baris dan kolom.

(AB)ij = Σ aikbkj

Operasi Baris Elementer (OBE)

Operasi baris digunakan untuk mengubah matriks menjadi bentuk yang lebih sederhana.

Terdapat tiga operasi dasar:

1. Tukar baris: Rᵢ ↔ Rⱼ
2. Kali baris: Rᵢ = kRᵢ
3. Kombinasi baris: Rᵢ = Rᵢ + kRⱼ

Operasi Eliminasi

Operasi yang paling sering digunakan dalam metode Gauss adalah:

Rᵢ = Rᵢ − kRⱼ

Tujuannya adalah membuat elemen tertentu menjadi nol.

Contoh Operasi Baris

[ 2 1 | 5 ]
[ 1 -1 | 1 ]

Eliminasi x₁ pada baris kedua:

R₂ = R₂ − (1/2)R₁

Hasil:

[ 2 1 | 5 ]
[ 0 -1.5 | -1.5 ]

Tujuan Operasi Matriks

Operasi ini digunakan untuk:

• Menyederhanakan sistem
• Mengubah ke bentuk segitiga
• Mempermudah pencarian solusi

Hubungan dengan Metode Gauss

Semua langkah dalam eliminasi Gauss berasal dari operasi baris elementer.

Tanpa memahami OBE, metode Gauss hanya akan terlihat sebagai langkah mekanis.

Intuisi

Operasi baris seperti “mengatur ulang” persamaan tanpa mengubah solusi.

Tujuannya adalah membuat sistem menjadi lebih mudah diselesaikan.

Kesimpulan

Representasi matriks memungkinkan SPL diolah secara sistematis.

Operasi matriks, terutama operasi baris, adalah kunci dalam metode eliminasi.

Pemahaman ini menjadi dasar untuk metode Gauss dan metode lanjutan lainnya.
