Metode Iteratif Jacobi

Metode Jacobi adalah metode iteratif untuk menyelesaikan Sistem Persamaan Linear dengan pendekatan berulang.

Berbeda dengan metode langsung, Jacobi tidak mencari solusi dalam satu proses, tetapi melalui iterasi bertahap.

Konsep Dasar

Setiap variabel dihitung ulang menggunakan nilai dari iterasi sebelumnya.

Semua nilai baru dihitung secara terpisah (tidak langsung diperbarui).

Bentuk Sistem

Diberikan SPL:

a₁₁x₁ + a₁₂x₂ + ... = b₁
a₂₁x₁ + a₂₂x₂ + ... = b₂

Setiap persamaan diubah untuk mengekspresikan satu variabel.

Rumus Iterasi Jacobi

xᵢ^(k+1) = (1 / aᵢᵢ) [ bᵢ − Σ (aᵢⱼ xⱼ^(k)) ]

Nilai baru dihitung dari nilai lama (iterasi sebelumnya).

Langkah-langkah Metode

1. Tentukan tebakan awal x⁰
2. Hitung semua variabel baru
3. Gunakan nilai lama dalam perhitungan
4. Ulangi hingga konvergen

Contoh Sistem

8x₁ + x₂ − x₃ = 8
2x₁ + x₂ + 9x₃ = 12
x₁ − 7x₂ + 2x₃ = -4

Tebakan awal: x₁⁰ = 0, x₂⁰ = 0, x₃⁰ = 0

Iterasi 1

x₁¹ = (8 − x₂⁰ + x₃⁰) / 8 = 1
x₂¹ = (12 − 2x₁⁰ − 9x₃⁰) / 1 = 12
x₃¹ = (-4 − x₁⁰ + 7x₂⁰) / 2 = -2

Iterasi Selanjutnya

Nilai iterasi sebelumnya digunakan kembali untuk menghitung nilai baru.

Proses diulang hingga nilai stabil.

Syarat Konvergensi

Metode Jacobi akan konvergen jika matriks memiliki dominasi diagonal.

|aᵢᵢ| > Σ |aᵢⱼ|

untuk setiap baris.

Kelebihan

• Mudah dipahami
• Mudah diimplementasikan
• Cocok untuk sistem besar

Kelemahan

• Konvergensi lambat
• Tidak selalu konvergen
• Sensitif terhadap bentuk matriks

Perbandingan dengan Metode Langsung

• Metode langsung → solusi sekali hitung
• Jacobi → solusi bertahap (iteratif)

Jacobi lebih fleksibel untuk sistem besar, tetapi membutuhkan iterasi.

Intuisi

Jacobi seperti “menebak lalu memperbaiki” semua variabel secara bersamaan.

Setiap iterasi mendekatkan hasil ke solusi sebenarnya.

Kesimpulan

Metode Jacobi adalah metode iteratif sederhana untuk menyelesaikan SPL.

Keberhasilannya bergantung pada struktur matriks dan kondisi konvergensi.
