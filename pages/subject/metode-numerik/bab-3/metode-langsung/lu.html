Metode Dekomposisi LU

Dekomposisi LU adalah metode untuk menyelesaikan Sistem Persamaan Linear dengan memfaktorkan matriks menjadi dua bagian.

Matriks A diuraikan menjadi:

A = L U

Konsep Dasar

Matriks koefisien dipecah menjadi:

L → matriks segitiga bawah
U → matriks segitiga atas

Dengan cara ini, penyelesaian SPL menjadi lebih terstruktur.

Bentuk Matriks L dan U

L (Lower Triangular):

[ 1 0 0 ]
[ l₂₁ 1 0 ]
[ l₃₁ l₃₂ 1 ]

U (Upper Triangular):

[ u₁₁ u₁₂ u₁₃ ]
[ 0 u₂₂ u₂₃ ]
[ 0 0 u₃₃ ]

Ide Utama

Daripada langsung menyelesaikan Ax = b, kita ubah menjadi:

LUx = b

Kemudian:

Ly = b → (forward substitution)
Ux = y → (back substitution)

Langkah-langkah Metode

1. Faktorkan A menjadi L dan U
2. Selesaikan Ly = b
3. Selesaikan Ux = y
4. Dapatkan solusi x

Contoh Matriks

A =

[ 2 3 ]
[ 4 7 ]

Dekomposisi

Misalkan:

L = [ 1 0 ]
[ 2 1 ]

U = [ 2 3 ]
[ 0 1 ]

Sehingga: A = LU

Forward Substitution

Ly = b

Digunakan untuk mencari vektor y dari matriks L.

Backward Substitution

Ux = y

Digunakan untuk mencari solusi akhir x.

Kelebihan

• Efisien untuk banyak sistem dengan A yang sama
• Mengurangi perhitungan berulang
• Dasar metode komputasi modern

Kelemahan

• Tidak semua matriks bisa langsung difaktorkan
• Memerlukan pivoting jika elemen nol muncul
• Lebih kompleks dari Gauss

Hubungan dengan Eliminasi Gauss

Dekomposisi LU sebenarnya berasal dari proses eliminasi Gauss.

Nilai pada matriks L berasal dari faktor eliminasi, sedangkan U adalah hasil akhirnya.

Intuisi

LU adalah cara “menyimpan langkah eliminasi” dalam bentuk matriks.

Jadi, kita tidak perlu mengulang proses setiap kali menyelesaikan sistem baru.

Kesimpulan

Dekomposisi LU memecah masalah besar menjadi dua langkah sederhana.

Metode ini sangat efisien terutama untuk sistem yang berulang.
