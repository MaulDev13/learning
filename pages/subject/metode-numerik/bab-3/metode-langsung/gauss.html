Metode Eliminasi Gauss

Metode Eliminasi Gauss adalah metode langsung untuk menyelesaikan Sistem Persamaan Linear dengan mengubah matriks menjadi bentuk segitiga atas.

Metode ini menggunakan operasi baris elementer untuk menyederhanakan sistem.

Konsep Dasar

Tujuan utama adalah mengubah matriks menjadi bentuk segitiga atas (upper triangular).

Setelah itu, solusi diperoleh dengan substitusi balik (back substitution).

Bentuk Segitiga Atas

Matriks diubah menjadi:

[ a₁₁ a₁₂ a₁₃ | b₁ ]
[ 0 a₂₂ a₂₃ | b₂ ]
[ 0 0 a₃₃ | b₃ ]

Semua elemen di bawah diagonal utama dibuat menjadi nol.

Langkah-langkah Metode

1. Bentuk matriks augmented
2. Gunakan operasi baris untuk membuat nol di bawah pivot
3. Ulangi hingga terbentuk segitiga atas
4. Lakukan substitusi balik

Contoh Kasus

2x + y − z = 8
-3x − y + 2z = -11
-2x + y + 2z = -3

Matriks augmented:

[ 2 1 -1 | 8 ]
[ -3 -1 2 | -11 ]
[ -2 1 2 | -3 ]

Eliminasi Langkah 1

Hilangkan elemen di bawah pivot pertama (a₁₁ = 2)

R₂ = R₂ + (3/2)R₁
R₃ = R₃ + R₁

Hasil:

[ 2 1 -1 | 8 ]
[ 0 0.5 0.5 | 1 ]
[ 0 2 1 | 5 ]

Eliminasi Langkah 2

Hilangkan elemen di bawah pivot kedua

R₃ = R₃ − 4R₂

Hasil:

[ 2 1 -1 | 8 ]
[ 0 0.5 0.5 | 1 ]
[ 0 0 -1 | 1 ]

Substitusi Balik

Dari baris terakhir:

-z = 1 → z = -1

Baris kedua:

0.5y + 0.5z = 1 → y = 3

Baris pertama:

2x + y − z = 8 → x = 2

Hasil Akhir

x = 2
y = 3
z = -1

Kelebihan

• Sistematis dan pasti selesai
• Cocok untuk berbagai ukuran sistem
• Dasar metode lain (LU, Gauss-Jordan)

Kelemahan

• Bisa terjadi pembulatan error
• Perlu banyak operasi untuk sistem besar
• Sensitif jika pivot kecil (perlu pivoting)

Intuisi

Metode Gauss adalah proses “menghilangkan variabel satu per satu”.

Sampai tersisa satu variabel, lalu diselesaikan kembali ke atas.

Kesimpulan

Metode Eliminasi Gauss adalah metode dasar untuk menyelesaikan SPL secara langsung.

Dengan operasi baris, sistem dapat disederhanakan hingga mudah diselesaikan.
