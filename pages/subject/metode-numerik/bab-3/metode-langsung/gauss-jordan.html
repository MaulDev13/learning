Metode Gauss-Jordan

Metode Gauss-Jordan adalah pengembangan dari metode Eliminasi Gauss untuk menyelesaikan Sistem Persamaan Linear.

Metode ini mengubah matriks menjadi bentuk eselon baris tereduksi (Reduced Row Echelon Form).

Konsep Dasar

Berbeda dengan Gauss, metode ini tidak hanya membuat nol di bawah pivot, tetapi juga di atas pivot.

Hasil akhirnya adalah matriks identitas.

Bentuk Akhir

Matriks akan menjadi:

[ 1 0 0 | x ]
[ 0 1 0 | y ]
[ 0 0 1 | z ]

Solusi dapat langsung dibaca.

Langkah-langkah Metode

1. Bentuk matriks augmented
2. Lakukan eliminasi seperti Gauss
3. Buat pivot menjadi 1
4. Hilangkan elemen di atas dan bawah pivot
5. Ulangi hingga menjadi matriks identitas

Contoh Kasus

2x + y − z = 8
-3x − y + 2z = -11
-2x + y + 2z = -3

Matriks augmented:

[ 2 1 -1 | 8 ]
[ -3 -1 2 | -11 ]
[ -2 1 2 | -3 ]

Langkah 1: Eliminasi ke Segitiga Atas

Gunakan langkah seperti Gauss:

[ 2 1 -1 | 8 ]
[ 0 0.5 0.5 | 1 ]
[ 0 0 -1 | 1 ]

Langkah 2: Normalisasi Pivot

Ubah pivot menjadi 1:

R₃ = -1 × R₃

[ 2 1 -1 | 8 ]
[ 0 0.5 0.5 | 1 ]
[ 0 0 1 | -1 ]

Langkah 3: Eliminasi ke Atas

Hilangkan elemen di atas pivot:

R₂ = R₂ − 0.5R₃
R₁ = R₁ + R₃

Hasil:

[ 2 1 0 | 7 ]
[ 0 0.5 0 | 1.5 ]
[ 0 0 1 | -1 ]

Langkah 4: Normalisasi Baris Kedua

R₂ = 2R₂

[ 2 1 0 | 7 ]
[ 0 1 0 | 3 ]
[ 0 0 1 | -1 ]

Langkah 5: Eliminasi ke Atas Lagi

R₁ = R₁ − R₂

[ 2 0 0 | 4 ]
[ 0 1 0 | 3 ]
[ 0 0 1 | -1 ]

Langkah 6: Normalisasi Akhir

R₁ = (1/2)R₁

[ 1 0 0 | 2 ]
[ 0 1 0 | 3 ]
[ 0 0 1 | -1 ]

Hasil Akhir

x = 2
y = 3
z = -1

Perbandingan dengan Gauss

• Gauss → butuh substitusi balik
• Gauss-Jordan → langsung dapat solusi

Gauss-Jordan lebih lengkap, tetapi membutuhkan lebih banyak operasi.

Kelebihan

• Solusi langsung terlihat
• Sistematis
• Cocok untuk komputer

Kelemahan

• Lebih banyak operasi
• Kurang efisien untuk sistem besar
• Lebih rentan terhadap error pembulatan

Intuisi

Jika Gauss adalah “setengah bersih”, maka Gauss-Jordan adalah “bersih total”.

Semua variabel dipisahkan secara langsung dalam bentuk matriks identitas.

Kesimpulan

Metode Gauss-Jordan menyelesaikan SPL dengan mengubah matriks menjadi identitas.

Metode ini memberikan solusi secara langsung, namun dengan biaya komputasi lebih besar.
