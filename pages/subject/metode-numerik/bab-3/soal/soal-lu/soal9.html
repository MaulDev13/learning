Penyelesaian Soal 9 (Metode LU)

Sistem ini diselesaikan menggunakan metode dekomposisi LU, dengan memfaktorkan matriks A menjadi matriks L dan U.

Metode: Dekomposisi LU

Sistem Persamaan

11x₁ − x₂ + x₃ = 13
−x₁ + 10x₂ − x₃ = 9
x₁ − x₂ + 9x₃ = 8

Matriks A

[ 11 -1 1 ]
[ -1 10 -1 ]
[ 1 -1 9 ]

Dekomposisi LU

L = [ 1 0 0 ]
[ -0.091 1 0 ]
[ 0.091 -0.082 1 ]

U = [ 11 -1 1 ]
[ 0 9.909 -0.909 ]
[ 0 0 8.826 ]

Penyelesaian Ly = b

y₁ = 13
y₂ = 9 + 0.091(13) ≈ 10.182
y₃ = 8 - 0.091(13) + 0.082(10.182) ≈ 7.653

Penyelesaian Ux = y

8.826x₃ = 7.653 → x₃ ≈ 0.867
9.909x₂ − 0.909x₃ = 10.182 → x₂ ≈ 1.106
11x₁ − x₂ + x₃ = 13 → x₁ ≈ 1.203

Hasil Akhir

x₁ ≈ 1.203
x₂ ≈ 1.106
x₃ ≈ 0.867

Metode LU memberikan solusi stabil dan akurat.
