Penyelesaian Soal 3 (Metode LU)

Sistem ini diselesaikan menggunakan metode dekomposisi LU, yaitu memfaktorkan matriks A menjadi matriks L dan U.

Metode: Dekomposisi LU

Sistem Persamaan

4x₁ + x₂ + x₃ = 7
x₁ + 5x₂ + x₃ = 8
x₁ + x₂ + 6x₃ = 9

Matriks A

[ 4 1 1 ]
[ 1 5 1 ]
[ 1 1 6 ]

Dekomposisi LU

L = [ 1 0 0 ]
[ 0.25 1 0 ]
[ 0.25 0.158 1 ]

U = [ 4 1 1 ]
[ 0 4.75 0.75 ]
[ 0 0 5.632 ]

Penyelesaian Ly = b

y₁ = 7
y₂ = 8 - 0.25(7) = 6.25
y₃ = 9 - 0.25(7) - 0.158(6.25) ≈ 6.2625

Penyelesaian Ux = y

5.632x₃ = 6.2625 → x₃ ≈ 1.111
4.75x₂ + 0.75x₃ = 6.25 → x₂ ≈ 1.143
4x₁ + x₂ + x₃ = 7 → x₁ ≈ 1.186

Hasil Akhir

x₁ ≈ 1.186
x₂ ≈ 1.143
x₃ ≈ 1.111

Metode LU memberikan solusi stabil dan akurat.
