Penyelesaian Soal 6 (Metode LU)

Sistem ini diselesaikan menggunakan metode dekomposisi LU, dengan memfaktorkan matriks A menjadi matriks L dan U.

Metode: Dekomposisi LU

Sistem Persamaan

3x₁ − x₂ + x₃ = 4
x₁ + 4x₂ − x₃ = 5
x₁ − x₂ + 5x₃ = 6

Matriks A

[ 3 -1 1 ]
[ 1 4 -1 ]
[ 1 -1 5 ]

Dekomposisi LU

L = [ 1 0 0 ]
[ 0.333 1 0 ]
[ 0.333 -0.091 1 ]

U = [ 3 -1 1 ]
[ 0 4.333 -1.333 ]
[ 0 0 4.545 ]

Penyelesaian Ly = b

y₁ = 4
y₂ = 5 - 0.333(4) ≈ 3.667
y₃ = 6 - 0.333(4) + 0.091(3.667) ≈ 5.00

Penyelesaian Ux = y

4.545x₃ = 5 → x₃ ≈ 1.100
4.333x₂ − 1.333x₃ = 3.667 → x₂ ≈ 1.200
3x₁ − x₂ + x₃ = 4 → x₁ ≈ 1.367

Hasil Akhir

x₁ ≈ 1.367
x₂ ≈ 1.200
x₃ ≈ 1.100

Metode LU memberikan solusi stabil.
