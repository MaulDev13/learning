Penyelesaian Soal 1 (Metode LU)

Sistem persamaan linear ini diselesaikan menggunakan dekomposisi LU, yaitu metode langsung dengan memfaktorkan matriks A menjadi L (lower) dan U (upper).

Metode: Dekomposisi LU

Sistem Persamaan

8x₁ + x₂ − x₃ = 8
2x₁ + x₂ + 9x₃ = 12
x₁ − 7x₂ + 2x₃ = -4

Matriks A

[ 8 1 -1 ]
[ 2 1 9 ]
[ 1 -7 2 ]

Dekomposisi LU

A = L · U

L = [ 1 0 0 ]
[ 0.25 1 0 ]
[ 0.125 -6 1 ]

U = [ 8 1 -1 ]
[ 0 0.75 9.25 ]
[ 0 0 57.5 ]

Penyelesaian Ly = b

y₁ = 8
y₂ = 12 - 0.25(8) = 10
y₃ = -4 - 0.125(8) + 6(10) = 55

Penyelesaian Ux = y

57.5x₃ = 55 → x₃ ≈ 0.957
0.75x₂ + 9.25x₃ = 10 → x₂ ≈ 1.523
8x₁ + x₂ - x₃ = 8 → x₁ ≈ 0.929

Hasil Akhir

x₁ ≈ 0.929
x₂ ≈ 1.523
x₃ ≈ 0.957

Metode LU memberikan solusi eksak (hingga pembulatan).
