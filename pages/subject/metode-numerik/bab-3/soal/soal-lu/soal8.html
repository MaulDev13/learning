Penyelesaian Soal 8 (Metode LU)

Sistem ini diselesaikan menggunakan metode dekomposisi LU, dengan memfaktorkan matriks A menjadi matriks L dan U.

Metode: Dekomposisi LU

Sistem Persamaan

6x₁ + x₂ − x₃ = 10
x₁ + 7x₂ + x₃ = 14
x₁ − x₂ + 6x₃ = 12

Matriks A

[ 6 1 -1 ]
[ 1 7 1 ]
[ 1 -1 6 ]

Dekomposisi LU

L = [ 1 0 0 ]
[ 0.167 1 0 ]
[ 0.167 -0.167 1 ]

U = [ 6 1 -1 ]
[ 0 6.833 1.167 ]
[ 0 0 6.028 ]

Penyelesaian Ly = b

y₁ = 10
y₂ = 14 - 0.167(10) ≈ 12.333
y₃ = 12 - 0.167(10) + 0.167(12.333) ≈ 12.389

Penyelesaian Ux = y

6.028x₃ = 12.389 → x₃ ≈ 2.055
6.833x₂ + 1.167x₃ = 12.333 → x₂ ≈ 1.467
6x₁ + x₂ − x₃ = 10 → x₁ ≈ 1.765

Hasil Akhir

x₁ ≈ 1.765
x₂ ≈ 1.467
x₃ ≈ 2.055

Metode LU memberikan solusi stabil dan akurat.
