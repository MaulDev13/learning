Metode Simpson 3/8

Metode Simpson 3/8 merupakan salah satu metode integrasi numerik yang menggunakan pendekatan polinomial kubik (orde tiga) untuk menghampiri nilai integral tentu.

Dibandingkan metode Simpson 1/3 yang menggunakan parabola, Simpson 3/8 menggunakan tiga subinterval sekaligus untuk membentuk pendekatan kurva yang lebih fleksibel.

Konsep Dasar

Diberikan integral tentu:

$$ \int_a^b f(x)\,dx $$

Interval \([a, b]\) dibagi menjadi \(n\) subinterval dengan syarat:

$$ n \equiv 0 \pmod{3} $$

Lebar subinterval:

$$ h = \frac{b-a}{n} $$

Rumus Simpson 3/8 Komposit

Metode Simpson 3/8 menggunakan pola bobot yang khas dengan siklus 3 titik.

$$ \int_a^b f(x)\,dx \approx \frac{3h}{8} \left[ f(x_0) + 3\sum_{i=1,2,4,5,\dots}^{n-1} f(x_i) + 2\sum_{i=3,6,9,\dots}^{n-3} f(x_i) + f(x_n) \right] $$

Pola bobotnya mengikuti siklus: 1 – 3 – 3 – 2 – 3 – 3 – 2 – ... – 3 – 3 – 1

Interpretasi Geometris

Metode Simpson 3/8 mengaproksimasi kurva menggunakan polinomial kubik yang melewati empat titik berturut-turut.

Dengan pendekatan ini, bentuk kurva yang lebih kompleks dapat direpresentasikan dengan lebih baik dibanding metode Simpson 1/3.

Hal ini membuat Simpson 3/8 lebih fleksibel, terutama ketika jumlah titik tidak cocok untuk metode Simpson 1/3.

Syarat Penggunaan

Metode Simpson 3/8 memiliki beberapa syarat penting:

- Jumlah subinterval \(n\) harus kelipatan 3
- Titik-titik harus berjarak sama
- Fungsi sebaiknya cukup halus pada interval yang dianalisis

Algoritma Simpson 3/8

Langkah umum penerapan metode Simpson 3/8:

1. Tentukan batas integral \(a\) dan \(b\).
2. Pilih jumlah subinterval \(n\) (kelipatan 3).
3. Hitung \(h = (b-a)/n\).
4. Evaluasi fungsi pada setiap titik \(x_i\).
5. Terapkan rumus Simpson 3/8 komposit.

Pseudocode Simpson 3/8

INPUT a, b, n IF n % 3 != 0 THEN ERROR "n harus kelipatan 3" END IF h = (b - a) / n sum = f(a) + f(b) FOR i = 1 TO n-1 DO x = a + i*h IF i % 3 == 0 THEN sum = sum + 2*f(x) ELSE sum = sum + 3*f(x) END IF END FOR result = (3*h/8) * sum OUTPUT result

Contoh Konseptual

Misalkan ingin menghitung:

$$ \int_0^3 x^2\,dx $$

Nilai eksak integral tersebut adalah:

$$ 9 $$

Dengan Simpson 3/8, pendekatan dilakukan menggunakan pembagian interval menjadi kelipatan 3 sehingga hasil aproksimasi semakin mendekati nilai eksak ketika jumlah subinterval diperbesar.

Kelebihan Simpson 3/8

- Lebih fleksibel untuk jumlah subinterval tertentu (kelipatan 3)
- Akurasi tinggi karena menggunakan polinomial kubik
- Lebih baik untuk fungsi dengan variasi kurva lebih kompleks

Keterbatasan Simpson 3/8

- Harus menggunakan jumlah subinterval kelipatan 3
- Lebih kompleks dibanding Simpson 1/3
- Tidak selalu lebih efisien jika fungsi sangat sederhana

Kesimpulan

Metode Simpson 3/8 merupakan pengembangan dari metode integrasi numerik yang menggunakan pendekatan polinomial kubik untuk meningkatkan akurasi aproksimasi.

Metode ini sering digunakan bersama Simpson 1/3 untuk menangani kasus ketika jumlah subinterval tidak memenuhi syarat metode 1/3.
