Metode Integrasi Simpson 3/8

Metode Simpson 3/8 adalah metode integrasi numerik yang digunakan untuk menghitung pendekatan luas di bawah kurva.

Metode ini menggunakan pendekatan polinomial kubik (derajat 3) yang melewati empat titik data.

Hasilnya lebih akurat dibanding Simpson 1/3 pada beberapa kasus, tetapi jarang digunakan secara tunggal.

Ide Dasar Metode

Simpson 3/8 mengaproksimasi fungsi dengan polinomial kubik pada setiap tiga sub-interval.

Dengan demikian, kurva diperkirakan lebih fleksibel dibanding pendekatan parabola pada Simpson 1/3.

Metode ini cocok untuk jumlah interval tertentu.

Syarat Penggunaan

Metode Simpson 3/8 memiliki syarat:

• Jumlah interval (n) harus kelipatan 3
• Jarak antar titik harus sama (uniform spacing)

Rumus Simpson 3/8

Untuk interval [a, b] dengan n = 3m:

h = (b - a) / n

∫ f(x) dx ≈ 3h/8 [ f(x₀) + f(xₙ) + 3Σ f(x tidak kelipatan 3) + 2Σ f(x kelipatan 3, selain awal & akhir) ]

Pola Koefisien

Pola koefisien Simpson 3/8 adalah:

1, 3, 3, 2, 3, 3, 2, ..., 3, 3, 1

Titik awal dan akhir bernilai 1, sedangkan sisanya mengikuti pola 3 dan 2.

Langkah-langkah

1. Tentukan interval [a, b]
2. Pastikan jumlah interval kelipatan 3
3. Hitung nilai h
4. Hitung f(x) di semua titik
5. Terapkan rumus Simpson 3/8

Contoh Sederhana

Diketahui:

f(x) = x², interval [0, 3], n = 3

Titik: x₀ = 0, x₁ = 1, x₂ = 2, x₃ = 3

Perhitungan

h = (3 - 0) / 3 = 1

f(0)=0, f(1)=1, f(2)=4, f(3)=9

∫ f(x) dx ≈ 3/8 [ 0 + 9 + 3(1 + 4) ]

= 3/8 [ 9 + 15 ] = 3/8 (24) = 9

Kelebihan

• Lebih fleksibel untuk fungsi tertentu
• Akurat untuk beberapa kasus non-linear
• Menggunakan pendekatan polinomial kubik

Kekurangan

• Harus kelipatan 3 interval
• Lebih kompleks dari Simpson 1/3
• Jarang digunakan sendiri dalam praktik

Kesimpulan

Metode Simpson 3/8 adalah teknik integrasi numerik yang menggunakan pendekatan polinomial kubik untuk menghitung luas di bawah kurva.

Meskipun akurat, penggunaannya biasanya dikombinasikan dengan Simpson 1/3 untuk fleksibilitas lebih tinggi.
