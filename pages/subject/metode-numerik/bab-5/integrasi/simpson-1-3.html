Metode Integrasi Simpson 1/3

Metode Simpson 1/3 adalah metode integrasi numerik yang digunakan untuk menghitung pendekatan luas di bawah kurva.

Metode ini menggunakan pendekatan parabola (polinomial derajat 2) untuk menggantikan kurva pada setiap dua interval.

Hasilnya biasanya lebih akurat dibanding metode trapesium.

Ide Dasar Metode

Simpson 1/3 mengaproksimasi kurva menggunakan parabola yang melewati tiga titik data.

Dengan pendekatan ini, bentuk kurva lebih realistis dibanding garis lurus pada metode trapesium.

Metode ini membutuhkan jumlah interval yang genap.

Syarat Penggunaan

Metode Simpson 1/3 memiliki syarat penting:

• Jumlah interval (n) harus genap
• Jarak antar titik harus sama (uniform spacing)

Rumus Simpson 1/3

Untuk interval [a, b] dengan n genap:

h = (b - a) / n

∫ f(x) dx ≈ h/3 [ f(x₀) + f(xₙ) + 4Σ f(x ganjil) + 2Σ f(x genap) ]

(tanpa menghitung titik awal dan akhir di penjumlahan)

Pola Koefisien

Dalam Simpson 1/3, koefisien memiliki pola:

1, 4, 2, 4, 2, ..., 4, 1

Titik awal dan akhir bernilai 1, sedangkan titik di tengah bergantian 4 dan 2.

Langkah-langkah

1. Tentukan interval [a, b]
2. Pilih jumlah interval n (harus genap)
3. Hitung jarak h
4. Hitung nilai f(x) pada setiap titik
5. Terapkan rumus Simpson 1/3

Contoh Sederhana

Diketahui:

f(x) = x², interval [0, 2], n = 2

Titik: x₀ = 0, x₁ = 1, x₂ = 2

Perhitungan

h = (2 - 0) / 2 = 1

f(0) = 0, f(1) = 1, f(2) = 4

∫ f(x) dx ≈ 1/3 [ 0 + 4(1) + 4 ]

= 1/3 (8) = 2.666...

Kelebihan

• Lebih akurat dari metode trapesium
• Menggunakan pendekatan parabola
• Efisien untuk fungsi halus

Kekurangan

• Harus menggunakan jumlah interval genap
• Kurang fleksibel untuk data tidak beraturan
• Lebih kompleks dari trapesium

Kesimpulan

Metode Simpson 1/3 adalah metode integrasi numerik yang lebih akurat dibanding trapesium karena menggunakan pendekatan parabola.

Metode ini sangat efektif untuk fungsi yang halus dengan interval yang seragam.
