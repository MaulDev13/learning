Metode Simpson 1/3

Metode Simpson 1/3 merupakan metode integrasi numerik yang menggunakan pendekatan polinomial kuadratik untuk menghampiri nilai integral tentu. Metode ini memberikan akurasi yang lebih tinggi dibandingkan metode trapesium karena pendekatan kurva tidak lagi linier, melainkan parabola.

Ide dasar metode ini adalah mendekati fungsi dengan parabola pada setiap dua subinterval. Oleh karena itu, jumlah partisi \(n\) harus genap.

Konsep Dasar

Diberikan integral tentu:

$$ \int_a^b f(x)\,dx $$

Interval \([a,b]\) dibagi menjadi \(n\) subinterval dengan \(n\) harus genap.

$$ h = \frac{b-a}{n} $$

Titik-titik partisi:

$$ x_0, x_1, x_2, \dots, x_n $$

Rumus Simpson 1/3 Komposit

Metode Simpson 1/3 menggunakan bobot berbeda untuk titik ganjil dan genap.

$$ \int_a^b f(x)\,dx \approx \frac{h}{3} \left[ f(x_0) + 4\sum_{i=1,3,5}^{n-1} f(x_i) + 2\sum_{i=2,4,6}^{n-2} f(x_i) + f(x_n) \right] $$

Pola koefisien 1-4-2-4-2-...-4-1 merupakan ciri khas metode Simpson 1/3.

Interpretasi Geometris

Berbeda dengan metode trapesium yang menggunakan garis lurus, Simpson 1/3 menggunakan parabola untuk mendekati bentuk kurva pada setiap dua interval.

Dengan pendekatan ini, perubahan kelengkungan fungsi dapat ditangkap dengan lebih baik, sehingga hasil aproksimasi menjadi lebih akurat.

Syarat Penggunaan

Metode Simpson 1/3 memiliki syarat penting:

- Jumlah subinterval \(n\) harus genap
- Fungsi harus cukup halus pada interval yang digunakan
- Lebar interval seragam

Algoritma Metode Simpson 1/3

Langkah-langkah umum metode Simpson 1/3:

1. Tentukan batas integral \(a\) dan \(b\).
2. Pilih \(n\) (harus genap).
3. Hitung \(h = (b-a)/n\).
4. Hitung nilai fungsi pada setiap titik \(x_i\).
5. Terapkan rumus Simpson 1/3.

Pseudocode Simpson 1/3

INPUT a, b, n IF n % 2 != 0 THEN ERROR "n harus genap" END IF h = (b - a) / n sum = f(a) + f(b) FOR i = 1 TO n-1 DO x = a + i*h IF i % 2 == 0 THEN sum = sum + 2*f(x) ELSE sum = sum + 4*f(x) END IF END FOR result = (h/3) * sum OUTPUT result

Contoh Konseptual

Misalkan ingin menghitung:

$$ \int_0^2 x^2\,dx $$

Nilai eksak:

$$ \frac{8}{3} $$

Dengan Simpson 1/3, pendekatan biasanya lebih dekat ke nilai eksak dibanding metode trapesium, terutama ketika jumlah subinterval cukup besar.

Kelebihan Simpson 1/3

- Akurasi lebih tinggi dibanding metode trapesium
- Menggunakan pendekatan parabola sehingga lebih sesuai untuk fungsi melengkung
- Error orde lebih kecil untuk fungsi halus

Keterbatasan Simpson 1/3

- Harus menggunakan jumlah subinterval genap
- Tidak optimal untuk fungsi yang sangat tidak halus atau diskontinu
- Lebih kompleks dibanding metode trapesium

Kesimpulan

Metode Simpson 1/3 merupakan metode integrasi numerik yang lebih akurat dibandingkan metode trapesium karena menggunakan pendekatan polinomial kuadratik.

Metode ini menjadi dasar penting sebelum mempelajari Simpson 3/8 yang menggunakan pendekatan polinomial derajat lebih tinggi.
