Metode Integrasi Trapesium

Metode trapesium adalah salah satu metode integrasi numerik untuk menghitung pendekatan luas di bawah kurva.

Metode ini mendekati daerah di bawah grafik fungsi dengan bentuk trapesium, bukan kurva.

Semakin kecil interval pembagian, semakin akurat hasilnya.

Ide Dasar Metode

Luas di bawah kurva f(x) pada interval [a, b] didekati dengan luas trapesium.

Titik ujung interval digunakan sebagai dasar perhitungan tinggi trapesium.

Kurva dianggap garis lurus di antara dua titik.

Rumus Trapesium Sederhana

Untuk satu interval [a, b]:

∫ f(x) dx ≈ (b - a) / 2 [ f(a) + f(b) ]

Rumus ini disebut single trapezoidal rule.

Metode Trapesium Komposit

Untuk meningkatkan akurasi, interval dibagi menjadi n bagian.

h = (b - a) / n

Rumus:

∫ f(x) dx ≈ h/2 [ f(x₀) + 2Σ f(xᵢ) + f(xₙ) ]

i = 1 sampai n-1

Langkah-langkah

1. Tentukan batas integral a dan b
2. Tentukan jumlah interval n
3. Hitung lebar interval h
4. Hitung nilai f(x) pada setiap titik
5. Substitusi ke rumus trapesium

Contoh Sederhana

Diketahui:

f(x) = x², interval [0, 2], n = 2

Titik: x₀ = 0, x₁ = 1, x₂ = 2

Perhitungan

h = (2 - 0) / 2 = 1

f(0) = 0, f(1) = 1, f(2) = 4

∫ f(x) dx ≈ 1/2 [0 + 2(1) + 4]

= 1/2 (6) = 3

Kelebihan

• Mudah dipahami dan diterapkan
• Cocok untuk data diskrit
• Lebih akurat dengan pembagian kecil

Kekurangan

• Kurang akurat untuk fungsi non-linear tajam
• Error masih cukup besar dibanding Simpson
• Bergantung pada jumlah interval

Kesimpulan

Metode trapesium adalah teknik integrasi numerik sederhana yang mendekati area di bawah kurva dengan bentuk trapesium.

Akurasi metode ini meningkat jika jumlah interval diperbanyak.
