Diferensiasi Mundur (Backward Difference)

Diferensiasi mundur adalah metode numerik untuk memperkirakan turunan suatu fungsi menggunakan dua titik berurutan, dengan pendekatan dari arah belakang.

Metode ini digunakan ketika data tersedia dari titik tertentu dan nilai sebelumnya sudah diketahui.

Cocok untuk data yang bergerak dari kanan ke kiri pada tabel diskrit.

Ide Dasar Metode

Turunan didekati menggunakan selisih nilai fungsi pada titik x dan titik sebelumnya (x - h).

Semakin kecil nilai h, semakin baik pendekatan turunan yang diperoleh.

Metode ini mengasumsikan perubahan fungsi linier pada interval kecil.

Rumus Diferensiasi Mundur

Turunan pertama:

f'(x) ≈ ( f(x) - f(x - h) ) / h

Dimana:

• h = jarak antar titik (step size)
• x - h = titik sebelumnya

Makna Geometris

Secara geometris, diferensiasi mundur menghitung kemiringan garis yang menghubungkan titik x dengan titik sebelumnya (x - h).

Garis ini juga disebut garis sekant, tetapi diambil dari arah belakang.

Langkah-langkah

1. Tentukan fungsi f(x)
2. Pilih titik x yang ingin dicari turunannya
3. Tentukan nilai h kecil
4. Hitung f(x) dan f(x - h)
5. Substitusikan ke rumus

Contoh Sederhana

Diketahui:

f(x) = x², x = 2, h = 0.1

Maka: f(2) dan f(1.9)

Perhitungan

f(2) = 4
f(1.9) = 3.61

f'(2) ≈ (4 - 3.61) / 0.1

f'(2) ≈ 0.39 / 0.1 = 3.9

Kelebihan

• Mudah digunakan
• Cocok untuk data di akhir tabel
• Rumus sederhana

Kekurangan

• Kurang akurat untuk h besar
• Error lebih tinggi dibanding central difference
• Bergantung pada ketersediaan data sebelumnya

Kesimpulan

Diferensiasi mundur adalah metode sederhana untuk memperkirakan turunan menggunakan nilai fungsi saat ini dan nilai sebelumnya.

Metode ini efektif untuk data diskrit, namun akurasinya bergantung pada ukuran langkah h.
