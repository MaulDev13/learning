Diferensiasi Maju (Forward Difference)

Diferensiasi maju adalah metode numerik untuk memperkirakan turunan suatu fungsi menggunakan dua titik berurutan.

Metode ini termasuk pendekatan paling sederhana dalam diferensiasi numerik.

Cocok digunakan ketika data tersedia dari titik awal ke arah kanan.

Ide Dasar Metode

Turunan suatu fungsi didekati menggunakan perubahan nilai fungsi pada selang kecil h ke depan.

Semakin kecil nilai h, semakin akurat hasil pendekatan turunan.

Metode ini mengasumsikan perubahan fungsi bersifat linear pada interval kecil.

Rumus Diferensiasi Maju

Turunan pertama:

f'(x) ≈ ( f(x + h) - f(x) ) / h

Dimana:

• h = jarak antar titik (step size)
• x + h = titik berikutnya

Makna Geometris

Secara geometris, diferensiasi maju menghitung kemiringan garis yang menghubungkan titik x dengan titik di depannya (x + h).

Garis ini disebut garis sekant.

Langkah-langkah

1. Tentukan fungsi f(x)
2. Pilih titik x yang ingin dicari turunannya
3. Tentukan nilai h kecil
4. Hitung f(x + h) dan f(x)
5. Substitusikan ke rumus

Contoh Sederhana

Diketahui:

f(x) = x², x = 2, h = 0.1

Maka: f(2) dan f(2.1)

Perhitungan

f(2) = 4
f(2.1) = 4.41

f'(2) ≈ (4.41 - 4) / 0.1

f'(2) ≈ 0.41 / 0.1 = 4.1

Kelebihan

• Rumus sederhana
• Mudah diimplementasikan
• Cocok untuk data berurutan

Kekurangan

• Kurang akurat untuk h besar
• Error truncation relatif tinggi
• Tidak cocok untuk data noise

Kesimpulan

Diferensiasi maju adalah metode sederhana untuk memperkirakan turunan dengan menggunakan nilai fungsi di titik saat ini dan titik berikutnya.

Metode ini efektif untuk pendekatan cepat, namun akurasinya bergantung pada ukuran langkah h.
