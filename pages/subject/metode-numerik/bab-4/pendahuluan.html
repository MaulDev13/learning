Polinomial dalam Metode Numerik

Polinomial adalah bentuk fungsi matematika yang paling sederhana namun sangat powerful. Dalam metode numerik, polinomial memegang peranan sentral karena kemampuannya untuk mengaproksimasi fungsi kompleks, menginterpolasi data, dan menjadi dasar dari banyak algoritma numerik.

Secara formal, polinomial derajat n dalam variabel x didefinisikan sebagai:

P_n(x) = a_n x^n + a_{n-1} x^{n-1} + ... + a_1 x + a_0

dengan a_n ≠ 0 dan a_i adalah koefisien (bisa bilangan real atau kompleks).

Mengapa Polinomial Penting dalam Metode Numerik?

Polinomial memiliki sifat-sifat istimewa yang membuatnya sangat berguna dalam komputasi numerik:

1. Mudah dievaluasi
Evaluasi polinomial cukup dengan operasi perkalian dan penjumlahan — sangat efisien di komputer.

2. Mudah diturunkan dan diintegralkan
Turunan dan integral polinomial juga berbentuk polinomial dengan derajat yang berkurang atau bertambah.

3. Teorema Aproksimasi Weierstrass
Setiap fungsi kontinu pada interval tertutup dapat didekati oleh polinomial dengan tingkat akurasi sebarang.

4. Mudah dicari akarnya
Meskipun tidak selalu trivial, polinomial memiliki algoritma numerik yang matang untuk pencarian akar.

"Polinomial adalah 'bahasa universal' dalam komputasi numerik
karena kesederhanaan dan fleksibilitasnya."

Bentuk Representasi Polinomial

Dalam praktik numerik, polinomial dapat direpresentasikan dalam beberapa bentuk, masing-masing dengan kelebihan dan kekurangan:

BentukRumusKelebihanKekurangan
Bentuk Standar a_n x^n + ... + a₀ Sederhana, mudah dipahami Sensitif terhadap perubahan koefisien
Bentuk Newton (Beda Terbagi) a₀ + a₁(x-x₀) + a₂(x-x₀)(x-x₁) + ... Mudah menambah titik data Kurang intuitif
Bentuk Lagrange Σ y_i · L_i(x) Eksplisit, cocok untuk interpolasi Mahal untuk evaluasi (O(n²))
Bentuk Horner a₀ + x(a₁ + x(a₂ + ...)) Efisien (O(n)), stabil Representasi tersembunyi
# Evaluasi polinomial bentuk Horner (paling efisien) # P(x) = a₀ + a₁·x + a₂·x² + ... + aₙ·xⁿ function horner(a, x): result = a[n] for i = n-1 down to 0: result = result * x + a[i] return result

Derajat dan Koefisien Polinomial

Derajat polinomial adalah pangkat tertinggi dari variabel dengan koefisien tidak nol. Beberapa istilah penting:

• Koefisien utama (leading coefficient) — koefisien dari suku berpangkat tertinggi
• Suku konstan (constant term) — koefisien dari x⁰ = a₀
• Polinomial monik (monic polynomial) — koefisien utama = 1

1Linearf(x) = 2x + 3
2Kuadratf(x) = x² - 4x + 4
3Kubikf(x) = x³ - 6x² + 11x - 6
4Kuartikf(x) = x⁴ - 5x² + 4

Akar-akar Polinomial

Akar atau solusi dari polinomial adalah nilai x yang memenuhi P(x) = 0. Teorema Dasar Aljabar menyatakan:

"Setiap polinomial derajat n (dengan koefisien kompleks)
memiliki tepat n akar (termasuk akar kompleks dan akar berulang)."

Beberapa fakta penting tentang akar polinomial:

• Akar real vs kompleks — polinomial dengan koefisien real memiliki akar kompleks yang berpasangan (konjugat).
• Akar berulang (multiple roots) — terjadi ketika polinomial dan turunannya memiliki akar yang sama.
• Akar rasional — jika koefisien bulat, akar rasional harus berbentuk p/q dengan p pembagi a₀ dan q pembagi aₙ.

Contoh: x² + 1 = 0 → akar: x = i dan x = -i (kompleks)
Contoh: (x-2)² = 0 → akar: x = 2 (akar ganda)

Operasi Dasar Polinomial

Dalam implementasi numerik, operasi polinomial yang sering dilakukan meliputi:

1. Penjumlahan dan Pengurangan
Menjumlahkan koefisien dengan derajat yang sama. Kompleksitas O(n).

2. Perkalian (Konvolusi)
Hasil kali polinomial derajat m dan n menghasilkan polinomial derajat m+n. Kompleksitas O(mn) secara naif, atau O(n log n) dengan FFT.

3. Pembagian
Polinomial dapat dibagi menggunakan algoritma pembagian panjang atau sintetik.

4. Turunan dan Integral
Turunan: P'(x) = Σ k·a_k·x^{k-1}
Integral: ∫P(x) dx = Σ a_k·x^{k+1}/(k+1) + C

# Pembagian polinomial (sintetik) # Membagi P(x) dengan (x - c) function synthetic_division(a, c): n = length(a) - 1 b = array of size n b[n-1] = a[n] for i = n-1 down to 1: b[i-1] = a[i] + c * b[i] remainder = a[0] + c * b[0] return b, remainder

Tantangan Numerik pada Polinomial

Meskipun polinomial sederhana secara matematis, komputasi numeriknya memiliki tantangan tersendiri:

MasalahPenyebabSolusi
Ketidakstabilan evaluasi Pembulatan pada polinomial derajat tinggi Gunakan bentuk Horner
Kondisi buruk (ill-conditioned) Koefisien sangat kecil/besar Skalakan polinomial (normalisasi)
Kehilangan akar presisi Akar sangat berdekatan Gunakan metode Bairstow atau Jenkins-Traub
Akar kompleks Polinomial real dengan akar kompleks Gunakan metode Bairstow atau QR
"Polinomial derajat tinggi sangat sensitif terhadap perubahan kecil pada koefisien.
Ini dikenal sebagai fenomena 'Runge' atau 'ill-conditioning'."

Poin Penting Pendahuluan Polinomial

Polinomial adalah fondasi dari banyak algoritma dalam metode numerik karena kesederhanaan, fleksibilitas, dan sifat matematisnya yang baik.

Ringkasan konsep kunci:

1. Polinomial derajat n: P_n(x) = a_n x^n + ... + a₀
2. Kelebihan: mudah dievaluasi, diturunkan, diintegralkan, dan diaproksimasi
3. Representasi: standar, Newton, Lagrange, Horner (paling efisien)
4. Teorema Dasar Aljabar: polinomial derajat n memiliki n akar
5. Tantangan numerik: ill-conditioning, kehilangan presisi, akar kompleks
6. Aplikasi: interpolasi, aproksimasi, integrasi, diferensiasi, ODE

Pada subbab berikutnya, kita akan mempelajari secara mendalam tentang:

• Interpolasi Polinomial — Lagrange, Newton, dan spline
• Mencari Akar Polinomial — metode Bairstow, Jenkins-Traub
• Aproksimasi dengan Polinomial — least squares, Chebyshev
• Integrasi Numerik — Newton-Cotes, Gauss quadrature
