Apa Itu Interpolasi?

Interpolasi adalah teknik untuk menebak nilai di antara titik-titik data yang sudah diketahui.

Bayangkan kamu punya data seperti ini:

xf(x)
12
24
36

Lalu seseorang bertanya: "Berapa nilai f(2.5)?"

Karena 2.5 tidak ada di tabel, kamu tidak bisa membaca langsung. Di sinilah interpolasi bekerja — membantu kita memperkirakan nilai di antara data yang tersedia.

"Interpolasi = Menebak nilai di sela-sela data yang sudah diketahui"

Intuisi Sederhana

Coba bayangkan kamu memiliki beberapa titik di kertas gambar:

• Titik A di (1,2)
• Titik B di (2,4)
• Titik C di (3,6)

Jika kamu menghubungkan titik-titik itu dengan garis atau kurva, maka kamu bisa melihat kira-kira di mana letak titik untuk x = 2.5.

Ilustrasi konsep: ↑ 6 | ● (3,6) 5 | / 4 | ● (2,4) 3 | / 2 | ● (1,2) 1 | 0 +---+---+---+---→ 1 2 3 4 Titik (2.5, ?) berada di antara titik (2,4) dan (3,6)

Intuisi interpolasi: "Menggambar garis/kurva yang melewati titik-titik data, lalu membaca nilai di antaranya."

Contoh dalam Kehidupan Sehari-hari

Sebenarnya, tanpa sadar kita sering melakukan interpolasi!

Contoh 1: Suhu Tubuh
Jam 08.00 suhu 36.5°C, jam 12.00 suhu 37.5°C. Kira-kira jam 10.00 suhu berapa? 37.0°C (tebakan tengah-tengah). Ini adalah interpolasi linear.

Contoh 2: Ramalan Populasi
Tahun 2000 populasi 100 juta, tahun 2010 populasi 120 juta. Perkiraan populasi tahun 2005? Tentu sekitar 110 juta.

Contoh 3: Mengisi Warna di Gambar
Photoshop atau aplikasi edit foto menggunakan interpolasi saat kamu memperbesar gambar. Program menebak warna pixel baru berdasarkan pixel di sekitarnya.

"Interpolasi ada di mana-mana: suhu, populasi, warna gambar, dan masih banyak lagi."

Definisi Formal Interpolasi

Secara matematis, interpolasi adalah proses menemukan fungsi P(x) yang melewati semua titik data (x₀, y₀), (x₁, y₁), ..., (xₙ, yₙ), kemudian menggunakan fungsi tersebut untuk menghitung nilai pada x yang tidak diketahui.

Diketahui: (x₀, y₀), (x₁, y₁), ..., (xₙ, yₙ)
Dicari: P(x) sehingga P(xᵢ) = yᵢ untuk semua i

Jika x berada di dalam rentang data [x₀, xₙ], ini disebut interpolasi. Jika di luar rentang, disebut ekstrapolasi (lebih berisiko, hasilnya bisa meleset jauh).

Jenis-Jenis Interpolasi

Berikut adalah metode interpolasi yang paling umum digunakan:

MetodePenjelasan SederhanaKapan Dipakai?
Interpolasi Linear Menghubungkan dua titik dengan garis lurus Data hampir membentuk garis, butuh cepat
Interpolasi Polinomial Menggunakan kurva lengkung yang melewati semua titik Data agak melengkung, butuh presisi lebih
Spline Interpolation Kurva halus yang terdiri dari potongan-potongan Banyak titik data, butuh hasil yang sangat mulus
Interpolasi Lagrange Bentuk khusus interpolasi polinomial dengan rumus eksplisit
Untuk pembelajaran, atau titik data sedikit
Interpolasi Newton Bentuk interpolasi polinomial yang mudah ditambah titik baru Data ditambahkan secara bertahap

Contoh Interpolasi Linear (Paling Sederhana)

Misalkan kita punya data:

xf(x)
24
36

Ditanya: f(2.5) = ?

Penyelesaian dengan interpolasi linear:

\text{Rumus: } f(x) = f(x₀) + \frac{f(x₁) - f(x₀)}{x₁ - x₀} (x - x₀)
f(2.5) = 4 + \frac{6 - 4}{3 - 2} (2.5 - 2) = 4 + \frac{2}{1} \times 0.5 = 4 + 1 = 5

Hasil: f(2.5) = 5

# Kode Python sederhana def interpolasi_linear(x0, y0, x1, y1, x): return y0 + (y1 - y0) * (x - x0) / (x1 - x0) print(interpolasi_linear(2, 4, 3, 6, 2.5)) # Output: 5.0

Logikanya: Karena 2.5 tepat di tengah antara 2 dan 3, maka nilainya juga di tengah antara 4 dan 6, yaitu 5.

Perbedaan Interpolasi dan Ekstrapolasi

Ini adalah konsep penting yang sering membingungkan:

InterpolasiEkstrapolasi
Artinya Menebak nilai di dalam rentang data Menebak nilai di luar rentang data
Contoh Data 2000-2010 → tebak 2005 Data 2000-2010 → tebak 2020
Tingkat kepercayaan Cukup akurat (masih dalam jangkauan) Berisiko tinggi (bisa meleset jauh)
Analogi Tebak posisi di antara dua kota yang diketahui Tebak posisi di luar peta yang tidak diketahui
"Interpolasi di dalam zona nyaman, ekstrapolasi di luar kemampuan prediksi."

Kapan Kita Memakai Interpolasi?

Dalam dunia nyata, interpolasi digunakan saat:

✓ Data tidak lengkap
Misalnya sensor hanya merekam setiap 10 menit, tapi kita butuh data tiap 1 menit.

✓ Ingin menghaluskan grafik
Data hanya berupa titik-titik, interpolasi membantu menggambar kurva yang mulus.

✓ Butuh nilai di posisi tertentu
Tabel trigonometri (sin, cos, tan) panjang disimpan, interpolasi digunakan untuk nilai di antaranya.

✓ Machine Learning dan Visi Komputer
Memperbesar gambar (resampling) menggunakan interpolasi warna.

✓ Simulasi dan Grafika Komputer
Membuat gerakan halus antar frame (keyframe interpolation).

Aplikasi Interpolasi

Interpolasi digunakan di berbagai bidang, antara lain:

BidangAplikasi
Teknik Sipil Memperkirakan bentuk permukaan tanah dari titik-titik survei
Kedokteran Rekonstruksi gambar CT Scan atau MRI dari irisan tipis
Meteorologi Memperkirakan suhu di suatu daerah dari stasiun cuaca terdekat
Geofisika Memetakan struktur bawah tanah dari data seismik
Animasi Membuat gerakan halus antar pose karakter (keyframe interpolation)

Kesimpulan (Ringkasan untuk Pemula)

Interpolasi itu sederhana:

📌 Definisi: Teknik menebak nilai di antara titik-titik data yang diketahui.
📌 Basis data: Kita tahu titik A dan B, kita ingin tahu titik di antaranya.
📌 Interpolasi Linear: Cara paling sederhana — hubungkan dengan garis lurus.
📌 Interpolasi vs Ekstrapolasi: Interpolasi = di dalam data (aman), ekstrapolasi = di luar data (berisiko).
📌 Penggunaan sehari-hari: Memperkirakan suhu, populasi, warna gambar, gerakan animasi.
📌 Metode yang akan dipelajari: Linear, Polinomial (Lagrange, Newton), Spline.

Intinya: Interpolasi adalah "seni menebak secara matematis" yang sangat berguna dalam sains, teknik, dan kehidupan sehari-hari.

"Selanjutnya, kita akan mempelajari metode-metode interpolasi secara lebih mendalam:
Linear → Lagrange → Newton → Spline. Semakin canggih, semakin akurat!"
