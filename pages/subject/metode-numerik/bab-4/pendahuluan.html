Apa Itu Interpolasi?

Interpolasi adalah teknik untuk menebak nilai di antara titik-titik data yang sudah diketahui.

Bayangkan kamu punya data seperti ini:

xf(x)
12
24
36

Lalu seseorang bertanya: "Berapa nilai f(2.5)?"

Karena 2.5 tidak ada di tabel, kamu tidak bisa membaca langsung. Di sinilah interpolasi bekerja — membantu kita memperkirakan nilai di antara data yang tersedia.

"Interpolasi = Menebak nilai di sela-sela data yang sudah diketahui"

Intuisi Sederhana

Bayangkan beberapa titik di kertas:

• Titik A di (1,2)
• Titik B di (2,4)
• Titik C di (3,6)

Menghubungkan titik-titik itu dengan garis atau kurva memberi kita gambaran kira-kira di mana titik untuk x = 2.5.

Ilustrasi konsep: ↑ 6 | ● (3,6) 5 | / 4 | ● (2,4) 3 | / 2 | ● (1,2) 1 | 0 +---+---+---+---→ 1 2 3 4 Titik (2.5, ?) berada di antara titik (2,4) dan (3,6)

Intuisi interpolasi: Menggambar garis/kurva melalui titik data, lalu membaca nilai di antaranya.

Contoh Kehidupan Sehari-hari

Interpolasi sering kita lakukan tanpa sadar:

Suhu Tubuh: Jam 08.00 suhu 36.5°C, jam 12.00 suhu 37.5°C. Jam 10.00 kira-kira 37.0°C — interpolasi linear.

Ramalan Populasi: Tahun 2000 populasi 100 juta, 2010 populasi 120 juta. Tahun 2005 kira-kira 110 juta.

Mengisi Warna di Gambar: Saat memperbesar gambar, software menebak warna pixel baru dari pixel sekitarnya.

"Interpolasi ada di mana-mana: suhu, populasi, warna gambar, dan masih banyak lagi."

Definisi Formal Interpolasi

Secara matematis, interpolasi adalah proses menemukan fungsi P(x) yang melewati semua titik data (x₀, y₀), (x₁, y₁), ..., (xₙ, yₙ), lalu digunakan untuk menghitung nilai pada x yang tidak diketahui.

Diketahui: (x₀, y₀), (x₁, y₁), ..., (xₙ, yₙ)\\ Dicari: P(x) sehingga P(xᵢ) = yᵢ untuk semua i

Jika x berada di rentang data [x₀, xₙ], disebut interpolasi. Di luar rentang disebut ekstrapolasi.

Jenis-Jenis Interpolasi

Metode interpolasi yang umum digunakan:

MetodePenjelasan SederhanaKapan Dipakai?
Linear Hubungkan dua titik dengan garis lurus Data hampir linier, butuh cepat
Polinomial Gunakan kurva yang melewati semua titik Data melengkung, butuh presisi
Spline Kurva halus dari potongan-potongan Banyak titik, hasil mulus
Lagrange Polinomial dengan rumus eksplisit Belajar, titik data sedikit
Newton Polinomial yang mudah ditambah titik baru Data bertambah bertahap

Contoh Interpolasi Linear

Data:

xf(x)
24
36

Ditanya: f(2.5) = ?

\text{Rumus: } f(x) = f(x₀) + \frac{f(x₁)-f(x₀)}{x₁-x₀} (x-x₀)
f(2.5) = 4 + \frac{6-4}{3-2} (2.5-2) = 4 + 2 \cdot 0.5 = 5

Hasil: f(2.5) = 5

def interpolasi_linear(x0, y0, x1, y1, x): return y0 + (y1 - y0) * (x - x0) / (x1 - x0) print(interpolasi_linear(2, 4, 3, 6, 2.5)) # Output: 5.0

Interpolasi vs Ekstrapolasi

InterpolasiEkstrapolasi
Artinya Menebak nilai di dalam rentang data Menebak nilai di luar rentang data
Contoh Tahun 2000–2010 → tebak 2005 Tahun 2000–2010 → tebak 2020
Tingkat kepercayaan Cukup akurat Berisiko tinggi
Analogi Tebak posisi di antara dua kota Tebak posisi di luar peta
"Interpolasi di zona nyaman, ekstrapolasi di luar kemampuan prediksi."

Kapan Menggunakan Interpolasi?

Interpolasi berguna ketika:

  • Data tidak lengkap (sensor merekam setiap 10 menit, tapi butuh tiap 1 menit)
  • Ingin menghaluskan grafik dari titik-titik data
  • Butuh nilai di posisi tertentu (misal tabel trigonometri)
  • Machine Learning & Visi Komputer (resampling, memperbesar gambar)
  • Simulasi & Grafika Komputer (gerakan halus antar frame)

Kesimpulan

Interpolasi itu sederhana:

  • Definisi: Menebak nilai di antara titik-titik data yang diketahui
  • Basis data: Diketahui titik A dan B, ingin tahu titik di antaranya
  • Interpolasi Linear: Hubungkan titik dengan garis lurus
  • Interpolasi vs Ekstrapolasi: Di dalam data (aman), di luar data (berisiko)
  • Penggunaan sehari-hari: Suhu, populasi, warna gambar, animasi
  • Metode: Linear, Polinomial (Lagrange, Newton), Spline
"Selanjutnya: Linear → Lagrange → Newton → Spline. Semakin canggih, semakin akurat!"
