Metode Horner (Skema Horner)

Metode Horner (atau Skema Horner) adalah algoritma efisien untuk mengevaluasi polinomial. Ditemukan oleh matematikawan Inggris William George Horner pada tahun 1819, metode ini secara signifikan mengurangi jumlah operasi perkalian yang diperlukan.

Misalkan kita memiliki polinomial derajat n:

P(x) = a_n x^n + a_{n-1} x^{n-1} + ... + a_1 x + a_0

Metode Horner menulis ulang polinomial dalam bentuk bersarang (nested):

P(x) = a_0 + x(a_1 + x(a_2 + ... + x(a_{n-1} + a_n x)...))

Mengapa Metode Horner Lebih Efisien?

Perbandingan jumlah operasi antara evaluasi standar dan metode Horner:

MetodePerkalianPenjumlahanContoh (n=5)
Evaluasi Standar n(n+1)/2 n 15 perkalian, 5 penjumlahan
Metode Horner n n 5 perkalian, 5 penjumlahan

Keuntungan Metode Horner:

• Mengurangi perkalian dari O(n²) menjadi O(n)
• Lebih stabil secara numerik (mengurangi error pembulatan)
• Juga dapat digunakan untuk menghitung turunan polinomial
• Membantu dalam pembagian polinomial (sintetik)

"Horner: n perkalian vs n(n+1)/2 perkalian. Untuk n=100, dari 5050 menjadi 100 operasi!"

Bentuk Nested Polinomial

Ide dasar metode Horner adalah menulis polinomial dalam bentuk bersarang (nested):

Polinomial standar:

P(x) = a_4 x^4 + a_3 x^3 + a_2 x^2 + a_1 x + a_0

Bentuk nested (Horner):

P(x) = a_0 + x(a_1 + x(a_2 + x(a_3 + a_4 x)))

Proses evaluasi dimulai dari dalam ke luar:

# Evaluasi P(3) untuk P(x) = 2x³ - 5x² + 3x - 7 # Bentuk nested: P(x) = -7 + x(3 + x(-5 + x(2))) # Evaluasi dengan Horner: b₃ = 2 b₂ = -5 + 3·2 = -5 + 6 = 1 b₁ = 3 + 3·1 = 3 + 3 = 6 b₀ = -7 + 3·6 = -7 + 18 = 11 Hasil: P(3) = 11

Algoritma Metode Horner

Untuk polinomial P(x) = a_n x^n + a_{n-1} x^{n-1} + ... + a_0, algoritma Horner melakukan iterasi:

b_n = a_n
b_{n-1} = a_{n-1} + x \cdot b_n
b_{n-2} = a_{n-2} + x \cdot b_{n-1}
\vdots
b_0 = a_0 + x \cdot b_1

Hasil akhir b_0 = P(x).

# Algoritma Horner sederhana def horner(koefisien, x): # koefisien: [a_n, a_{n-1}, ..., a_0] hasil = koefisien[0] # mulai dari a_n for i in range(1, len(koefisien)): hasil = hasil * x + koefisien[i] return hasil # Contoh: P(x) = 2x³ - 5x² + 3x - 7 koef = [2, -5, 3, -7] print(horner(koef, 3)) # Output: 11

Contoh Perhitungan Manual

Hitung P(2) untuk polinomial:

P(x) = 3x^4 - 2x^3 + 5x^2 - 4x + 7

Langkah-langkah dengan skema Horner:

LangkahOperasiHasil
Inisialisasi b₄ = a₄ = 3 3
b₃ = a₃ + x·b₄ -2 + 2·3 = -2 + 6 4
b₂ = a₂ + x·b₃ 5 + 2·4 = 5 + 8 13
b₁ = a₁ + x·b₂ -4 + 2·13 = -4 + 26 22
b₀ = a₀ + x·b₁ 7 + 2·22 = 7 + 44 51

Hasil: P(2) = 51

# Verifikasi dengan metode standar # 3·16 - 2·8 + 5·4 - 4·2 + 7 = 48 - 16 + 20 - 8 + 7 = 51 ✓

Skema Tabel Horner

Metode Horner sering disajikan dalam bentuk tabel untuk memudahkan perhitungan manual:

Contoh: P(x) = 2x⁴ - 3x³ + 0x² + 5x - 8, evaluasi di x = 3 2 -3 0 5 -8 ↓ 3 → 6 9 27 96 --------------------------- 2 3 9 32 88 Cara membaca: - Baris pertama: koefisien polinomial (a₄, a₃, a₂, a₁, a₀) - Baris kedua: hasil perkalian (x × hasil sebelumnya) - Baris ketiga: hasil penjumlahan (bₙ, bₙ₋₁, ...) Penjelasan: b₄ = a₄ = 2 b₃ = a₃ + x·b₄ = -3 + 3·2 = 3 b₂ = a₂ + x·b₃ = 0 + 3·3 = 9 b₁ = a₁ + x·b₂ = 5 + 3·9 = 32 b₀ = a₀ + x·b₁ = -8 + 3·32 = 88 Hasil: P(3) = 88

Metode Horner untuk Menghitung Turunan

Kelebihan lain metode Horner: kita dapat menghitung P'(x) (turunan) secara bersamaan.

Dari skema Horner, koefisien b_i yang dihasilkan adalah koefisien polinomial hasil bagi ketika membagi P(x) dengan (x - c).

P(x) = (x - c) Q(x) + P(c)

dan P'(c) = Q(c).

Dengan kata lain, jika kita melakukan skema Horner sekali lagi pada koefisien b_i, kita mendapatkan P'(c).

# Menghitung P(3) dan P'(3) untuk P(x) = 2x³ - 5x² + 3x - 7 # Skema Horner pertama: koef = [2, -5, 3, -7] x = 3 b₂ = 2 b₁ = -5 + 3·2 = 1 b₀ = 3 + 3·1 = 6 sisa = -7 + 3·6 = 11 ← P(3) = 11 # Skema Horner kedua pada [b₂, b₁, b₀] = [2, 1, 6] c₁ = 2 c₀ = 1 + 3·2 = 7 sisa_turunan = 6 + 3·7 = 27 ← P'(3) = 27 Verifikasi: P'(x) = 6x² - 10x + 3 P'(3) = 6·9 - 30 + 3 = 54 - 30 + 3 = 27 ✓

Pembagian Polinomial (Pembagi Linear)

Metode Horner juga dikenal sebagai Pembagian Sintetik (Synthetic Division) untuk pembagi berbentuk (x - c).

Contoh: Bagi P(x) = 2x³ - 5x² + 3x - 7 dengan (x - 3).

Skema Horner: 2 -5 3 -7 3 → 6 3 18 -------------------- 2 1 6 11 Hasil bagi: Q(x) = 2x² + 1x + 6 Sisa: S = 11 Sehingga: P(x) = (x - 3)(2x² + x + 6) + 11

Interpretasi:

• Koefisien hasil bagi: [2, 1, 6] (satu derajat lebih rendah)
• Sisa pembagian: 11 (sama dengan P(3))
• Ini membuktikan teorema sisa: P(c) = sisa

AspekKelebihanKekurangan

Stabilitas numerik Mengurangi error pembulatan dibanding metode standar Masih bisa bermasalah untuk koefisien ekstrem
Fleksibilitas Dapat digunakan untuk evaluasi, turunan, dan pembagian Hanya untuk pembagi linear (x - c)
Kesederhanaan Algoritma sangat mudah diimplementasikan Perlu koefisien untuk semua derajat (termasuk yang nol)
Aplikasi luas Digunakan dalam berbagai algoritma numerik -
"Horner: Sederhana, efisien, dan stabil. Standar industri untuk evaluasi polinomial."

Kesimpulan dan Pseudocode Lengkap

Metode Horner adalah algoritma standar untuk evaluasi polinomial karena efisiensi dan stabilitas numeriknya.

Pseudoce lengkap:

PROCEDURE Horner(koef, x) INPUT: koef : array of koefisien [a_n, a_{n-1}, ..., a_0] x : nilai evaluasi OUTPUT: hasil : nilai P(x) n ← length(koef) - 1 hasil ← koef[0] {mulai dari a_n} FOR i = 1 TO n DO hasil ← hasil * x + koef[i] END FOR RETURN hasil END PROCEDURE PROCEDURE HornerWithDerivative(koef, x) INPUT: koef : array of koefisien [a_n, ..., a_0] x : nilai evaluasi OUTPUT: P_x : nilai P(x) P'_x : nilai P'(x) n ← length(koef) - 1 b ← array of size n+1 b[n] ← koef[n] FOR i = n-1 DOWN TO 0 DO b[i] ← koef[i] + x * b[i+1] END FOR P_x ← b[0] {Hitung P'(x) dari koefisien b} c ← array of size n c[n-1] ← b[n] FOR i = n-2 DOWN TO 0 DO c[i] ← b[i+1] + x * c[i+1] END FOR P'_x ← c[0] RETURN P_x, P'_x END PROCEDURE

Ringkasan poin penting:

1. Rumus: P(x) = a₀ + x(a₁ + x(a₂ + ... + a_n x)...)
2. Kompleksitas: O(n) (n perkalian + n penjumlahan)
3. Kelebihan: efisien, stabil, mudah diimplementasikan
4. Aplikasi: evaluasi polinomial, turunan, pembagian sintetik
5. Metode Horner adalah fondasi untuk algoritma numerik lainnya
