Metode Horner (Skema Horner)

Metode Horner (atau Skema Horner) adalah algoritma efisien untuk mengevaluasi polinomial. Ditemukan oleh William George Horner (1819), metode ini secara signifikan mengurangi jumlah operasi perkalian.

Misalkan polinomial derajat n:

P(x) = a_n x^n + a_{n-1} x^{n-1} + ... + a_1 x + a_0

Metode Horner menulis ulang polinomial dalam bentuk bersarang (nested):

P(x) = a_0 + x(a_1 + x(a_2 + ... + x(a_{n-1} + a_n x)...))

Mengapa Metode Horner Lebih Efisien?

Perbandingan jumlah operasi antara evaluasi standar dan metode Horner:

MetodePerkalianPenjumlahanContoh (n=5)
Evaluasi Standar n(n+1)/2 n 15 perkalian, 5 penjumlahan
Metode Horner n n 5 perkalian, 5 penjumlahan

Keuntungan Metode Horner:

  • Mengurangi perkalian dari O(n²) menjadi O(n)
  • Lebih stabil secara numerik (mengurangi error pembulatan)
  • Dapat digunakan untuk menghitung turunan polinomial
  • Membantu dalam pembagian polinomial (sintetik)
"Horner: n perkalian vs n(n+1)/2 perkalian. Contoh: n=100 → 5050 → 100 operasi!"

Bentuk Nested Polinomial

Polinomial standar:

P(x) = a_4 x^4 + a_3 x^3 + a_2 x^2 + a_1 x + a_0

Bentuk nested (Horner):

P(x) = a_0 + x(a_1 + x(a_2 + x(a_3 + a_4 x)))

Evaluasi dimulai dari dalam ke luar:

# Evaluasi P(3) untuk P(x) = 2x³ - 5x² + 3x - 7 P(x) = -7 + x(3 + x(-5 + x(2))) b₃ = 2 b₂ = -5 + 3*2 = 1 b₁ = 3 + 3*1 = 6 b₀ = -7 + 3*6 = 11 Hasil: P(3) = 11

Algoritma Metode Horner

Iterasi Horner untuk P(x) = a_n x^n + ... + a_0:

b_n = a_n
b_{n-1} = a_{n-1} + x·b_n
...
b_0 = a_0 + x·b_1

Hasil akhir: b_0 = P(x)

def horner(koefisien, x): hasil = koefisien[0] for i in range(1, len(koefisien)): hasil = hasil * x + koefisien[i] return hasil koef = [2, -5, 3, -7] print(horner(koef, 3)) # Output: 11

Hitung P(2) untuk:

P(x) = 3x^4 - 2x^3 + 5x^2 - 4x + 7

Langkah-langkah Horner:

LangkahOperasiHasil

b₃ = a₃ + x·b₄-2 + 2·3 = 44

b₁ = a₁ + x·b₂-4 + 2·13 = 2222
b₀ = a₀ + x·b₁7 + 2·22 = 5151

Hasil: P(2) = 51

# Verifikasi metode standar 3*16 - 2*8 + 5*4 - 4*2 + 7 = 51 ✓

Skema Tabel Horner

# P(x) = 2x⁴ - 3x³ + 0x² + 5x - 8, evaluasi x=3 Koefisien: 2 -3 0 5 -8 Proses: ↓ 3 → 6 9 27 96 ------------------- 2 3 9 32 88 Hasil: P(3) = 88

Metode Horner untuk Turunan

Koefisien dari skema Horner juga dapat digunakan untuk menghitung P'(x).

P(x) = (x - c) Q(x) + P(c), \quad P'(c) = Q(c)
# P(x) = 2x³ - 5x² + 3x - 7, evaluasi x=3 Koef = [2, -5, 3, -7] b₂=2, b₁=-5+3*2=1, b₀=3+3*1=6, P(3)=11 # Turunan dari [b₂, b₁, b₀] = [2,1,6] c₁=2, c₀=1+3*2=7, P'(3)=6+3*7=27 ✓

Pembagian Polinomial (Synthetic Division)

Bagi P(x) = 2x³ - 5x² + 3x - 7 dengan (x-3):

Koef: 2 -5 3 -7 3 → 6 3 18 ---------------- 2 1 6 11 Q(x) = 2x² + x + 6, Sisa = 11 P(x) = (x-3)(2x² + x + 6) + 11

Kesimpulan dan Pseudocode Lengkap

Metode Horner efisien, stabil, dan menjadi fondasi evaluasi polinomial.

PROCEDURE Horner(koef, x) hasil ← koef[0] FOR i = 1 TO length(koef)-1 hasil ← hasil * x + koef[i] END FOR RETURN hasil END PROCEDURE PROCEDURE HornerWithDerivative(koef, x) n ← length(koef)-1 b[n] ← koef[n] FOR i = n-1 DOWNTO 0 b[i] ← koef[i] + x*b[i+1] END FOR P_x ← b[0] c[n-1] ← b[n] FOR i = n-2 DOWNTO 0 c[i] ← b[i+1] + x*c[i+1] END FOR P'_x ← c[0] RETURN P_x, P'_x END PROCEDURE

Ringkasan: O(n), efisien, stabil, evaluasi, turunan, pembagian sintetik.
