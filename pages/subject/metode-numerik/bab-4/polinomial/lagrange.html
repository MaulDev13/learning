Interpolasi Polinomial - Bentuk Lagrange

Bentuk Lagrange (Lagrange Interpolation) adalah metode untuk membangun polinomial interpolasi yang mengekspresikan solusi secara langsung sebagai kombinasi linear dari nilai fungsi pada titik-titik data.

Ide utamanya adalah: daripada mencari koefisien polinomial, kita membangun polinomial basis yang bernilai 1 pada satu titik data dan 0 pada titik data lainnya.

P_n(x) = \sum_{i=0}^{n} y_i \cdot L_i(x)

dengan L_i(x) adalah polinomial Lagrange basis yang memenuhi L_i(x_j) = 1 jika i = j dan 0 jika i \neq j.

Mengapa Perlu Bentuk Lagrange?

Setiap bentuk interpolasi memiliki kelebihan masing-masing. Berikut perbandingan antara bentuk Lagrange, Newton, dan Standar:

Menambah titik data Perlu hitung ulang semua Tambah satu suku (O(n²)) Perlu hitung ulang semua basis
Evaluasi tunggal O(n) dengan Horner O(n) dengan nested O(n²) untuk hitung basis
Eksplisit formula Ya, dalam koefisien Ya, dalam beda terbagi Ya, langsung dalam y_i
Pembacaan hasil Perlu hitung koefisien Perlu hitung koefisien Langsung terlihat kontribusi tiap titik
Komputasi ulang basis Tidak ada basis Tidak ada basis Setiap x baru perlu hitung ulang L_i(x)
"Lagrange: Paling eksplisit secara matematis, tetapi paling mahal untuk evaluasi berulang."

Polinomial Basis Lagrange

Polinomial basis Lagrange L_i(x) untuk titik x_i didefinisikan sebagai:

L_i(x) = \prod_{\substack{j=0 \\ j \neq i}}^{n} \frac{x - x_j}{x_i - x_j}

Dengan kata lain:

L_i(x) = \frac{(x - x_0)(x - x_1)...(x - x_{i-1})(x - x_{i+1})...(x - x_n)}{(x_i - x_0)(x_i - x_1)...(x_i - x_{i-1})(x_i - x_{i+1})...(x_i - x_n)}

Sifat-sifat L_i(x):

• L_i(x_i) = 1 (pembilang = penyebut)
• L_i(x_j) = 0 untuk i \neq j (pembilang memiliki faktor (x_j - x_j) = 0)
• Derajat L_i(x) adalah n

Bentuk Umum Interpolasi Lagrange

Untuk n+1 titik data (x₀, y₀), (x₁, y₁), ..., (xₙ, yₙ), polinomial interpolasi Lagrange derajat n adalah:

P_n(x) = \sum_{i=0}^{n} y_i \cdot L_i(x) = y_0 L_0(x) + y_1 L_1(x) + ... + y_n L_n(x)

Untuk n = 1 (linear):

P_1(x) = y_0 \frac{x - x_1}{x_0 - x_1} + y_1 \frac{x - x_0}{x_1 - x_0}

Untuk n = 2 (kuadrat):

P_2(x) = y_0 \frac{(x - x_1)(x - x_2)}{(x_0 - x_1)(x_0 - x_2)} + y_1 \frac{(x - x_0)(x - x_2)}{(x_1 - x_0)(x_1 - x_2)} + y_2 \frac{(x - x_0)(x - x_1)}{(x_2 - x_0)(x_2 - x_1)}

Contoh: Interpolasi Lagrange Linear (n=1)

Diberikan dua titik data: (2, 5) dan (4, 9). Hitung nilai pada x = 3 dengan Lagrange.

Langkah 1: Tentukan polinomial basis

L_0(x) = \frac{x - x_1}{x_0 - x_1} = \frac{x - 4}{2 - 4} = \frac{x - 4}{-2} = -\frac{1}{2}(x - 4)
L_1(x) = \frac{x - x_0}{x_1 - x_0} = \frac{x - 2}{4 - 2} = \frac{x - 2}{2}

Langkah 2: Bentuk polinomial interpolasi

P_1(x) = y_0 L_0(x) + y_1 L_1(x) = 5 \cdot \left(-\frac{1}{2}(x - 4)\right) + 9 \cdot \left(\frac{x - 2}{2}\right)

Langkah 3: Evaluasi di x = 3

P_1(3) = 5 \cdot \left(-\frac{1}{2}(-1)\right) + 9 \cdot \left(\frac{1}{2}\right) = 5 \cdot \left(\frac{1}{2}\right) + 9 \cdot \left(\frac{1}{2}\right) = \frac{5}{2} + \frac{9}{2} = \frac{14}{2} = 7

Hasil: P(3) = 7 (sama dengan interpolasi linear biasa)

Contoh: Interpolasi Lagrange Kuadrat (n=2)

Diberikan tiga titik data: (1, 2), (2, 3), (4, 5). Hitung nilai pada x = 3.

Langkah 1: Hitung polinomial basis

L_0(x) = \frac{(x - x_1)(x - x_2)}{(x_0 - x_1)(x_0 - x_2)} = \frac{(x - 2)(x - 4)}{(1 - 2)(1 - 4)} = \frac{(x - 2)(x - 4)}{(-1)(-3)} = \frac{(x - 2)(x - 4)}{3}
L_1(x) = \frac{(x - x_0)(x - x_2)}{(x_1 - x_0)(x_1 - x_2)} = \frac{(x - 1)(x - 4)}{(2 - 1)(2 - 4)} = \frac{(x - 1)(x - 4)}{(1)(-2)} = -\frac{(x - 1)(x - 4)}{2}
L_2(x) = \frac{(x - x_0)(x - x_1)}{(x_2 - x_0)(x_2 - x_1)} = \frac{(x - 1)(x - 2)}{(4 - 1)(4 - 2)} = \frac{(x - 1)(x - 2)}{(3)(2)} = \frac{(x - 1)(x - 2)}{6}

Langkah 2: Evaluasi di x = 3

L_0(3) = \frac{(1)(-1)}{3} = -\frac{1}{3}
L_1(3) = -\frac{(2)(-1)}{2} = -\frac{-2}{2} = 1
L_2(3) = \frac{(2)(1)}{6} = \frac{2}{6} = \frac{1}{3}

Langkah 3: Hitung P(3)

P(3) = y_0 L_0(3) + y_1 L_1(3) + y_2 L_2(3) = 2 \cdot \left(-\frac{1}{3}\right) + 3 \cdot 1 + 5 \cdot \frac{1}{3} = -\frac{2}{3} + 3 + \frac{5}{3} = 3 + \frac{3}{3} = 4

Hasil: P(3) = 4

Contoh Aplikasi: Interpolasi Populasi

Data populasi suatu kota pada tahun tertentu:

Tahun (x)Populasi (y) dalam ribu
2000 100
2005 120
2010 150

Hitung perkiraan populasi tahun 2008 menggunakan interpolasi Lagrange.

x₀=2000, y₀=100; x₁=2005, y₁=120; x₂=2010, y₂=150; x=2008

L_0(2008) = \frac{(2008-2005)(2008-2010)}{(2000-2005)(2000-2010)} = \frac{(3)(-2)}{(-5)(-10)} = \frac{-6}{50} = -0.12
L_1(2008) = \frac{(2008-2000)(2008-2010)}{(2005-2000)(2005-2010)} = \frac{(8)(-2)}{(5)(-5)} = \frac{-16}{-25} = 0.64
L_2(2008) = \frac{(2008-2000)(2008-2005)}{(2010-2000)(2010-2005)} = \frac{(8)(3)}{(10)(5)} = \frac{24}{50} = 0.48
P(2008) = 100(-0.12) + 120(0.64) + 150(0.48) = -12 + 76.8 + 72 = 136.8

Perkiraan populasi tahun 2008 adalah 136.800 jiwa (sama dengan hasil metode Newton).

# Kode Python untuk verifikasi def lagrange_basis(x_data, i, x): result = 1.0 for j in range(len(x_data)): if j != i: result *= (x - x_data[j]) / (x_data[i] - x_data[j]) return result def lagrange_interpolation(x_data, y_data, x): total = 0.0 for i in range(len(x_data)): total += y_data[i] * lagrange_basis(x_data, i, x) return total x_data = [2000, 2005, 2010] y_data = [100, 120, 150] x = 2008 print(lagrange_interpolation(x_data, y_data, x)) # Output: 136.8

Algoritma Interpolasi Lagrange

Berikut prosedur sistematis untuk interpolasi Lagrange:

Langkah 1: Tentukan titik-titik data (x_i, y_i) untuk i = 0, 1, ..., n

Langkah 2: Untuk setiap i, hitung polinomial basis L_i(x)

Langkah 3: Hitung P(x) = Σ y_i · L_i(x)

PROCEDURE LagrangeInterpolation(x_data, y_data, x) INPUT: x_data : array of x_i (i = 0..n) y_data : array of y_i (i = 0..n) x : nilai yang akan diinterpolasi OUTPUT: y : nilai interpolasi n ← length(x_data) - 1 total ← 0.0 FOR i = 0 TO n DO {Hitung L_i(x)} Li ← 1.0 FOR j = 0 TO n DO IF j ≠ i THEN Li ← Li * (x - x_data[j]) / (x_data[i] - x_data[j]) END IF END FOR total ← total + y_data[i] * Li END FOR RETURN total END PROCEDURE

Kompleksitas waktu: O(n²) untuk setiap evaluasi karena loop bersarang.

Kelebihan dan Kekurangan Bentuk Lagrange

AspekKelebihanKekurangan
Kejelasan formula Rumus sangat eksplisit dan simetris Tidak efisien untuk n besar
Penambahan titik Tidak perlu hitung ulang koefisien Perlu hitung ulang semua basis dari nol
Evaluasi banyak x - Setiap x baru perlu O(n²) operasi
Stabilitas numerik Cukup stabil untuk n moderat Untuk n besar, rawan error pembulatan

"Lagrange: Hebat untuk dipelajari, tetapi Newton lebih praktis untuk implementasi."

Kesimpulan: Perbandingan Lagrange dengan Newton

Baik bentuk Lagrange maupun Newton menghasilkan polinomial interpolasi yang sama persis untuk himpunan titik data yang diberikan. Perbedaannya terletak pada representasi dan efisiensi komputasi.

AspekLagrangeNewton
Bentuk polinomial Kombinasi linear y_i · L_i(x) Kombinasi linear beda terbagi · basis Newton
Menambah titik data Perlu hitung ulang semua basis Tambah satu suku (lebih efisien)
Evaluasi satu x O(n²) O(n) setelah koefisien diperoleh
Evaluasi banyak x O(m·n²) untuk m titik O(n² + m·n) lebih efisien
Pembacaan hasil Setiap y_i berkontribusi langsung Perlu interpretasi beda terbagi

Ringkasan poin penting:

1. Polinomial Lagrange: P_n(x) = \sum y_i · L_i(x)
2. Basis Lagrange: L_i(x) = \prod (x - x_j)/(x_i - x_j) untuk j \neq i
3. Sifat: L_i(x_i) = 1 dan L_i(x_j) = 0 untuk i \neq j
4. Kelebihan: simetris, eksplisit, mudah dipahami konsepnya
5. Kekurangan: kompleksitas O(n²) per evaluasi, tidak efisien untuk n besar
6. Rekomendasi: gunakan Newton untuk implementasi, Lagrange untuk pembelajaran

"Lagrange: Elegansi matematika. Newton: Efisiensi komputasi.
Hasilnya sama, pilih sesuai kebutuhan."
