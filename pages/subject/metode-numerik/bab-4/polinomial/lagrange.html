Interpolasi Polinomial Lagrange

Interpolasi Lagrange adalah metode untuk membentuk polinomial yang melalui seluruh titik data yang diberikan.

Metode ini tidak memerlukan penyelesaian sistem persamaan, melainkan menggunakan rumus langsung.

Cocok digunakan untuk jumlah data yang tidak terlalu banyak.

Ide Dasar

Diberikan titik data: (x₀, y₀), (x₁, y₁), ..., (xₙ, yₙ)

Kita membentuk polinomial P(x) yang melewati semua titik tersebut.

Setiap titik memiliki fungsi basis tersendiri.

Rumus Lagrange

Bentuk umum:

P(x) = Σ (yᵢ · Lᵢ(x))

Dengan:

Lᵢ(x) = Π ( (x - xⱼ) / (xᵢ - xⱼ) ), j ≠ i

Lᵢ(x) disebut sebagai polinomial basis Lagrange.

Penjelasan Konsep

Setiap Lᵢ(x) bernilai 1 pada x = xᵢ dan 0 pada titik lainnya.

Hal ini memastikan bahwa polinomial melewati semua titik data.

Langkah-langkah

1. Tentukan titik data
2. Bentuk fungsi basis Lᵢ(x)
3. Kalikan dengan nilai yᵢ
4. Jumlahkan semua hasil
5. Sederhanakan polinomial

Contoh Perhitungan

Diketahui titik:

(1, 1), (2, 4), (3, 9)

Akan dibentuk polinomial interpolasi.

Ilustrasi Hasil

Dari titik tersebut diperoleh:

P(x) = x²

Polinomial ini melalui semua titik yang diberikan.

Kelebihan

• Tidak perlu menyelesaikan sistem persamaan
• Bentuk rumus langsung
• Mudah dipahami secara konsep

Kekurangan

• Perhitungan bisa panjang untuk banyak titik
• Tidak efisien jika data sering berubah
• Rentan terhadap error untuk derajat tinggi

Kesimpulan

Interpolasi Lagrange adalah metode efektif untuk membentuk polinomial dari data yang diketahui.

Metode ini sederhana, namun kurang efisien untuk jumlah data yang besar.
