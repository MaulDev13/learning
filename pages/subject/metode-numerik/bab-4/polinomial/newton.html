Interpolasi Polinomial - Bentuk Newton

Bentuk Newton (Newton's Divided Difference Interpolation) adalah metode untuk membangun polinomial interpolasi yang efisien ketika menambah titik data baru. Berbeda dengan bentuk standar yang harus menghitung ulang semua koefisien dari awal, bentuk Newton memungkinkan kita menambahkan titik baru tanpa mengganggu perhitungan sebelumnya.

Ide utamanya adalah merepresentasikan polinomial interpolasi dalam bentuk:

P_n(x) = a₀ + a₁(x - x₀) + a₂(x - x₀)(x - x₁) + ... + a_n(x - x₀)(x - x₁)...(x - x_{n-1})

Koefisien aₖ dihitung menggunakan beda terbagi (divided difference) dari titik-titik data yang diketahui.

Mengapa Perlu Bentuk Newton?

Dalam praktik interpolasi, kita sering tidak tahu berapa banyak titik data yang akan digunakan. Bisa jadi kita memulai dengan 3 titik, kemudian mendapatkan data ke-4, ke-5, dan seterusnya.

"Newton: Tambah titik baru, tambah suku baru. Tidak perlu hitung ulang dari nol."

Bentuk Umum Polinomial Newton

Polinomial interpolasi Newton derajat n ditulis sebagai:

P_n(x) = f[x₀] + f[x₀,x₁](x - x₀) + f[x₀,x₁,x₂](x - x₀)(x - x₁) + ... + f[x₀,x₁,...,x_n](x - x₀)(x - x₁)...(x - x_{n-1})

dengan notasi f[x₀, x₁, ..., x_k] adalah beda terbagi (divided difference) ke-k.

Untuk kasus 3 titik data (x₀, y₀), (x₁, y₁), (x₂, y₂):

P₂(x) = f[x₀] + f[x₀,x₁](x - x₀) + f[x₀,x₁,x₂](x - x₀)(x - x₁)

Jika ditambah titik ke-4 (x₃, y₃), cukup tambah satu suku baru:

P₃(x) = P₂(x) + f[x₀,x₁,x₂,x₃](x - x₀)(x - x₁)(x - x₂)

Beda Terbagi (Divided Difference)

Beda terbagi adalah alat untuk menghitung koefisien polinomial Newton secara rekursif.

Definisi rekursif:

f[x_i] = y_i (beda terbagi orde 0)
f[x_i, x_{i+1}] = ( f[x_{i+1}] - f[x_i] ) / ( x_{i+1} - x_i ) [orde 1]
f[x_i, x_{i+1}, x_{i+2}] = ( f[x_{i+1}, x_{i+2}] - f[x_i, x_{i+1}] ) / ( x_{i+2} - x_i ) [orde 2]

Secara umum, beda terbagi orde k:

f[x_i, ..., x_{i+k}] = ( f[x_{i+1}, ..., x_{i+k}] - f[x_i, ..., x_{i+k-1}] ) / ( x_{i+k} - x_i )

Tabel Beda Terbagi

Perhitungan beda terbagi disusun dalam bentuk tabel untuk memudahkan:

x₀ = 1, y₀ = 2 x₁ = 2, y₁ = 3 x₂ = 4, y₂ = 5 x₃ = 5, y₃ = 6 TABEL BEDA TERBAGI: x | f[x] (orde 0) | f[ , ] (orde 1) | f[ , , ] (orde 2) | f[ , , , ] (orde 3) -----|---------------|-----------------|-------------------|-------------------- 1 | 2 | | | | | (3-2)/(2-1)=1 | | 2 | 3 | | (1-1)/(4-1)=0 | | | (5-3)/(4-2)=1 | | (0-0)/(5-1)=0 4 | 5 | | (1-1)/(5-2)=0 | | | (6-5)/(5-4)=1 | | 5 | 6 | | | Koefisien Newton (ambil baris pertama setiap kolom): a₀ = f[x₀] = 2 a₁ = f[x₀,x₁] = 1 a₂ = f[x₀,x₁,x₂] = 0 a₃ = f[x₀,x₁,x₂,x₃] = 0

Koefisien polinomial Newton adalah entri pada baris pertama setiap kolom.

Contoh: Perhitungan Beda Terbagi

Diberikan titik data: (1, 2), (2, 3), (4, 5), (5, 6)

Buat tabel beda terbagi:

x₀=1, y₀=2 x₁=2, y₁=3 x₂=4, y₂=5 x₃=5, y₃=6 Orde 0: f[x₀]=2, f[x₁]=3, f[x₂]=5, f[x₃]=6 Orde 1: f[x₀,x₁] = (3-2)/(2-1) = 1/1 = 1 f[x₁,x₂] = (5-3)/(4-2) = 2/2 = 1 f[x₂,x₃] = (6-5)/(5-4) = 1/1 = 1 Orde 2: f[x₀,x₁,x₂] = (1 - 1)/(4-1) = 0/3 = 0 f[x₁,x₂,x₃] = (1 - 1)/(5-2) = 0/3 = 0 Orde 3: f[x₀,x₁,x₂,x₃] = (0 - 0)/(5-1) = 0/4 = 0

Koefisien Newton: a₀ = 2, a₁ = 1, a₂ = 0, a₃ = 0

Polinomial Newton:

P₃(x) = 2 + 1·(x-1) + 0·(x-1)(x-2) + 0·(x-1)(x-2)(x-4) = x + 1

Ternyata polinomialnya adalah garis lurus y = x + 1 karena titik-titik data memang terletak pada garis tersebut.

Contoh Aplikasi Interpolasi Newton

Diketahui data populasi suatu kota:

Tahun (x)Populasi (y) dalam ribu
2000 100
2005 120
2010 150

Hitung polinomial interpolasi Newton dan perkirakan populasi tahun 2008.

Langkah 1: Hitung beda terbagi

x₀=2000, y₀=100 x₁=2005, y₁=120 x₂=2010, y₂=150 Orde 1: f[x₀,x₁] = (120-100)/(2005-2000) = 20/5 = 4 f[x₁,x₂] = (150-120)/(2010-2005) = 30/5 = 6 Orde 2: f[x₀,x₁,x₂] = (6 - 4)/(2010-2000) = 2/10 = 0.2

Langkah 2: Polinomial Newton

P₂(x) = 100 + 4·(x - 2000) + 0.2·(x - 2000)(x - 2005)

Langkah 3: Evaluasi untuk x = 2008

P₂(2008) = 100 + 4·(8) + 0.2·(8)(3) = 100 + 32 + 0.2·24 = 100 + 32 + 4.8 = 136.8

Perkiraan populasi tahun 2008 adalah 136.800 jiwa.

Algoritma Interpolasi Newton

Berikut prosedur sistematis untuk interpolasi Newton:

Langkah 1: Hitung tabel beda terbagi
Buat matriks beda terbagi berukuran n×n.

Langkah 2: Ambil koefisien
Koefisien Newton adalah entri diagonal atas (baris pertama setiap kolom).

Langkah 3: Evaluasi polinomial
Gunakan metode nested (Horner) untuk mengevaluasi.

PROCEDURE NewtonInterpolation(x_data, y_data, x) INPUT: x_data : array of x_i (i = 0..n) y_data : array of y_i (i = 0..n) x : nilai yang akan diinterpolasi OUTPUT: y : nilai interpolasi n ← length(x_data) - 1 {Tabel beda terbagi} f ← matrix(n+1, n+1) FOR i = 0 TO n DO f[i][0] ← y_data[i] END FOR FOR j = 1 TO n DO FOR i = 0 TO n - j DO f[i][j] ← (f[i+1][j-1] - f[i][j-1]) / (x_data[i+j] - x_data[i]) END FOR END FOR {Evaluasi polinomial Newton (metode nested)} y ← f[0][n] FOR i = n-1 DOWN TO 0 DO y ← f[0][i] + (x - x_data[i]) * y END FOR RETURN y END PROCEDURE

Kesimpulan Interpolasi Bentuk Newton

Bentuk Newton adalah representasi polinomial interpolasi yang sangat berguna ketika kita perlu menambah titik data secara bertahap.

Ringkasan poin penting:

1. Polinomial Newton: P_n(x) = a₀ + a₁(x-x₀) + a₂(x-x₀)(x-x₁) + ...
2. Koefisien aₖ = f[x₀, x₁, ..., x_k] (beda terbagi orde k)
3. Beda terbagi dihitung rekursif: f[x_i,...,x_{i+k}] = (f[x_{i+1},...,x_{i+k}] - f[x_i,...,x_{i+k-1}]) / (x_{i+k} - x_i)
4. Kelebihan: menambah titik data tanpa hitung ulang, efisien
5. Kekurangan: perlu tabel beda terbagi, rawan error pembulatan
6. Aplikasi: interpolasi data bertahap, basis untuk metode numerik lain

Bentuk Newton menjadi fondasi penting sebelum mempelajari:

• Interpolasi Lagrange (bentuk eksplisit untuk evaluasi tunggal)
• Interpolasi Hermite (memasukkan turunan)
• Spline (interpolasi mulus untuk banyak titik)
