Interpolasi Linear

Interpolasi linear adalah metode paling sederhana dalam interpolasi numerik yang digunakan untuk memperkirakan nilai di antara dua titik data.

Metode ini mengasumsikan bahwa hubungan antara dua titik tersebut dapat didekati dengan garis lurus.

Cocok digunakan jika perubahan data bersifat hampir linear.

Ide Dasar

Diberikan dua titik: (x₀, y₀) dan (x₁, y₁)

Kita membentuk garis lurus yang menghubungkan kedua titik tersebut.

Nilai di antara x₀ dan x₁ dapat diperkirakan menggunakan persamaan garis tersebut.

Rumus Interpolasi Linear

Rumus umum:

f(x) = y₀ + ( (y₁ - y₀) / (x₁ - x₀) ) (x - x₀)

Atau dapat ditulis sebagai:

f(x) = y₀ * (x₁ - x)/(x₁ - x₀) + y₁ * (x - x₀)/(x₁ - x₀)

Langkah-langkah

1. Tentukan dua titik data terdekat
2. Gunakan rumus interpolasi linear
3. Substitusikan nilai x yang ingin dicari
4. Hitung hasilnya

Contoh Perhitungan

Diketahui:

(2, 4) dan (6, 8)

Cari nilai saat x = 4

Penyelesaian

f(4) = 4 + ((8 - 4)/(6 - 2)) (4 - 2)

f(4) = 4 + (4/4)(2)

f(4) = 4 + 2 = 6

Kelebihan

• Mudah dipahami dan dihitung
• Cepat digunakan
• Cocok untuk data sederhana

Kekurangan

• Kurang akurat untuk data yang melengkung
• Hanya menggunakan dua titik
• Tidak cocok untuk perubahan non linear

Kesimpulan

Interpolasi linear adalah metode sederhana untuk memperkirakan nilai di antara dua titik data.

Meskipun mudah digunakan, metode ini memiliki keterbatasan jika data tidak bersifat linear.
