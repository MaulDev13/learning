Interpolasi Linear

Interpolasi Linear adalah metode paling sederhana untuk memperkirakan nilai di antara dua titik data yang diketahui. Metode ini mengasumsikan bahwa perubahan antara dua titik bersifat linear (membentuk garis lurus).

Diberikan dua titik (x₀, y₀) dan (x₁, y₁), interpolasi linear menghubungkan kedua titik tersebut dengan garis lurus, kemudian menggunakan garis tersebut untuk mencari nilai y pada sembarang x di antara x₀ dan x₁.

Diketahui: (x₀, y₀) dan (x₁, y₁)
Dicari: y untuk suatu x di antara x₀ dan x₁

Rumus Interpolasi Linear

Persamaan garis yang melalui dua titik (x₀, y₀) dan (x₁, y₁) adalah:

y = y₀ + \frac{y₁ - y₀}{x₁ - x₀} (x - x₀)

Alternatif lain, dengan parameter t yang menunjukkan posisi relatif:

y = (1 - t) y₀ + t y₁, \quad \text{dengan } t = \frac{x - x₀}{x₁ - x₀}

Parameter t berada dalam rentang [0, 1]:

• t = 0 → x = x₀ → y = y₀
• t = 1 → x = x₁ → y = y₁
• t = 0.5 → x di tengah → y = (y₀ + y₁)/2

# Rumus interpolasi linear (dua bentuk) # Bentuk 1: langsung y = y0 + (y1 - y0) * (x - x0) / (x1 - x0) # Bentuk 2: dengan parameter t t = (x - x0) / (x1 - x0) y = (1 - t) * y0 + t * y1

Contoh 1: Interpolasi Suhu

Seorang peneliti mencatat suhu pada waktu tertentu:

Waktu (jam)Suhu (°C)
6:00 (x₀ = 6)20.0 (y₀)
10:00 (x₁ = 10)32.0 (y₁)

Perkirakan suhu pada pukul 8:00 (x = 8).

y = y₀ + \frac{y₁ - y₀}{x₁ - x₀} (x - x₀)
y = 20 + \frac{32 - 20}{10 - 6} (8 - 6)
y = 20 + \frac{12}{4} × 2
y = 20 + 6 = 26

Jadi, perkiraan suhu pada pukul 8:00 adalah 26°C.

Contoh 2: Interpolasi Tekanan Udara

Data tekanan udara pada ketinggian tertentu:

Ketinggian (km)Tekanan (kPa)
0 (x₀ = 0)101.3 (y₀)
2 (x₁ = 2)80.0 (y₁)

Perkirakan tekanan udara pada ketinggian 1.5 km (x = 1.5).

y = 101.3 + \frac{80.0 - 101.3}{2 - 0} (1.5 - 0)
y = 101.3 + \frac{-21.3}{2} × 1.5
y = 101.3 + (-10.65) × 1.5
y = 101.3 - 15.975 = 85.325

Tekanan udara pada ketinggian 1.5 km diperkirakan 85.33 kPa.

# Kode Python untuk interpolasi linear def interpolasi_linear(x0, y0, x1, y1, x): return y0 + (y1 - y0) * (x - x0) / (x1 - x0) # Contoh 1 suhu = interpolasi_linear(6, 20, 10, 32, 8) print(f"Suhu pada jam 8: {suhu}°C") # Output: 26.0°C # Contoh 2 tekanan = interpolasi_linear(0, 101.3, 2, 80.0, 1.5) print(f"Tekanan pada 1.5 km: {tekanan:.2f} kPa") # Output: 85.33 kPa

Algoritma Interpolasi Linear

Berikut prosedur sistematis untuk melakukan interpolasi linear pada data titik:

Langkah 1: Tentukan dua titik yang mengapit nilai x yang dicari, yaitu (x₀, y₀) dan (x₁, y₁) dengan x₀ ≤ x ≤ x₁.

Langkah 2: Hitung kemiringan (slope) garis: m = (y₁ - y₀) / (x₁ - x₀)

Langkah 3: Hitung nilai interpolasi: y = y₀ + m × (x - x₀)

PROCEDURE InterpolasiLinear(x0, y0, x1, y1, x) INPUT: x0, y0 : titik pertama x1, y1 : titik kedua x : nilai yang akan diinterpolasi OUTPUT: y : nilai interpolasi IF x < x0 OR x > x1 THEN RETURN "Peringatan: x di luar interval [x0, x1]" END IF m ← (y1 - y0) / (x1 - x0) y ← y0 + m * (x - x0) RETURN y END PROCEDURE

Interpolasi Linear pada Kumpulan Data

Jika kita memiliki lebih dari dua titik data, kita melakukan interpolasi per segmen (piecewise linear interpolation):

PROCEDURE InterpolasiPiecewiseLinear(Titik, x) INPUT: Titik : array of (x_i, y_i), i = 0..n x : nilai yang akan diinterpolasi OUTPUT: y : nilai interpolasi {Cari segmen yang mengandung x} FOR i = 0 TO n-1 DO IF x >= Titik[i].x AND x <= Titik[i+1].x THEN x0 ← Titik[i].x y0 ← Titik[i].y x1 ← Titik[i+1].x y1 ← Titik[i+1].y m ← (y1 - y0) / (x1 - x0) y ← y0 + m * (x - x0) RETURN y END IF END FOR RETURN "x di luar domain data" END PROCEDURE

Contoh Piecewise Linear

Diberikan data populasi suatu kota setiap 5 tahun:

Tahun (x)Populasi (ribu) (y)
2000100
2005125
2010160
2015190
2020210

Perkirakan populasi pada tahun 2008 dan 2018.

Untuk tahun 2008: berada di antara 2005 (x₀=2005, y₀=125) dan 2010 (x₁=2010, y₁=160)

y = 125 + \frac{160 - 125}{2010 - 2005} (2008 - 2005)
y = 125 + \frac{35}{5} × 3 = 125 + 21 = 146

Untuk tahun 2018: berada di antara 2015 (x₀=2015, y₀=190) dan 2020 (x₁=2020, y₁=210)

y = 190 + \frac{210 - 190}{2020 - 2015} (2018 - 2015)
y = 190 + \frac{20}{5} × 3 = 190 + 12 = 202

Hasil: populasi tahun 2008 ≈ 146.000 jiwa, tahun 2018 ≈ 202.000 jiwa.

Kelebihan dan Kekurangan Interpolasi Linear

AspekKelebihanKekurangan
Kesederhanaan Rumus sangat sederhana, mudah diimplementasikan Hasil berupa garis patah-patah (tidak mulus)
Kecepatan Komputasi sangat cepat, O(1) setelah ditemukan interval -
Akurasi Akurat jika fungsi hampir linear pada interval kecil Error besar jika fungsi sangat melengkung
Kontinuitas Fungsi hasil P(x) kontinu Turunan P'(x) tidak kontinu (patah di titik data)
Ekstrapolasi - Sangat tidak akurat untuk ekstrapolasi

Estimasi Error Interpolasi Linear

Misalkan f(x) adalah fungsi asli (tidak diketahui) dan P₁(x) adalah hasil interpolasi linear. Error pada suatu titik x di antara x₀ dan x₁ adalah:

E(x) = f(x) - P₁(x) = \frac{f''(\xi)}{2} (x - x₀)(x - x₁)

dengan ξ adalah suatu titik di antara x₀ dan x₁.

Batas maksimum error pada interval [x₀, x₁]:

|E(x)| \leq \frac{M}{8} (x₁ - x₀)², \quad \text{dengan } M = \max_{x₀ ≤ x ≤ x₁} |f''(x)|

Interpretasi:

• Error sebanding dengan (Δx)² — semakin rapat titik data, semakin kecil error
• Error sebanding dengan |f''(x)| — semakin melengkung fungsi, semakin besar error
• Error maksimum terjadi di tengah interval x = (x₀ + x₁)/2

Kesimpulan Interpolasi Linear

Interpolasi linear adalah metode paling fundamental dalam interpolasi numerik.

Ringkasan poin penting:

1. Rumus: y = y₀ + (y₁ - y₀)(x - x₀)/(x₁ - x₀)
2. Atau: y = (1 - t)y₀ + t y₁ dengan t = (x - x₀)/(x₁ - x₀)
3. Interpolasi linear mengasumsikan perubahan linear antara dua titik
4. Untuk banyak titik → interpolasi piecewise linear per segmen
5. Kelebihan: sederhana, cepat, mudah diimplementasikan
6. Kekurangan: hasil tidak mulus, error besar untuk fungsi melengkung
7. Estimasi error: |E(x)| ≤ M·(Δx)²/8

Interpolasi linear menjadi fondasi untuk mempelajari metode interpolasi yang lebih canggih seperti interpolasi kuadrat, Lagrange, Newton, dan spline — yang akan memberikan hasil lebih mulus dan akurat untuk fungsi yang kompleks.
