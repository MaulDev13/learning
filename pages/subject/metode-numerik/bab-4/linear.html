Interpolasi Linear

Interpolasi Linear adalah metode paling sederhana untuk memperkirakan nilai di antara dua titik data yang diketahui. Metode ini mengasumsikan bahwa perubahan antara dua titik bersifat linear (membentuk garis lurus).

Diberikan dua titik (x₀, y₀) dan (x₁, y₁), interpolasi linear menghubungkan kedua titik tersebut dengan garis lurus, kemudian menggunakan garis tersebut untuk mencari nilai y pada sembarang x di antara x₀ dan x₁.

Diketahui: (x₀, y₀) dan (x₁, y₁)
Dicari: y untuk suatu x di antara x₀ dan x₁

Rumus Interpolasi Linear

Persamaan garis yang melalui dua titik (x₀, y₀) dan (x₁, y₁) adalah:

y = y₀ + \frac{y₁ - y₀}{x₁ - x₀} (x - x₀)

Alternatif dengan parameter t yang menunjukkan posisi relatif:

y = (1 - t) y₀ + t y₁, \quad \text{dengan } t = \frac{x - x₀}{x₁ - x₀}

Parameter t berada dalam rentang [0,1]:

• t = 0 → x = x₀ → y = y₀
• t = 1 → x = x₁ → y = y₁
• t = 0.5 → x di tengah → y = (y₀ + y₁)/2

# Bentuk langsung y = y0 + (y1 - y0) * (x - x0) / (x1 - x0) # Bentuk parameter t t = (x - x0) / (x1 - x0) y = (1 - t) * y0 + t * y1

Contoh 1: Interpolasi Suhu

Data suhu:

Waktu (jam)Suhu (°C)
6:00 (x₀ = 6)20.0 (y₀)
10:00 (x₁ = 10)32.0 (y₁)

Perkirakan suhu pada pukul 8:00 (x=8)

y = 20 + \frac{32 - 20}{10 - 6} (8 - 6) = 20 + 6 = 26

Jadi, suhu ≈ 26°C

Contoh 2: Interpolasi Tekanan Udara

Data tekanan udara:

Ketinggian (km)Tekanan (kPa)
0 (x₀=0)101.3 (y₀)
2 (x₁=2)80.0 (y₁)

Perkirakan tekanan pada 1.5 km (x=1.5)

y = 101.3 + \frac{80 - 101.3}{2 - 0} (1.5 - 0) = 101.3 - 15.975 = 85.325

Tekanan ≈ 85.33 kPa

def interpolasi_linear(x0, y0, x1, y1, x): return y0 + (y1 - y0) * (x - x0) / (x1 - x0) # Contoh suhu = interpolasi_linear(6, 20, 10, 32, 8) print(suhu) # 26.0 tekanan = interpolasi_linear(0, 101.3, 2, 80.0, 1.5) print(tekanan) # 85.33

Algoritma Interpolasi Linear

Prosedur sistematis:

PROCEDURE InterpolasiLinear(x0, y0, x1, y1, x) IF x < x0 OR x > x1 THEN RETURN "x di luar interval" END IF m ← (y1 - y0) / (x1 - x0) y ← y0 + m * (x - x0) RETURN y END PROCEDURE

Piecewise Linear Interpolation

Jika ada banyak titik, lakukan per segmen:

PROCEDURE PiecewiseLinear(Titik[], x) FOR i = 0 TO n-1 DO IF x ≥ Titik[i].x AND x ≤ Titik[i+1].x THEN x0, y0 ← Titik[i].x, Titik[i].y x1, y1 ← Titik[i+1].x, Titik[i+1].y m ← (y1 - y0)/(x1 - x0) RETURN y0 + m*(x - x0) END IF END FOR RETURN "x di luar domain data" END PROCEDURE

Estimasi Error

E(x) = f(x) - P₁(x) = \frac{f''(\xi)}{2}(x-x₀)(x-x₁), \quad ξ \in [x₀,x₁]
|E(x)| ≤ \frac{M}{8} (x₁ - x₀)^2, \quad M = \max_{x₀ ≤ x ≤ x₁}|f''(x)|

Interpretasi:

  • Error ∝ (Δx)² — titik lebih rapat → error lebih kecil
  • Error ∝ |f''(x)| — fungsi lebih melengkung → error lebih besar
  • Error maksimum di tengah interval

Kesimpulan Interpolasi Linear

  • Rumus utama: y = y₀ + (y₁ - y₀)(x - x₀)/(x₁ - x₀)
  • Bentuk parameter: y = (1 - t)y₀ + t y₁, t=(x-x₀)/(x₁-x₀)
  • Asumsi linear antar titik
  • Banyak titik → Piecewise Linear per segmen
  • Kelebihan: sederhana, cepat, mudah
  • Kekurangan: tidak mulus, error besar untuk fungsi melengkung
  • Estimasi error: |E(x)| ≤ M·(Δx)²/8
