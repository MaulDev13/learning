Implementasi SPL di Python

Sistem Persamaan Linear (SPL) adalah kumpulan persamaan linear yang memiliki beberapa variabel.

Dalam metode numerik, SPL diselesaikan menggunakan pendekatan algoritma seperti eliminasi Gauss, Gauss-Jordan, dan metode matriks.

Konsep Dasar SPL

Bentuk umum SPL:

Ax = b

Dengan: A = matriks koefisien
x = vektor variabel
b = vektor hasil

Metode Eliminasi Gauss

Metode ini mengubah matriks menjadi bentuk segitiga atas untuk mempermudah penyelesaian.

Implementasi Python: 

import numpy as np

A = np.array([
    [2, 1, -1],
    [-3, -1, 2],
    [-2, 1, 2]
], dtype=float)

b = np.array([8, -11, -3], dtype=float)

n = len(b)

# Eliminasi maju
for i in range(n):
    for j in range(i+1, n):
        ratio = A[j][i] / A[i][i]
        A[j] = A[j] - ratio * A[i]
        b[j] = b[j] - ratio * b[i]

# Substitusi balik
x = np.zeros(n)

for i in range(n-1, -1, -1):
    x[i] = (b[i] - np.dot(A[i][i+1:], x[i+1:])) / A[i][i]

print(x)

Metode Gauss-Jordan

Metode ini mengubah matriks menjadi bentuk identitas.

Implementasi Python: 

import numpy as np

A = np.array([
    [2, 1, -1],
    [-3, -1, 2],
    [-2, 1, 2]
], dtype=float)

b = np.array([8, -11, -3], dtype=float)

n = len(b)

# Eliminasi
for i in range(n):
    A[i] = A[i] / A[i][i]
    b[i] = b[i] / A[i][i]

    for j in range(n):
        if i != j:
            ratio = A[j][i]
            A[j] = A[j] - ratio * A[i]
            b[j] = b[j] - ratio * b[i]

print(b)

Metode Invers Matriks

Penyelesaian SPL dengan invers:

x = A⁻¹ b

Implementasi Python: 

import numpy as np

A = np.array([
    [2, 1, -1],
    [-3, -1, 2],
    [-2, 1, 2]
], dtype=float)

b = np.array([8, -11, -3], dtype=float)

x = np.linalg.inv(A).dot(b)

print(x)

Metode Langsung (NumPy Solve)

Cara paling efisien di Python menggunakan library NumPy.

Implementasi: 

import numpy as np

A = np.array([
    [2, 1, -1],
    [-3, -1, 2],
    [-2, 1, 2]
], dtype=float)

b = np.array([8, -11, -3], dtype=float)

x = np.linalg.solve(A, b)

print(x)

Perbandingan Metode

• Gauss → manual, dasar konsep
• Gauss-Jordan → hasil langsung
• Invers → konsep matematis
• NumPy Solve → paling cepat dan praktis

Kesimpulan

Implementasi SPL di Python dapat dilakukan dengan berbagai metode, mulai dari algoritma manual hingga fungsi bawaan NumPy.

Pemilihan metode tergantung pada kebutuhan: pemahaman konsep atau efisiensi komputasi.
