Implementasi Interpolasi Numerik di Python

Interpolasi numerik adalah metode untuk memperkirakan nilai fungsi di antara titik-titik data yang sudah diketahui.

Dalam Python, interpolasi dapat diimplementasikan menggunakan perhitungan manual atau library seperti NumPy dan SciPy.

Di sini kita fokus pada implementasi dasar secara manual.

Interpolasi Linier

Rumus: y = y₀ + (x - x₀)(y₁ - y₀) / (x₁ - x₀)

Implementasi Python: 

x0, y0 = 1, 2
x1, y1 = 3, 6

x = 2

y = y0 + (x - x0) * (y1 - y0) / (x1 - x0)

print(y)

Interpolasi Lagrange

Rumus umum: P(x) = Σ yi * Li(x)

Implementasi Python: 

def lagrange(x, x_points, y_points):
    n = len(x_points)
    result = 0

    for i in range(n):
        term = y_points[i]
        for j in range(n):
            if i != j:
                term *= (x - x_points[j]) / (x_points[i] - x_points[j])
        result += term

    return result

x_points = [1, 2, 3]
y_points = [2, 4, 6]

x = 2.5

print(lagrange(x, x_points, y_points))

Interpolasi Newton (Divided Difference)

Metode Newton menggunakan tabel beda terbagi.

Implementasi Python sederhana: 

def newton_interpolation(x, x_points, y_points):
    n = len(x_points)
    coef = y_points.copy()

    for j in range(1, n):
        for i in range(n-1, j-1, -1):
            coef[i] = (coef[i] - coef[i-1]) / (x_points[i] - x_points[i-j])

    result = coef[0]
    product = 1

    for i in range(1, n):
        product *= (x - x_points[i-1])
        result += coef[i] * product

    return result

x_points = [1, 2, 3]
y_points = [2, 4, 6]

x = 2.5

print(newton_interpolation(x, x_points, y_points))

Konsep Implementasi

Semua metode interpolasi pada dasarnya melakukan:

• Menggunakan data titik yang diketahui
• Membentuk fungsi pendekatan
• Menghitung nilai di titik yang belum diketahui

Python memudahkan proses ini dengan loop dan fungsi matematis.

Perbandingan Metode

• Linier → paling sederhana, cepat, kurang akurat
• Lagrange → fleksibel, cocok untuk semua titik
• Newton → efisien untuk penambahan data baru

Kesimpulan

Implementasi interpolasi di Python memungkinkan kita untuk memperkirakan nilai fungsi dengan berbagai metode numerik.

Setiap metode memiliki kelebihan masing-masing tergantung jumlah data dan kebutuhan akurasi.
