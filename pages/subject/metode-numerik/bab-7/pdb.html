Implementasi PDB di Python

Persamaan Diferensial Biasa (PDB) dapat diselesaikan secara numerik menggunakan metode iteratif dalam Python.

Tujuannya adalah menghitung pendekatan nilai fungsi y(x) pada titik-titik tertentu ketika solusi eksak sulit diperoleh.

Konsep Implementasi

Semua metode PDB numerik memiliki pola umum:

• Diketahui: dy/dx = f(x, y)
• Diketahui kondisi awal y(x₀) = y₀
• Gunakan step size h
• Lakukan iterasi untuk mencari nilai berikutnya

Metode Euler

Rumus: yₙ₊₁ = yₙ + h f(xₙ, yₙ)

Implementasi Python: 

def f(x, y):
    return x + y

x = 0
y = 1
h = 0.1

for i in range(10):
    y = y + h * f(x, y)
    x = x + h
    print(x, y)

Metode Heun (Improved Euler)

Menggunakan rata-rata slope awal dan prediksi.

Implementasi Python: 

def f(x, y):
    return x + y

x = 0
y = 1
h = 0.1

for i in range(10):
    k1 = f(x, y)
    y_predict = y + h * k1
    k2 = f(x + h, y_predict)

    y = y + (h/2) * (k1 + k2)
    x = x + h

    print(x, y)

Metode Runge-Kutta Orde 2 (RK2)

Menggunakan dua evaluasi slope untuk meningkatkan akurasi.

Implementasi Python: 

def f(x, y):
    return x + y

x = 0
y = 1
h = 0.1

for i in range(10):
    k1 = f(x, y)
    k2 = f(x + h, y + h * k1)

    y = y + h * k2
    x = x + h

    print(x, y)

Metode Runge-Kutta Orde 4 (RK4)

Metode paling umum karena akurasi tinggi.

Implementasi Python: 

def f(x, y):
    return x + y

x = 0
y = 1
h = 0.1

for i in range(10):
    k1 = f(x, y)
    k2 = f(x + h/2, y + h*k1/2)
    k3 = f(x + h/2, y + h*k2/2)
    k4 = f(x + h, y + h*k3)

    y = y + (h/6)*(k1 + 2*k2 + 2*k3 + k4)
    x = x + h

    print(x, y)

Interpretasi Numerik

Semua metode PDB numerik bekerja dengan cara:

• Menggunakan slope (turunan) untuk prediksi
• Melakukan iterasi langkah demi langkah
• Mendekati kurva solusi sebenarnya

Perbandingan Metode

• Euler → sederhana tapi kurang akurat
• Heun → lebih baik dari Euler
• RK2 → cukup akurat dan stabil
• RK4 → paling akurat dan umum digunakan

Kesimpulan

Implementasi PDB di Python dilakukan dengan metode iteratif yang menggunakan nilai turunan untuk memperkirakan solusi fungsi.

Semakin tinggi orde metode (misalnya RK4), semakin akurat hasil yang diperoleh.
