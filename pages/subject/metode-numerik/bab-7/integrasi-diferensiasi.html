Implementasi Integrasi & Diferensiasi di Python

Dalam metode numerik, Python digunakan untuk mengimplementasikan pendekatan integrasi numerik dan diferensiasi numerik secara komputasi.

Tujuannya adalah mengubah rumus matematika menjadi algoritma yang dapat dihitung secara otomatis.

Biasanya digunakan perulangan dan fungsi untuk melakukan perhitungan bertahap.

Implementasi Diferensiasi Numerik

Contoh: Diferensiasi pusat (central difference)

Rumus: f'(x) ≈ (f(x+h) - f(x-h)) / (2h)

Implementasi Python: 

def f(x):
    return x**2

x = 2
h = 0.1

df = (f(x + h) - f(x - h)) / (2 * h)
print(df)

Diferensiasi Maju (Forward Difference)

Rumus: f'(x) ≈ (f(x+h) - f(x)) / h

Implementasi: 

df = (f(x + h) - f(x)) / h
print(df)

Diferensiasi Mundur (Backward Difference)

Rumus: f'(x) ≈ (f(x) - f(x-h)) / h

Implementasi: 

df = (f(x) - f(x - h)) / h
print(df)

Integrasi Trapesium

Rumus: ∫ f(x) dx ≈ (h/2)[f(x0) + f(xn) + 2Σf(xi)]

Implementasi Python: 

def f(x):
    return x**2

a = 0
b = 2
n = 4

h = (b - a) / n

x = a
total = f(a) + f(b)

for i in range(1, n):
    x = a + i * h
    total += 2 * f(x)

integral = (h / 2) * total
print(integral)

Simpson 1/3

Implementasi dasar Simpson 1/3: 

n = 4  # harus genap
h = (b - a) / n

total = f(a) + f(b)

for i in range(1, n):
    x = a + i * h
    if i % 2 == 0:
        total += 2 * f(x)
    else:
        total += 4 * f(x)

integral = (h / 3) * total
print(integral)

Simpson 3/8

Implementasi dasar Simpson 3/8: 

n = 3  # kelipatan 3
h = (b - a) / n

total = f(a) + f(b)

for i in range(1, n):
    x = a + i * h
    if i % 3 == 0:
        total += 2 * f(x)
    else:
        total += 3 * f(x)

integral = (3 * h / 8) * total
print(integral)

Kesimpulan

Implementasi integrasi dan diferensiasi di Python memanfaatkan fungsi, perulangan, dan logika kondisional untuk merealisasikan rumus metode numerik.

Dengan pendekatan ini, perhitungan matematika kompleks dapat diselesaikan secara otomatis dan efisien.
