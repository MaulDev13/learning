Implementasi Metode Akar di Python

Metode akar (root finding) adalah metode numerik untuk mencari solusi dari persamaan non-linear berbentuk f(x) = 0.

Dalam Python, metode ini diimplementasikan menggunakan iterasi hingga nilai akar mendekati nol dalam batas toleransi tertentu.

Metode Bisection

Metode ini membagi interval menjadi dua bagian dan memilih sub-interval yang mengandung akar.

Syarat: f(a) dan f(b) harus berbeda tanda.

Implementasi Python: 

def f(x):
    return x**2 - 4

a = 0
b = 3
tol = 1e-6

while (b - a) / 2 > tol:
    c = (a + b) / 2

    if f(c) == 0:
        break
    elif f(a) * f(c) < 0:
        b = c
    else:
        a = c

print((a + b) / 2)

Metode Newton-Raphson

Metode ini menggunakan turunan untuk mempercepat pencarian akar.

Rumus: xₙ₊₁ = xₙ - f(x) / f'(x)

Implementasi Python: 

def f(x):
    return x**2 - 4

def df(x):
    return 2*x

x = 3
tol = 1e-6

while abs(f(x)) > tol:
    x = x - f(x) / df(x)

print(x)

Metode Secant

Metode secant adalah pengembangan Newton-Raphson tanpa turunan eksplisit.

Rumus: xₙ₊₁ = xₙ - f(xₙ)(xₙ - xₙ₋₁) / (f(xₙ) - f(xₙ₋₁))

Implementasi Python: 

def f(x):
    return x**2 - 4

x0 = 1
x1 = 3
tol = 1e-6

while abs(x1 - x0) > tol:
    x2 = x1 - f(x1) * (x1 - x0) / (f(x1) - f(x0))
    x0 = x1
    x1 = x2

print(x1)

Metode Fixed Point Iteration

Mengubah f(x)=0 menjadi bentuk x = g(x).

Implementasi Python: 

def g(x):
    return (x + 4/x) / 2

x = 3
tol = 1e-6

for i in range(100):
    x_new = g(x)
    if abs(x_new - x) < tol:
        break
    x = x_new

print(x)

Konsep Umum Implementasi

Semua metode akar di Python memiliki pola yang sama:

• Tentukan fungsi f(x)
• Tentukan tebakan awal atau interval
• Lakukan iterasi
• Hentikan jika error < toleransi

Perbandingan Metode

• Bisection → paling stabil, tapi lambat
• Newton-Raphson → cepat, butuh turunan
• Secant → tanpa turunan, cukup cepat
• Fixed Point → sederhana, tergantung konvergensi

Kesimpulan

Implementasi metode akar di Python menggunakan pendekatan iteratif untuk menemukan solusi f(x)=0 secara numerik.

Pemilihan metode tergantung pada kebutuhan stabilitas, kecepatan, dan ketersediaan turunan.
