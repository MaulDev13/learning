Dasar Python dalam Metode Numerik

Python adalah bahasa pemrograman yang banyak digunakan dalam komputasi numerik karena sintaksnya sederhana dan memiliki banyak pustaka ilmiah.

Dalam metode numerik, Python digunakan untuk mengimplementasikan algoritma seperti pencarian akar, SPL, interpolasi, integrasi, dan PDB.

Fokus utama adalah menerjemahkan rumus matematika menjadi kode program.

Mengapa Python?

Python dipilih dalam metode numerik karena:

• Sintaks sederhana dan mudah dibaca
• Banyak library ilmiah (NumPy, SciPy, Matplotlib)
• Cocok untuk komputasi numerik dan visualisasi
• Cepat dalam prototyping algoritma

Konsep Dasar Python

Beberapa konsep dasar yang penting:

• Variabel → tempat menyimpan nilai
• Tipe data → int, float, string, list
• Percabangan → if, else
• Perulangan → for, while
• Fungsi → blok kode yang dapat dipakai ulang

Variabel dan Tipe Data

Contoh sederhana:

x = 5
y = 2.5
nama = "Numerik"

Python secara otomatis menentukan tipe data variabel.

Operasi Dasar

Operasi matematika dalam Python:

• Penjumlahan: +
• Pengurangan: -
• Perkalian: *
• Pembagian: /
• Pangkat: **

Contoh: 2 ** 3 = 8

Struktur Kontrol

Contoh if statement:

if x > 0:
  print("Positif")
else:
  print("Negatif")

Digunakan untuk pengambilan keputusan dalam algoritma numerik.

Perulangan

Contoh for loop:

for i in range(5):
  print(i)

Digunakan dalam iterasi metode numerik seperti Euler atau RK.

Fungsi dalam Python

Contoh fungsi:

def f(x):
  return x**2

Fungsi sangat penting dalam implementasi metode numerik karena merepresentasikan persamaan matematika.

Contoh Sederhana Numerik

Menghitung nilai fungsi:

def f(x):
  return x + 1

print(f(2)) → 3

Kesimpulan

Dasar Python adalah fondasi penting untuk implementasi metode numerik.

Dengan memahami variabel, fungsi, dan perulangan, kita dapat mengubah rumus matematika menjadi algoritma komputasi.
