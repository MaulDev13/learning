SPNL - Metode Secant

Metode Secant adalah metode numerik untuk menyelesaikan sistem persamaan non linear yang merupakan alternatif dari Newton-Raphson tanpa memerlukan perhitungan turunan (derivatif).

Ide utamanya adalah mengganti turunan f'(x) pada metode Newton dengan pendekatan beda hingga (finite difference) menggunakan dua iterasi sebelumnya. Dengan cara ini, metode Secant tetap memiliki konvergensi yang cepat meskipun tidak secepat Newton-Raphson.

Newton-Raphson: x_{k+1} = x_k - f(x_k) / f'(x_k)
Secant: x_{k+1} = x_k - f(x_k) · (x_k - x_{k-1}) / (f(x_k) - f(x_{k-1}))

Penurunan Rumus (Kasus Satu Variabel)

Metode Newton-Raphson memerlukan turunan f'(x):

x_{k+1} = x_k - \frac{f(x_k)}{f'(x_k)}

Jika turunan tidak tersedia atau sulit dihitung, kita dapat mengaproksimasi turunan dengan beda hingga:

f'(x_k) \approx \frac{f(x_k) - f(x_{k-1})}{x_k - x_{k-1}}

Substitusi ke rumus Newton menghasilkan:

x_{k+1} = x_k - f(x_k) \cdot \frac{x_k - x_{k-1}}{f(x_k) - f(x_{k-1})}

Inilah rumus iterasi Secant untuk satu variabel. Metode ini membutuhkan dua tebakan awal x₀ dan x₁.

# Rumus Secant (1 variabel) x_{k+1} = x_k - f(x_k) * (x_k - x_{k-1}) / (f(x_k) - f(x_{k-1}))

Interpretasi Geometris

Secara geometris, metode Secant menggantikan garis singgung pada Newton-Raphson dengan garis secant yang melalui dua titik terakhir.

(x_{k-1}, f(x_{k-1})) \text{ dan } (x_k, f(x_k))

Persamaan garis secant:

y - f(x_k) = \frac{f(x_k) - f(x_{k-1})}{x_k - x_{k-1}} (x - x_k)

Titik potong dengan sumbu-x (y = 0) menghasilkan iterasi Secant.

"Secant = Newton tanpa turunan, menggunakan dua titik terakhir."

Secant untuk SPNL: Metode Broyden

Untuk sistem multivariabel, konsep Secant digeneralisasi menjadi Metode Broyden.

Newton: X_{k+1} = X_k - J_k^{-1} · F(X_k)
Broyden: X_{k+1} = X_k - B_k^{-1} · F(X_k)

Matriks aproksimasi Jacobian diperbarui dengan prinsip secant:

B_{k+1} = B_k + \frac{(ΔF_k - B_k ΔX_k) ΔX_k^T}{ΔX_k^T ΔX_k}

Langkah-langkah Metode Secant

Langkah 1: Tentukan dua tebakan awal x₀ dan x₁
Langkah 2: Hitung f(x₀) dan f(x₁)
Langkah 3: Gunakan rumus Secant
Langkah 4: Geser nilai iterasi
Langkah 5: Cek konvergensi

x_prev = x0 x_curr = x1 for k = 1 to max_iter: x_next = x_curr - f(x_curr)*(x_curr - x_prev)/(f(x_curr)-f(x_prev)) if |x_next - x_curr| < tol: stop x_prev = x_curr x_curr = x_next

Pseudocode Metode Secant

PROCEDURE Secant(f, x0, x1, tol, max_iter) x_prev ← x0 x_curr ← x1 FOR k = 1 TO max_iter DO IF |f(x_curr) - f(x_prev)| < eps THEN RETURN "divergen / pembagi nol" END IF x_next ← x_curr - f(x_curr)*(x_curr - x_prev)/(f(x_curr)-f(x_prev)) IF |x_next - x_curr| < tol THEN RETURN x_next END IF x_prev ← x_curr x_curr ← x_next END FOR RETURN x_curr END PROCEDURE

Poin Penting

1. Tidak membutuhkan turunan
2. Memerlukan dua tebakan awal
3. Konvergensi superlinear
4. Lebih cepat dari Fixed Point
5. Lebih murah dari Newton-Raphson

Metode Secant = pendekatan praktis Newton tanpa derivatif.
