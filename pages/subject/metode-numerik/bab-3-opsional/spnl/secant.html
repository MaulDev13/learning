Metode Secant

Metode Secant adalah metode numerik untuk menyelesaikan sistem persamaan non linear yang merupakan alternatif dari Newton-Raphson tanpa memerlukan perhitungan turunan (derivatif).

Ide utamanya adalah mengganti turunan f'(x) pada metode Newton dengan pendekatan beda hingga (finite difference) menggunakan dua iterasi sebelumnya. Dengan cara ini, metode Secant tetap memiliki konvergensi yang cepat meskipun tidak secepat Newton-Raphson.

Newton-Raphson: x_{k+1} = x_k - f(x_k) / f'(x_k)
Secant: x_{k+1} = x_k - f(x_k) · (x_k - x_{k-1}) / (f(x_k) - f(x_{k-1}))

Penurunan Rumus (Kasus Satu Variabel)

Metode Newton-Raphson memerlukan turunan f'(x):

x_{k+1} = x_k - \frac{f(x_k)}{f'(x_k)}

Jika turunan tidak tersedia atau sulit dihitung, kita dapat mengaproksimasi turunan dengan beda hingga mundur:

f'(x_k) \approx \frac{f(x_k) - f(x_{k-1})}{x_k - x_{k-1}}

Substitusi aproksimasi ini ke rumus Newton:

x_{k+1} = x_k - f(x_k) \cdot \frac{x_k - x_{k-1}}{f(x_k) - f(x_{k-1})}

Inilah rumus iterasi Secant untuk satu variabel. Perhatikan bahwa metode ini memerlukan dua tebakan awal x₀ dan x₁.

# Rumus Secant (1 variabel) x_{k+1} = x_k - f(x_k) * (x_k - x_{k-1}) / (f(x_k) - f(x_{k-1}))

Interpretasi Geometris

Secara geometris, metode Secant menggantikan garis singgung (turunan) pada Newton-Raphson dengan garis secant yang melalui dua titik terakhir (x_{k-1}, f(x_{k-1})) dan (x_k, f(x_k)).

Garis secant: y - f(x_k) = \frac{f(x_k) - f(x_{k-1})}{x_k - x_{k-1}} (x - x_k)

Akar persamaan f(x) = 0 adalah perpotongan garis secant dengan sumbu x (y = 0):

0 - f(x_k) = \frac{f(x_k) - f(x_{k-1})}{x_k - x_{k-1}} (x_{k+1} - x_k)

Menyelesaikan untuk x_{k+1} menghasilkan rumus Secant yang sama.

"Secant = Newton tanpa turunan, menggunakan dua titik terakhir
untuk mengaproksimasi kemiringan kurva."

Secant untuk SPNL: Metode Broyden

Untuk sistem dengan banyak variabel, ekstensi dari metode Secant disebut Metode Broyden atau Quasi-Newton. Ide utamanya adalah mengaproksimasi matriks Jacobian J secara iteratif tanpa menghitung turunan parsial.

Newton: X_{k+1} = X_k - J_k^{-1} · F(X_k)
Broyden: X_{k+1} = X_k - B_k^{-1} · F(X_k)

Matriks B_k adalah aproksimasi Jacobian yang diperbarui menggunakan secant update formula:

B_{k+1} = B_k + \frac{(ΔF_k - B_k ΔX_k) ΔX_k^T}{ΔX_k^T ΔX_k}

di mana ΔX_k = X_{k+1} - X_k dan ΔF_k = F(X_{k+1}) - F(X_k).

Langkah-langkah Metode Secant (Satu Variabel)

Berikut prosedur sistematis untuk menerapkan metode Secant:

Langkah 1: Tentukan Dua Tebakan Awal
Pilih x₀ dan x₁ yang mendekati solusi yang diharapkan.

Langkah 2: Hitung f(x₀) dan f(x₁)
Evaluasi fungsi pada kedua tebakan awal.

Langkah 3: Iterasi dengan Rumus Secant
Hitung x_{k+1} menggunakan rumus Secant.

Langkah 4: Update Nilai
Geser: x_{k-1} ← x_k, x_k ← x_{k+1}.

Langkah 5: Cek Konvergensi
Jika |x_{k+1} - x_k| < ε atau |f(x_{k+1})| < ε, maka berhenti.

# Algoritma Secant (konsep sederhana) x_prev = x₀ x_curr = x₁ f_prev = f(x₀) f_curr = f(x₁) for k = 1 to max_iter: x_next = x_curr - f_curr * (x_curr - x_prev) / (f_curr - f_prev) if |x_next - x_curr| < toleransi: stop x_prev = x_curr f_prev = f_curr x_curr = x_next f_curr = f(x_next)

Pseudocode Metode Secant untuk Satu Variabel

PROCEDURE Secant(f, x0, x1, tol, max_iter) INPUT: f : fungsi f(x) x0, x1 : dua tebakan awal tol : toleransi error max_iter: iterasi maksimum OUTPUT: x (solusi pendekatan) x_prev ← x0 x_curr ← x1 f_prev ← f(x_prev) f_curr ← f(x_curr) FOR k = 1 TO max_iter DO {Hindari pembagian dengan nol} IF |f_curr - f_prev| < eps THEN RETURN "Pembagi mendekati nol" END IF x_next ← x_curr - f_curr * (x_curr - x_prev) / (f_curr - f_prev) IF |x_next - x_curr| < tol THEN RETURN x_next END IF x_prev ← x_curr f_prev ← f_curr x_curr ← x_next f_curr ← f(x_curr) END FOR RETURN x_curr {tidak konvergen dalam max_iter} END PROCEDURE

Contoh: Mencari Akar Persamaan

Diberikan persamaan non linear:

f(x) = x² - 4 = 0

Persamaan ini memiliki akar di x = 2 dan x = -2. Tebakan awal: x₀ = 1 dan x₁ = 3.

Iterasi (k)x_{k-1}x_kx_{k+1}f(x_{k+1})
0 → 1 1.0000 3.0000 2.0000 0.0000
1 → 2 3.0000 2.0000 2.0000 0.0000

Menakjubkan! Hanya dalam satu iterasi, metode Secant langsung menemukan solusi eksak x = 2 karena fungsi yang dicari adalah kuadrat (kasus khusus).

Untuk fungsi non linear yang lebih kompleks, biasanya dibutuhkan lebih banyak iterasi.

Laju Konvergensi

Metode Secant memiliki laju konvergensi superlinear dengan orde sekitar φ ≈ 1.618 (rasio emas).

|e_{k+1}| ≈ C · |e_k|^φ, \quad \text{dimana } φ = \frac{1+\sqrt{5}}{2} \approx 1.618

Artinya, metode Secant lebih cepat dari Fixed Point (linear) tetapi tidak secepat Newton-Raphson (kuadratik).

Poin Penting Metode Secant

Metode Secant adalah pilihan cerdas ketika turunan sulit atau mahal dihitung, tetapi kita tetap menginginkan konvergensi yang lebih cepat dari Fixed Point.

Ringkasan konsep kunci:

1. Rumus iterasi: x_{k+1} = x_k - f(x_k)·(x_k - x_{k-1}) / (f(x_k) - f(x_{k-1}))
2. Memerlukan dua tebakan awal
3. Laju konvergensi: superlinear (orde ≈1.618)
4. Kelebihan: tanpa turunan, lebih cepat dari Fixed Point
5. Kekurangan: lebih lambat dari Newton-Raphson, butuh dua tebakan
6. Ekstensi untuk SPNL: Metode Broyden (Quasi-Newton)

Metode Secant sering menjadi pilihan praktis dalam aplikasi rekayasa di mana turunan analitik tidak tersedia atau terlalu kompleks untuk dihitung. Untuk sistem dengan banyak variabel, metode Broyden (yang merupakan generalisasi Secant) menjadi alternatif populer selain Newton-Raphson.
