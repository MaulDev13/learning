Metode Broyden (Quasi-Newton)

Metode Broyden adalah generalisasi dari metode Secant untuk sistem persamaan non linear dengan banyak variabel. Metode ini termasuk dalam keluarga Quasi-Newton karena mengaproksimasi matriks Jacobian tanpa menghitung turunan parsial secara eksplisit.

Ide utamanya adalah membangun dan memperbarui aproksimasi Jacobian secara iteratif menggunakan informasi dari iterasi sebelumnya. Dengan cara ini, Broyden mempertahankan konvergensi superlinear seperti Secant, tetapi dapat diterapkan pada sistem multivariabel.

Newton: X_{k+1} = X_k - J_k^{-1} · F(X_k)
Broyden: X_{k+1} = X_k - B_k^{-1} · F(X_k)

di mana B_k adalah aproksimasi dari matriks Jacobian pada iterasi ke-k.

Mengapa Perlu Metode Broyden?

Metode Newton-Raphson untuk SPNL memang sangat cepat (konvergensi kuadratik), tetapi memiliki kelemahan signifikan:

1. Perhitungan Jacobian mahal
Untuk sistem dengan n variabel, matriks Jacobian berukuran n × n dan memerlukan n² turunan parsial. Setiap turunan bisa rumit jika fungsi kompleks.

2. Solusi sistem linear mahal
Setiap iterasi Newton memerlukan solusi sistem linear J·ΔX = -F dengan kompleksitas O(n³).

3. Turunan tidak selalu tersedia
Dalam banyak aplikasi praktis, fungsi hanya diketahui sebagai "black box" tanpa rumus analitik untuk turunannya.

Metode Broyden hadir sebagai solusi — mempertahankan kecepatan Newton tanpa perlu menghitung Jacobian secara eksplisit.

"Broyden: Newton tanpa turunan, Secant untuk sistem besar."

Konsep Dasar Quasi-Newton

Ide fundamental Quasi-Newton adalah mempertahankan secant condition yang menghubungkan perubahan variabel dan perubahan fungsi:

B_{k+1} · ΔX_k = ΔF_k

di mana:

ΔX_k = X_{k+1} - X_k
ΔF_k = F(X_{k+1}) - F(X_k)

Kondisi ini analog dengan rumus Secant untuk satu variabel:

(f(x_{k+1}) - f(x_k)) ≈ f'(x_k) · (x_{k+1} - x_k)

Pada metode Broyden, kita memulai dengan tebakan awal matriks Jacobian (biasanya B₀ = I atau matriks identitas), kemudian memperbaruinya setiap iterasi agar memenuhi secant condition.

Rumus Update Broyden (Rank-1 Update)

Broyden mengusulkan rank-1 update untuk memperbarui aproksimasi Jacobian:

B_{k+1} = B_k + \frac{(ΔF_k - B_k ΔX_k) ΔX_k^T}{ΔX_k^T ΔX_k}

Rumus ini memastikan bahwa B_{k+1} memenuhi secant condition dan merupakan modifikasi minimal dari B_k.

Untuk menghindari inversi matriks pada setiap iterasi, kita dapat memperbarui invers Jacobian secara langsung menggunakan Sherman-Morrison formula:

B_{k+1}^{-1} = B_k^{-1} + \frac{(ΔX_k - B_k^{-1} ΔF_k) ΔX_k^T B_k^{-1}}{ΔX_k^T B_k^{-1} ΔF_k}

Dengan rumus ini, setiap iterasi hanya memerlukan operasi O(n²) dibandingkan O(n³) pada Newton-Raphson.

# Rank-1 update untuk aproksimasi Jacobian ΔX = X_new - X_old ΔF = F(X_new) - F(X_old) # Update B (aproksimasi Jacobian) B = B + (ΔF - B·ΔX) · ΔXᵀ / (ΔXᵀ·ΔX)

Langkah-langkah Metode Broyden

Berikut prosedur sistematis untuk menerapkan metode Broyden:

Langkah 1: Tentukan Tebakan Awal
Pilih X₀ dan aproksimasi Jacobian awal B₀ (biasanya matriks identitas).

Langkah 2: Hitung F(Xₖ)
Evaluasi fungsi pada titik Xₖ.

Langkah 3: Selesaikan Sistem Linear
Cari ΔX yang memenuhi Bₖ·ΔX = -F(Xₖ).

Langkah 4: Update Solusi
X_{k+1} = X_k + ΔX

Langkah 5: Update Aproksimasi Jacobian
Hitung ΔF = F(X_{k+1}) - F(X_k) dan update B menggunakan rumus rank-1.

Langkah 6: Cek Konvergensi
Jika ||ΔX|| < ε atau ||F(X_{k+1})|| < ε, maka berhenti. Jika tidak, ulangi dari langkah 2.

# Algoritma Broyden (konsep sederhana) X = X₀ B = I (matriks identitas) for k = 1 to max_iter: Fx = F(X) selesaikan B · ΔX = -Fx X_new = X + ΔX if |ΔX| < toleransi: stop ΔF = F(X_new) - Fx B = B + (ΔF - B·ΔX) · ΔXᵀ / (ΔXᵀ·ΔX) X = X_new

Pseudocode Metode Broyden

PROCEDURE Broyden(F, X0, tol, max_iter) INPUT: F : fungsi sistem F(X) X0 : vektor tebakan awal tol : toleransi error max_iter: iterasi maksimum OUTPUT: X (solusi pendekatan) n ← length(X0) X ← X0 B ← identity_matrix(n) {aproksimasi Jacobian awal} FOR k = 1 TO max_iter DO Fx ← F(X) {Selesaikan B·ΔX = -Fx} ΔX ← SOLVE(B, -Fx) X_new ← X + ΔX error ← max(|ΔX|) IF error < tol THEN RETURN X_new END IF {Update aproksimasi Jacobian} ΔF ← F(X_new) - Fx numerator ← (ΔF - B·ΔX) · (ΔXᵀ) denominator ← ΔXᵀ · ΔX IF |denominator| > eps THEN B ← B + numerator / denominator END IF X ← X_new END FOR RETURN X {tidak konvergen dalam max_iter} END PROCEDURE

Perhatikan bahwa fungsi SOLVE(B, -Fx) menyelesaikan sistem linear. Kita dapat menggunakan eliminasi Gauss atau dekomposisi LU.

Tebakan awal: (x₀, y₀) = (5, 2)
Aproksimasi Jacobian awal: B₀ = I (matriks identitas)

Laju Konvergensi Broyden

Metode Broyden memiliki laju konvergensi superlinear, mirip dengan metode Secant (orde ≈1.618). Meskipun tidak secepat Newton (kuadratik), Broyden jauh lebih efisien secara komputasi untuk sistem besar.

Poin Penting Metode Broyden

Metode Broyden adalah pilihan utama untuk menyelesaikan SPNL berukuran besar ketika turunan Jacobian tidak tersedia atau terlalu mahal untuk dihitung.

Ringkasan konsep kunci:

1. Broyden adalah generalisasi Secant untuk sistem multivariabel
2. Mengaproksimasi Jacobian dengan rank-1 update tanpa turunan
3. Rumus update: B_{k+1} = B_k + (ΔF - B_k ΔX) ΔXᵀ / (ΔXᵀ ΔX)
4. Laju konvergensi: superlinear (orde ≈1.618)
5. Kelebihan: lebih cepat dari Fixed Point, tanpa turunan, O(n²) per iterasi
6. Kekurangan: membutuhkan memori O(n²), konvergensi tidak secepat Newton
7. Variasi lain: Broyden's good method, Broyden's bad method, BFGS

"Untuk sistem dengan n besar dan Jacobian sulit dihitung,
Broyden adalah pilihan paling praktis."

Metode Broyden telah menjadi standar dalam banyak library komputasi ilmiah (seperti SciPy's scipy.optimize.root dengan method='broyden1') karena keseimbangan antara kecepatan, efisiensi, dan kemudahan implementasi.
