Metode Newton Multivariable

Metode Newton Multivariable adalah pengembangan dari metode Newton-Raphson untuk menyelesaikan sistem persamaan non linear dengan lebih dari satu variabel.

Metode ini menggunakan pendekatan turunan (Jacobian) untuk memperbaiki solusi secara iteratif hingga mendekati nilai sebenarnya.

Cocok digunakan untuk SPNL dengan dua atau lebih variabel.

Ide Dasar Metode

Ide utama metode ini adalah melakukan pendekatan linear terhadap fungsi non linear di sekitar titik tebakan awal.

Kemudian, solusi diperbaiki secara bertahap menggunakan informasi turunan parsial dari fungsi tersebut.

Setiap iterasi akan menghasilkan pendekatan yang lebih baik.

Bentuk Umum Sistem

Sistem persamaan non linear ditulis sebagai:

f₁(x, y, ...) = 0
f₂(x, y, ...) = 0
...

Dalam bentuk vektor: F(x) = 0

Matriks Jacobian

Jacobian adalah matriks yang berisi turunan parsial dari setiap fungsi terhadap setiap variabel.

Bentuk umum:

J(x) = [ ∂f₁/∂x ∂f₁/∂y ]
[ ∂f₂/∂x ∂f₂/∂y ]

Jacobian digunakan untuk menentukan arah perbaikan solusi.

Rumus Iterasi Newton

Rumus utama metode Newton multivariable:

x^(k+1) = x^(k) - J⁻¹(x^(k)) · F(x^(k))

Artinya: solusi baru diperoleh dari solusi lama yang dikoreksi menggunakan Jacobian.

Langkah-langkah

1. Tentukan tebakan awal (initial guess)
2. Hitung nilai fungsi F(x)
3. Hitung matriks Jacobian J(x)
4. Selesaikan sistem linear
5. Perbarui nilai variabel
6. Ulangi sampai konvergen

Contoh Sistem

f₁(x, y) = x² + y² - 4
f₂(x, y) = x - y

Tebakan awal: x⁰ = 1, y⁰ = 1

Ilustrasi Iterasi

Hitung Jacobian:

J(x,y) = [ 2x 2y ]
[ 1 -1 ]

Nilai fungsi dan Jacobian digunakan untuk mendapatkan koreksi nilai x dan y.

Kelebihan Metode

• Konvergensi cepat (jika mendekati solusi)
• Akurat untuk banyak kasus
• Efisien untuk sistem kecil hingga menengah

Kekurangan Metode

• Perlu menghitung turunan parsial
• Membutuhkan invers matriks Jacobian
• Sensitif terhadap tebakan awal
• Bisa tidak konvergen

Konvergensi

Metode Newton akan konvergen jika:

• Tebakan awal cukup dekat dengan solusi
• Fungsi bersifat kontinu dan halus
• Jacobian tidak singular

Kesimpulan

Metode Newton Multivariable adalah metode yang kuat untuk menyelesaikan SPNL dengan pendekatan iteratif.

Dengan bantuan Jacobian, metode ini mampu memberikan solusi yang cepat dan akurat, namun memerlukan perhitungan yang lebih kompleks.
