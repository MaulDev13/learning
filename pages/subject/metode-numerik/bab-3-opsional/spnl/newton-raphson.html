SPNL - Metode Newton-Raphson

Metode Newton-Raphson adalah metode numerik yang paling populer untuk menyelesaikan sistem persamaan non linear (SPNL). Metode ini menggunakan pendekatan deret Taylor untuk melakukan linearisasi di sekitar tebakan awal, kemudian menyelesaikan sistem linear yang dihasilkan.

Dibandingkan dengan metode Fixed Point, Newton-Raphson memiliki konvergensi kuadratik, yaitu jumlah digit akurat dapat meningkat secara signifikan pada setiap iterasi. Hal ini menjadikannya metode utama dalam komputasi ilmiah.

Ide utama: Linearisasi fungsi non linear di sekitar titik tebakan
f(x) ≈ f(x₀) + f'(x₀)(x - x₀)

Penurunan Rumus (Kasus Satu Variabel)

Misalkan kita ingin menyelesaikan persamaan f(x) = 0. Ekspansi deret Taylor di sekitar xₖ:

f(x) = f(xₖ) + f'(xₖ)(x - xₖ) + \frac{f''(ξ)}{2!}(x - xₖ)² + ...

Dengan mengabaikan suku orde tinggi (linear aproksimasi), diperoleh:

0 ≈ f(xₖ) + f'(xₖ)(x - xₖ)

Sehingga iterasi Newton-Raphson menjadi:

x_{k+1} = x_k - \frac{f(x_k)}{f'(x_k)}
# Newton-Raphson (1 variabel) x_{k+1} = x_k - f(x_k) / f'(x_k)

Newton-Raphson untuk SPNL (Multivariabel)

Untuk sistem dengan n variabel:

F(X) = 0, \quad X = [x₁, x₂, ..., xₙ]^T

Bentuk iterasi Newton-Raphson:

X_{k+1} = X_k - J(X_k)^{-1} \cdot F(X_k)

dengan J(X) adalah matriks Jacobian:

J(X) = \begin{bmatrix} \frac{∂f₁}{∂x₁} & \cdots & \frac{∂f₁}{∂xₙ} \\ \vdots & \ddots & \vdots \\ \frac{∂fₙ}{∂x₁} & \cdots & \frac{∂fₙ}{∂xₙ} \end{bmatrix}

Langkah-langkah Metode Newton-Raphson

Langkah 1: Tentukan tebakan awal X₀
Langkah 2: Hitung nilai fungsi F(Xₖ)
Langkah 3: Hitung Jacobian J(Xₖ)
Langkah 4: Selesaikan sistem linear J·ΔX = -F
Langkah 5: Update solusi X_{k+1} = X_k + ΔX
Langkah 6: Cek konvergensi

X = X₀ for k = 1 to max_iter: F = F(X) J = J(X) ΔX = solve(J, -F) X = X + ΔX if norm(ΔX) < tol: stop

Pseudocode Metode Newton-Raphson

PROCEDURE NewtonRaphson(F, J, X0, tol, max_iter) X ← X0 FOR k = 1 TO max_iter DO Fx ← F(X) Jx ← J(X) ΔX ← SOLVE(Jx, -Fx) X ← X + ΔX error ← max(|ΔX|) IF error < tol THEN RETURN X END IF END FOR RETURN X END PROCEDURE

Poin Penting

1. Newton-Raphson adalah metode utama untuk SPNL
2. Menggunakan linearisasi Jacobian
3. Konvergensi kuadratik sangat cepat
4. Membutuhkan Jacobian dan invers matriks
5. Sensitif terhadap tebakan awal

Metode ini menjadi dasar berbagai metode lanjutan seperti Broyden dan Quasi-Newton.
