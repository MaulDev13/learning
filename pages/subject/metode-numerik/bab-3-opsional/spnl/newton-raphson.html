Metode Newton-Raphson

Metode Newton-Raphson adalah metode numerik yang paling populer untuk menyelesaikan sistem persamaan non linear. Metode ini menggunakan pendekatan deret Taylor untuk melakukan linearisasi di sekitar tebakan awal, kemudian menyelesaikan sistem linear yang dihasilkan.

Dibandingkan dengan metode Fixed Point, Newton-Raphson memiliki konvergensi kuadratik — artinya jumlah digit akurat kira-kira berlipat ganda setiap iterasi. Inilah mengapa metode ini menjadi andalan dalam komputasi ilmiah.

Ide utama: Linearisasi fungsi non linear di sekitar titik tebakan
f(x) ≈ f(x₀) + f'(x₀)(x - x₀)

Penurunan Rumus (Kasus Satu Variabel)

Misalkan kita ingin menyelesaikan persamaan f(x) = 0. Ekspansi deret Taylor di sekitar xₖ:

f(x) = f(xₖ) + f'(xₖ)(x - xₖ) + \frac{f''(ξ)}{2!}(x - xₖ)² + ...

Dengan mengabaikan suku order tinggi (linearisasi), kita peroleh:

0 ≈ f(xₖ) + f'(xₖ)(x - xₖ)

Selesaikan untuk x = x_{k+1}:

x_{k+1} = x_k - \frac{f(x_k)}{f'(x_k)}

Inilah rumus iterasi Newton-Raphson untuk satu variabel.

# Rumus Newton-Raphson (1 variabel) x_{k+1} = x_k - f(x_k) / f'(x_k)

Newton-Raphson untuk SPNL (Multivariabel)

Untuk sistem dengan n variabel dan n persamaan:

f₁(x₁, x₂, ..., xₙ) = 0
f₂(x₁, x₂, ..., xₙ) = 0
⋮
fₙ(x₁, x₂, ..., xₙ) = 0

Bentuk vektornya: F(X) = 0, dengan X = [x₁, x₂, ..., xₙ]ᵀ.

Linearisasi dengan deret Taylor multivariabel menghasilkan:

X_{k+1} = X_k - J(X_k)^{-1} · F(X_k)

di mana J(X) adalah Matriks Jacobian yang berisi semua turunan parsial:

J(X) = \begin{bmatrix} \frac{∂f₁}{∂x₁} & \frac{∂f₁}{∂x₂} & \cdots & \frac{∂f₁}{∂xₙ} \\ \frac{∂f₂}{∂x₁} & \frac{∂f₂}{∂x₂} & \cdots & \frac{∂f₂}{∂xₙ} \\ \vdots & \vdots & \ddots & \vdots \\ \frac{∂fₙ}{∂x₁} & \frac{∂fₙ}{∂x₂} & \cdots & \frac{∂fₙ}{∂xₙ} \end{bmatrix}

Langkah-langkah Metode Newton-Raphson

Berikut prosedur sistematis untuk menerapkan Newton-Raphson pada SPNL:

Langkah 1: Tentukan Tebakan Awal
Pilih X₀ yang sedekat mungkin dengan solusi yang diharapkan.

Langkah 2: Hitung F(Xₖ)
Evaluasi semua fungsi pada titik Xₖ.

Langkah 3: Hitung Matriks Jacobian J(Xₖ)
Hitung semua turunan parsial pada titik Xₖ.

Langkah 4: Selesaikan Sistem Linear
Cari ΔX yang memenuhi J·ΔX = -F.

Langkah 5: Update Solusi
X_{k+1} = X_k + ΔX

Langkah 6: Cek Konvergensi
Jika ||ΔX|| < ε atau ||F(X_{k+1})|| < ε, maka berhenti. Jika tidak, ulangi dari langkah 2.

# Algoritma Newton-Raphson (konsep sederhana) X = X₀ for k = 1 to max_iter: F = hitung_fungsi(X) J = hitung_jacobian(X) selesaikan J * ΔX = -F X = X + ΔX if norm(ΔX) < toleransi: stop

Pseudocode Metode Newton-Raphson untuk SPNL

PROCEDURE NewtonRaphson(F, J, X0, tol, max_iter) INPUT: F : fungsi sistem F(X) J : fungsi Jacobian J(X) X0 : vektor tebakan awal tol : toleransi error max_iter: iterasi maksimum OUTPUT: X (solusi pendekatan) X ← X0 FOR k = 1 TO max_iter DO Fx ← F(X) {hitung nilai fungsi} Jx ← J(X) {hitung matriks Jacobian} {Selesaikan sistem linear Jx · ΔX = -Fx} ΔX ← SOLVE(Jx, -Fx) X ← X + ΔX {update solusi} error ← max(|ΔX|) {atau max(|Fx|)} IF error < tol THEN RETURN X END IF END FOR RETURN X {tidak konvergen dalam max_iter} END PROCEDURE

Fungsi SOLVE(J, b) dapat menggunakan eliminasi Gauss atau dekomposisi LU untuk menyelesaikan sistem linear.

Syarat Konvergensi Newton-Raphson

Newton-Raphson memiliki konvergensi yang sangat cepat jika syarat berikut terpenuhi:

SyaratPenjelasan
Tebakan awal cukup dekat Semakin dekat tebakan ke solusi, semakin terjamin konvergensi
J(X*) tidak singular Matriks Jacobian pada solusi harus memiliki invers
f''(x) kontinu Fungsi harus cukup halus (dapat diturunkan dua kali)
f'(x*) ≠ 0 Akar tidak boleh merupakan akar ganda
Laju konvergensi Newton-Raphson (kuadratik):
|e_{k+1}| ≈ C · |e_k|²
di mana e_k = x_k - x* (error pada iterasi ke-k)

Artinya, jika error saat ini sekitar 10⁻², error berikutnya sekitar 10⁻⁴, kemudian 10⁻⁸, dan seterusnya.

Contoh: Sistem 2 Variabel

Diberikan sistem persamaan non linear:

f₁(x,y) = x² + y² - 25 = 0
f₂(x,y) = x·y - 12 = 0

Matriks Jacobian:

J(x,y) = \begin{bmatrix} 2x & 2y \\ y & x \end{bmatrix}

Tebakan awal: (x₀, y₀) = (5, 2)

Poin Penting Metode Newton-Raphson

Metode Newton-Raphson adalah metode numerik paling populer untuk SPNL karena konvergensi kuadratiknya yang sangat cepat.

Ringkasan konsep kunci:

1. Rumus iterasi: X_{k+1} = X_k - J(X_k)^{-1}·F(X_k)
2. Konvergensi kuadratik: |e_{k+1}| ≈ C·|e_k|²
3. Kelebihan: konvergensi sangat cepat (jika konvergen)
4. Kekurangan: perlu Jacobian, mahal per iterasi, bisa gagal jika Jacobian singular
5. Syarat konvergensi: tebakan awal cukup dekat, Jacobian non singular

Pada praktiknya, Newton-Raphson menjadi pilihan utama untuk sistem dengan ukuran moderat dan fungsi yang dapat diturunkan dengan mudah. Untuk sistem yang sangat besar atau Jacobian sulit dihitung, digunakan variasi seperti Quasi-Newton (Broyden) yang akan dipelajari kemudian.
