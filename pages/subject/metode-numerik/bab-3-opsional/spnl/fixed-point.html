Metode Fixed Point (Titik Tetap)

Metode Fixed Point adalah metode iteratif untuk menyelesaikan sistem persamaan non linear dengan mengubah persamaan ke bentuk tertentu.

Ide utamanya adalah mencari nilai yang tidak berubah ketika dimasukkan kembali ke dalam fungsi.

Nilai tersebut disebut sebagai titik tetap (fixed point).

Ide Dasar Metode

Persamaan non linear diubah ke bentuk: x = g(x)

Kemudian dilakukan iterasi:

x^(k+1) = g(x^(k))

Proses ini diulang sampai nilai x tidak banyak berubah.

Fixed Point untuk SPNL

Untuk sistem dengan banyak variabel, bentuknya menjadi:

x₁ = g₁(x₁, x₂, ...)
x₂ = g₂(x₁, x₂, ...)
...

Setiap variabel dihitung ulang berdasarkan fungsi masing-masing.

Bentuk Iterasi

Dalam bentuk vektor: X^(k+1) = G(X^(k))

Dimana:

• X adalah vektor variabel
• G adalah fungsi transformasi

Langkah-langkah

1. Ubah persamaan ke bentuk x = g(x)
2. Tentukan tebakan awal
3. Hitung nilai baru menggunakan fungsi g
4. Ulangi iterasi
5. Berhenti jika konvergen

Contoh Sederhana

Misalkan:

x = (y + 3) / 4
y = (x + 5) / 3

Tebakan awal: x⁰ = 0, y⁰ = 0

Proses Iterasi

Iterasi dilakukan dengan memasukkan nilai lama ke dalam fungsi:

x¹ = (y⁰ + 3) / 4
y¹ = (x⁰ + 5) / 3

Nilai ini terus diperbarui hingga stabil.

Syarat Konvergensi

Metode Fixed Point akan konvergen jika fungsi g memenuhi syarat tertentu.

Salah satu syarat penting:

|g'(x)| < 1

Untuk multivariable, nilai turunan (Jacobian) harus cukup kecil agar iterasi stabil.

Kelebihan Metode

• Konsep sederhana
• Mudah diimplementasikan
• Tidak memerlukan turunan eksplisit (bisa disesuaikan)

Kekurangan Metode

• Konvergensi tidak selalu terjamin
• Sensitif terhadap bentuk fungsi g(x)
• Bisa lambat dibanding metode lain
• Pemilihan bentuk transformasi sangat penting

Kesimpulan

Metode Fixed Point adalah metode iteratif yang sederhana untuk menyelesaikan SPNL dengan pendekatan titik tetap.

Keberhasilan metode ini sangat bergantung pada bentuk fungsi dan pemilihan tebakan awal yang tepat.
