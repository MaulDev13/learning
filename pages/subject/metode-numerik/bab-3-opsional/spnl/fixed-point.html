Metode Fixed Point (Titik Tetap)

Metode Fixed Point atau Metode Iterasi Titik Tetap adalah metode numerik yang digunakan untuk menyelesaikan sistem persamaan non linear dengan mengubah persamaan asli F(x) = 0 menjadi bentuk x = G(x).

Ide utamanya adalah mencari titik tetap (fixed point) — yaitu nilai x yang tidak berubah ketika dimasukkan kembali ke dalam fungsi G. Dengan kata lain, jika x* = G(x*), maka x* adalah titik tetap sekaligus solusi dari persamaan asli.

Bentuk umum: x = G(x)
Titik tetap: x* = G(x*)

Ide Dasar Metode

Misalkan kita memiliki persamaan non linear:

f(x) = 0

Persamaan ini dapat diubah secara aljabar menjadi bentuk:

x = g(x)

Selanjutnya, kita melakukan iterasi dengan rumus:

x^{(k+1)} = g(x^{(k)})

Proses ini diulang terus sampai nilai x tidak banyak berubah (konvergen) atau mencapai batas iterasi maksimum.

# Ide dasar dalam bentuk sederhana x₀ = tebakan_awal Ulangi: x_baru = g(x_lama) jika |x_baru - x_lama| < toleransi: stop x_lama = x_baru

Fixed Point untuk Sistem Persamaan (SPNL)

Untuk sistem dengan banyak variabel, bentuk x = G(x) diperluas menjadi sistem persamaan:

x₁ = g₁(x₁, x₂, ..., xₙ)
x₂ = g₂(x₁, x₂, ..., xₙ)
⋮
xₙ = gₙ(x₁, x₂, ..., xₙ)

Atau dalam bentuk vektor:

X^{(k+1)} = G(X^{(k)})

di mana X = [x₁, x₂, ..., xₙ]ᵀ dan G = [g₁, g₂, ..., gₙ]ᵀ.

Pada setiap iterasi, semua variabel dihitung ulang berdasarkan nilai variabel dari iterasi sebelumnya.

Langkah-langkah Metode Fixed Point

Berikut adalah prosedur sistematis untuk menerapkan metode Fixed Point:

Langkah 1: Transformasi Persamaan
Ubah bentuk F(x) = 0 menjadi x = G(x). Pemilihan bentuk G sangat menentukan keberhasilan metode.

Langkah 2: Tentukan Tebakan Awal
Pilih nilai awal X⁰ yang mendekati solusi yang diharapkan.

Langkah 3: Iterasi
Hitung X^{(k+1)} = G(X^{(k)}) berulang kali.

Langkah 4: Cek Konvergensi
Berhenti jika ||X^{(k+1)} - X^{(k)}|| < ε atau mencapai iterasi maksimum.

# Algoritma Fixed Point untuk SPNL # Input : G(x), tebakan awal X0, toleransi tol, max_iter # Output: solusi X X = X0 for k = 1 to max_iter: X_baru = G(X) error = max(|X_baru - X|) if error < tol: return X_baru X = X_baru return X # tidak konvergen dalam max_iter

Syarat Konvergensi

Tidak semua bentuk G(x) menghasilkan iterasi yang konvergen. Metode Fixed Point akan konvergen ke titik tetap jika memenuhi syarat berikut:

KasusSyarat KonvergensiKeterangan
Satu variabel |g'(x)| < 1 di sekitar titik tetap
Banyak variabel ρ(J_G) < 1 spektral radius Jacobian G kurang dari 1
Kondisi cukup ||J_G|| < 1 norm matriks Jacobian kurang dari 1

Interpretasi praktis: Nilai turunan (atau Jacobian) dari fungsi G harus cukup kecil di sekitar solusi. Semakin kecil nilai turunannya, semakin cepat konvergensi.

Syarat konvergensi (satu variabel): |g'(x*)| < 1
Jika |g'(x*)| > 1 → divergen
Jika |g'(x*)| = 1 → tidak dapat ditentukan (mungkin konvergen lambat)

Contoh: Sistem 2 Variabel

Diberikan sistem persamaan non linear:

x = (y + 3) / 4
y = (x + 5) / 3

Sistem ini sudah dalam bentuk X = G(X) sehingga siap untuk diiterasi. Tebakan awal yang dipilih:

x⁰ = 0, y⁰ = 0

Proses iterasi:

Pseudocode Metode Fixed Point untuk SPNL

Berikut adalah pseudocode lengkap yang dapat diimplementasikan ke dalam bahasa pemrograman:

PROCEDURE FixedPoint(G, X0, tol, max_iter) INPUT: G : fungsi transformasi G(X) X0 : vektor tebakan awal tol : toleransi error max_iter: iterasi maksimum OUTPUT: X (solusi pendekatan) X ← X0 FOR k = 1 TO max_iter DO X_baru ← G(X) error ← max(|X_baru - X|) IF error < tol THEN RETURN X_baru END IF X ← X_baru END FOR RETURN X {tidak konvergen dalam max_iter} END PROCEDURE

Pada praktiknya, fungsi G(X) harus didefinisikan sesuai dengan sistem persamaan yang akan diselesaikan.

Poin Penting Metode Fixed Point

Metode Fixed Point adalah fondasi penting dalam penyelesaian SPNL secara numerik.

Ringkasan konsep kunci:

1. Mengubah F(x) = 0 menjadi x = G(x)
2. Iterasi: X^{(k+1)} = G(X^{(k)})
3. Konvergen jika |g'(x*)| < 1 (untuk 1 variabel)
4. Kelebihan: sederhana, tidak perlu turunan
5. Kekurangan: konvergensi lambat, tidak selalu konvergen

Metode ini menjadi dasar pemahaman sebelum mempelajari metode yang lebih canggih seperti Newton-Raphson yang memiliki konvergensi lebih cepat.
