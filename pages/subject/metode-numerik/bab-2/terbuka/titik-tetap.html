Metode Titik Tetap (Fixed Point Iteration)

Metode titik tetap adalah metode numerik untuk mencari akar persamaan dengan mengubah bentuk persamaan menjadi iterasi.

Metode ini termasuk metode terbuka karena tidak memerlukan interval awal.

Konsep Dasar

Persamaan f(x) = 0 diubah menjadi bentuk:

x = g(x)

Nilai x dicari sebagai titik tetap, yaitu nilai yang tidak berubah saat dimasukkan kembali.

Definisi Titik Tetap

Jika suatu nilai r memenuhi:

r = g(r)

maka r disebut titik tetap dari fungsi g(x), dan merupakan solusi dari persamaan awal.

Ide Utama

Dimulai dari tebakan awal x⁰, kemudian dihitung secara berulang:

xi+1 = g(xi)

Proses diulang sampai nilai konvergen.

Langkah-langkah Metode

1. Ubah f(x) = 0 menjadi x = g(x)
2. Tentukan tebakan awal x⁰
3. Hitung x¹ = g(x⁰)
4. Hitung x² = g(x¹)
5. Ulangi hingga memenuhi toleransi

Contoh Kasus

f(x) = x² − 4 = 0

Salah satu bentuk:

x = √4 = 2

Atau bisa ditulis:

x = √(4)

(Contoh ini sederhana, tetapi pada kasus nyata bentuk g(x) lebih kompleks)

Proses Iterasi

Misalkan: x⁰ = 1

x¹ = g(1)
x² = g(x¹)
x³ = g(x²)

Nilai akan mendekati titik tetap jika metode konvergen.

Syarat Konvergensi

Metode akan konvergen jika:

|g'(x)| < 1

di sekitar akar.

Jika tidak terpenuhi, iterasi bisa divergen.

Kelebihan

• Sederhana dalam konsep
• Tidak memerlukan interval
• Mudah diimplementasikan

Kelemahan

• Sangat bergantung pada bentuk g(x)
• Bisa divergen jika salah transformasi
• Konvergensi tidak selalu cepat

Intuisi

Metode ini seperti “memasukkan hasil ke dalam fungsi secara berulang sampai tidak berubah lagi”.

Titik di mana nilai tidak berubah adalah solusi yang dicari.

Kesimpulan

Metode titik tetap mencari solusi melalui proses iterasi sederhana.

Keberhasilan metode sangat bergantung pada bentuk fungsi yang digunakan.

Metode ini menjadi dasar untuk memahami metode terbuka lainnya seperti Newton-Raphson.
