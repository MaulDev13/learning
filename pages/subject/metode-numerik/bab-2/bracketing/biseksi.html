Metode Biseksi

Metode biseksi adalah metode numerik untuk mencari akar persamaan dengan membagi interval menjadi dua bagian.

Metode ini termasuk metode tertutup karena menggunakan batas interval yang jelas.

Konsep Dasar

Metode biseksi bekerja berdasarkan perubahan tanda fungsi.

Jika:

f(a) × f(b) < 0

maka terdapat akar di antara a dan b.

Ide Utama

Interval [a, b] dibagi menjadi dua bagian.

Titik tengah dihitung:

c = (a + b) / 2

Kemudian dipilih sub-interval yang masih mengandung akar.

Langkah-langkah Metode Biseksi

1. Tentukan interval awal [a, b]
2. Pastikan f(a) × f(b) < 0
3. Hitung titik tengah c
4. Evaluasi f(c)
5. Tentukan interval baru:
   • Jika f(a) × f(c) < 0 → b = c
   • Jika f(c) × f(b) < 0 → a = c
6. Ulangi sampai memenuhi toleransi

Contoh Kasus

f(x) = x² − 4

Interval awal: [a, b] = [1, 3]

f(1) = -3
f(3) = 5

Karena f(1) × f(3) < 0, maka terdapat akar di antara 1 dan 3.

Iterasi 1

c = (1 + 3) / 2 = 2

f(2) = 0

Akar ditemukan secara langsung.

Proses Iterasi

Jika f(c) ≠ 0, maka interval akan terus dipersempit.

Setiap iterasi mengurangi panjang interval menjadi setengah.

Kriteria Berhenti

Iterasi dihentikan jika:

• |f(c)| mendekati 0
• Interval sangat kecil
• Galat aproksimasi < toleransi

Kelebihan Metode Biseksi

• Sederhana dan mudah dipahami
• Selalu konvergen jika syarat terpenuhi
• Stabil untuk berbagai fungsi

Kelemahan Metode Biseksi

• Konvergensi lambat
• Membutuhkan interval awal
• Tidak efisien untuk presisi tinggi

Intuisi

Metode biseksi seperti “mencari sesuatu dengan membelah dua terus-menerus”.

Setiap langkah mempersempit lokasi akar hingga ditemukan.

Kesimpulan

Metode biseksi adalah metode pencarian akar yang sederhana dan stabil.

Dengan membagi interval secara berulang, akar dapat ditemukan secara bertahap.

Meskipun lambat, metode ini sangat andal sebagai dasar metode numerik lainnya.
