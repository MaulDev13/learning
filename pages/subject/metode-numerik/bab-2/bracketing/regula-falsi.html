Metode Regula Falsi

Metode Regula Falsi adalah metode numerik untuk mencari akar persamaan non-linear dengan menggunakan pendekatan garis lurus.

Metode ini termasuk metode tertutup karena menggunakan interval [a, b] yang mengandung akar.

Konsep Dasar

Sama seperti metode biseksi, Regula Falsi membutuhkan:

f(a) × f(b) < 0

yang menunjukkan adanya akar di antara a dan b.

Ide Utama

Berbeda dengan biseksi, titik tengah tidak dihitung sebagai rata-rata.

Metode ini menggunakan garis lurus yang menghubungkan titik (a, f(a)) dan (b, f(b)).

Titik potong garis tersebut dengan sumbu x digunakan sebagai pendekatan akar.

Rumus Regula Falsi

c = b − [ f(b)(b − a) / (f(b) − f(a)) ]

Titik c adalah pendekatan akar hasil interpolasi linear.

Langkah-langkah Metode

1. Tentukan interval awal [a, b]
2. Pastikan f(a) × f(b) < 0
3. Hitung titik c menggunakan rumus
4. Evaluasi f(c)
5. Tentukan interval baru:
   • Jika f(a) × f(c) < 0 → b = c
   • Jika f(c) × f(b) < 0 → a = c
6. Ulangi hingga memenuhi toleransi

Contoh Kasus

f(x) = x² − 4

Interval awal: [1, 3]

f(1) = -3
f(3) = 5

Karena terdapat perubahan tanda, metode dapat digunakan.

Iterasi 1

c = 3 − [5(3 − 1) / (5 − (-3))]

c = 3 − (10 / 8) = 1.75

f(1.75) = -0.9375

Akar berada di antara 1.75 dan 3.

Iterasi Selanjutnya

Proses diulang dengan interval baru hingga nilai mendekati akar sebenarnya.

Setiap iterasi menggunakan pendekatan garis, sehingga lebih cepat dibanding biseksi.

Perbandingan dengan Biseksi

• Biseksi → membagi dua interval
• Regula Falsi → menggunakan interpolasi linear

Regula Falsi biasanya lebih cepat karena mempertimbangkan bentuk fungsi.

Kelebihan

• Lebih cepat dari biseksi
• Tetap stabil (menggunakan interval)
• Mudah dipahami

Kelemahan

• Bisa konvergen lambat pada kasus tertentu
• Salah satu batas interval bisa tetap
• Tidak selalu lebih efisien dari metode lain

Intuisi

Jika biseksi “membelah dua”, maka Regula Falsi “menarik garis” untuk memperkirakan posisi akar.

Pendekatan ini membuat pencarian lebih cerdas dibanding sekadar membagi interval.

Kesimpulan

Metode Regula Falsi adalah pengembangan dari metode biseksi dengan pendekatan linear.

Metode ini lebih cepat dalam banyak kasus, namun tetap mempertahankan stabilitas.

Cocok sebagai jembatan menuju metode yang lebih kompleks seperti Newton-Raphson.
