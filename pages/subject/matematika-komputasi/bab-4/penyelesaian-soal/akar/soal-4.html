Penyelesaian Soal 4 (Metode Bisection)

Soal ini mencari akar dari fungsi non-linear:

f(x) = cos(x) - x

Karena tidak dapat diselesaikan secara aljabar, digunakan metode numerik Bisection (Metode Bagi Dua).

Metode: Bisection Method (Iteratif Interval)

Prinsip Dasar

Metode ini mencari akar dengan cara mempersempit interval [a, b] yang memiliki tanda fungsi berbeda:

f(a) * f(b) < 0

Titik tengah dihitung dengan:

c = (a + b) / 2

Implementasi Python (Siap Pakai)

import numpy as np def f(x): return np.cos(x) - x # interval awal a = 0 b = 1 # toleransi error tol = 1e-6 max_iter = 20 if f(a) * f(b) >= 0: print("Interval tidak valid") else: for i in range(max_iter): c = (a + b) / 2 print(f"Iterasi {i+1}: c = {c}, f(c) = {f(c)}") if abs(f(c)) < tol: break if f(a) * f(c) < 0: b = c else: a = c print("\nAkar pendekatan:", c)

Penjelasan Komputasi

1. Fungsi np.cos(x) menghitung nilai cos(x) secara numerik.

2. Interval awal dipilih:

a = 0, b = 1

karena f(0) dan f(1) memiliki tanda berbeda (syarat Bisection terpenuhi).

3. Setiap iterasi menghitung titik tengah:

c = (a + b) / 2

4. Sistem mengecek:

if f(a) * f(c) < 0: b = c else: a = c

untuk mempersempit interval yang mengandung akar.

5. Proses berhenti ketika nilai mendekati nol (konvergen).

Hasil Numerik

Akar persamaan berada di sekitar:

x ≈ 0.739

Ini adalah titik potong antara grafik cos(x) dan garis y = x.
