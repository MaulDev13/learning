Penyelesaian Soal 1 (Metode Bisection)

Metode Bisection digunakan untuk mencari akar persamaan dengan cara membagi interval menjadi dua bagian dan memilih sub-interval yang mengandung perubahan tanda. Metode ini dijamin konvergen jika fungsi kontinu dan interval awal tepat.

Metode: Bisection Method (Root Finding)

Persamaan

f(x) = x² - 4x - 5 = 0

Interval awal dipilih: [a, b] = [0, 6]

Implementasi Python (Siap Run di VS Code)

import numpy as np # Definisi fungsi def f(x): return x**2 - 4*x - 5 # Interval awal a = 0 b = 6 # Parameter iterasi max_iter = 30 tolerance = 1e-6 print("Iterasi Metode Bisection:") for i in range(max_iter): c = (a + b) / 2 # titik tengah print(f"Iterasi {i+1}: a={a}, b={b}, c={c}, f(c)={f(c)}") # Cek akar ditemukan if abs(f(c)) < tolerance: print("\nAkar ditemukan!") break # Menentukan interval baru if f(a) * f(c) < 0: b = c else: a = c print("\nAkar pendekatan:", c)

Penjelasan Komputasi

1. Fungsi didefinisikan sebagai: f(x) = x² - 4x - 5

2. Interval awal dipilih [0, 6] karena f(0) dan f(6) memiliki tanda berbeda.

3. Titik tengah dihitung dengan:

c = (a + b) / 2

4. Interval diperbarui berdasarkan tanda: jika f(a)·f(c) < 0 → akar di kiri, jika tidak → di kanan.

Interpretasi Hasil

Metode Bisection secara bertahap memperkecil interval hingga mendekati akar. Untuk fungsi ini, akar yang ditemukan mendekati nilai eksak: x = -1 dan x = 5.

Metode ini stabil, tetapi relatif lambat dibanding Newton-Raphson.
