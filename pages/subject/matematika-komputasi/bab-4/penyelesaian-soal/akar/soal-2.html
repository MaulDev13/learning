Akar Persamaan dalam Komputasi Matematika

Pengantar

Akar persamaan adalah nilai x yang membuat fungsi f(x) = 0. Dalam banyak kasus matematika terapan, akar tidak dapat dihitung secara analitik, sehingga diperlukan metode numerik.

Python digunakan untuk menyelesaikan akar persamaan menggunakan pendekatan numerik seperti Bisection, Newton-Raphson, dan Secant Method.

Konsep Dasar

Kita mencari nilai x yang memenuhi:

f(x) = 0

Jika tidak bisa diselesaikan secara aljabar, maka digunakan metode iteratif.

Metode Numerik yang Digunakan

Metode Karakteristik
Bisection Memotong interval secara bertahap
Newton-Raphson Menggunakan turunan fungsi
Secant Tanpa turunan, menggunakan dua titik

Latihan Soal Akar Persamaan

Soal 1

f(x) = x² - 4x + 3 = 0

Soal 2

f(x) = x³ - x - 2 = 0

Soal 3

f(x) = e^x - 3x = 0

Soal 4

f(x) = cos(x) - x = 0

Soal 5

f(x) = ln(x) + x - 2 = 0
