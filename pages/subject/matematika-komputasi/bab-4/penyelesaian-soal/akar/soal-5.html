Penyelesaian Soal 5 (Metode Secant)

Soal ini mencari akar dari fungsi non-linear:

f(x) = ln(x) + x - 2

Karena tidak mudah diselesaikan secara analitik, digunakan metode numerik Secant Method yang tidak memerlukan turunan.

Metode: Secant (Iterasi Dua Titik)

Rumus Secant Method

x_{n+1} = x_n - f(x_n) (x_n - x_{n-1}) / (f(x_n) - f(x_{n-1}))

Metode ini menggunakan dua titik awal untuk mendekati akar tanpa turunan.

Implementasi Python (Siap Pakai)

import numpy as np def f(x): return np.log(x) + x - 2 # dua tebakan awal x0 = 1.0 x1 = 2.0 tol = 1e-6 max_iter = 20 for i in range(max_iter): if f(x1) - f(x0) == 0: print("Pembagian nol terjadi") break x2 = x1 - f(x1) * (x1 - x0) / (f(x1) - f(x0)) print(f"Iterasi {i+1}: x = {x2}") if abs(x2 - x1) < tol: break x0 = x1 x1 = x2 print("\nAkar pendekatan:", x2)

Penjelasan Komputasi

1. Fungsi np.log(x) digunakan untuk logaritma natural ln(x).

2. Dua titik awal dipilih:

x0 = 1.0, x1 = 2.0

karena fungsi memiliki perubahan nilai yang baik di interval tersebut.

3. Rumus Secant menggunakan pendekatan garis lurus antara dua titik:

garis sekant → estimasi akar baru

4. Iterasi dilakukan sampai selisih antar nilai cukup kecil (konvergen).

5. Tidak memerlukan turunan → lebih fleksibel dibanding Newton-Raphson.
