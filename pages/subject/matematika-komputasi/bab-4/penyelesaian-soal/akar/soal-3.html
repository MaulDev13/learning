Penyelesaian Soal 3 (Metode Newton-Raphson)

Soal ini mencari akar dari fungsi non-linear:

f(x) = e^x - 3x

Karena fungsi ini tidak memiliki solusi analitik sederhana, maka digunakan metode numerik Newton-Raphson.

Metode: Newton-Raphson Iteratif

Turunan Fungsi

f(x) = e^x - 3x f'(x) = e^x - 3

Newton-Raphson menggunakan rumus iterasi:

x_{n+1} = x_n - f(x_n) / f'(x_n)

Implementasi Python (Siap Pakai)

import numpy as np # fungsi utama def f(x): return np.exp(x) - 3*x # turunan fungsi def df(x): return np.exp(x) - 3 # tebakan awal x = 1.0 # toleransi error tol = 1e-6 # maksimal iterasi max_iter = 20 for i in range(max_iter): x_new = x - f(x) / df(x) print(f"Iterasi {i+1}: x = {x_new}") if abs(x_new - x) < tol: break x = x_new print("\nAkar pendekatan:", x)

Penjelasan Komputasi

1. np.exp(x) digunakan untuk menghitung fungsi eksponensial e^x secara vektorisasi.

2. Fungsi f(x) adalah fungsi utama yang ingin dicari akarnya (f(x)=0).

3. Fungsi df(x) adalah turunan yang dibutuhkan untuk metode Newton-Raphson.

4. Rumus iterasi:

x_{n+1} = x_n - f(x_n)/f'(x_n)

digunakan untuk memperbaiki nilai x secara bertahap sampai konvergen.

5. Loop for menjalankan iterasi hingga:
- mencapai toleransi error - atau mencapai maksimum iterasi

Interpretasi Hasil

Iterasi akan menghasilkan nilai x yang mendekati akar fungsi. Secara numerik, akar berada di sekitar:

x ≈ 0.619

Ini adalah titik di mana grafik f(x) memotong sumbu x.
