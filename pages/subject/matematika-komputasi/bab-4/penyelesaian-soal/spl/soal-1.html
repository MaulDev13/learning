Penyelesaian Soal 1 (Metode Gauss-Seidel)

Metode Gauss-Seidel merupakan metode iteratif untuk menyelesaikan Sistem Persamaan Linear (SPL). Ide utamanya adalah memperbarui nilai variabel secara berurutan menggunakan nilai terbaru yang sudah dihitung dalam satu iterasi.

Metode: Iteratif (Gauss-Seidel) | Library: NumPy

Sistem Persamaan

3x + y + z = 9
x + 4y - 2z = 2
2x - y + 3z = 7

Bentuk Iteratif (Rearranged)

x = (9 - y - z) / 3 y = (2 - x + 2z) / 4 z = (7 - 2x + y) / 3

Setiap variabel diekspresikan ulang agar dapat diperbarui secara iteratif.

Implementasi Python

import numpy as np # Inisialisasi nilai awal x, y, z = 0.0, 0.0, 0.0 # Jumlah iterasi n_iter = 10 for i in range(n_iter): x_new = (9 - y - z) / 3 y_new = (2 - x_new + 2*z) / 4 z_new = (7 - 2*x_new + y_new) / 3 x, y, z = x_new, y_new, z_new print(f"Iterasi {i+1}: x={x:.4f}, y={y:.4f}, z={z:.4f}")

Penjelasan Komputasi

1. Variabel awal x, y, z diinisialisasi dengan nilai 0. Ini disebut initial guess.

2. Pada setiap iterasi, nilai variabel diperbarui secara berurutan: x → y → z. Hal ini penting karena Gauss-Seidel menggunakan nilai terbaru, bukan nilai lama seperti Jacobi.

3. Perulangan for digunakan untuk melakukan iterasi sebanyak n kali, hingga solusi mendekati konvergen.

4. Format :.4f digunakan untuk membatasi tampilan angka desimal agar lebih mudah dibaca.

Hasil Iterasi (Konvergensi)

Iterasi 10: x ≈ 1.8824 y ≈ -0.0588 z ≈ 2.6471

Nilai ini merupakan pendekatan numerik dari solusi SPL menggunakan metode Gauss-Seidel. Semakin banyak iterasi, hasil akan semakin mendekati solusi eksak.

Kesimpulan

Metode Gauss-Seidel lebih cepat konvergen dibanding Jacobi karena menggunakan nilai terbaru dalam setiap iterasi. Metode ini sangat efektif untuk sistem linear besar yang jarang diselesaikan dengan metode langsung.
