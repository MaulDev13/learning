Penyelesaian Soal 7 (Metode Jacobi)

Metode Jacobi adalah metode iteratif untuk menyelesaikan Sistem Persamaan Linear (SPL). Setiap variabel diperbarui berdasarkan nilai iterasi sebelumnya hingga sistem mencapai konvergensi.

Metode: Jacobi Iteration (NumPy)

Sistem Persamaan Linear

4x + y + z = 7 x + 5y + z = -8 2x + y + 6z = 6

Implementasi Python (Siap Run di VS Code)

import numpy as np # Matriks koefisien A A = np.array([ [4, 1, 1], [1, 5, 1], [2, 1, 6] ], dtype=float) # Vektor hasil b b = np.array([7, -8, 6], dtype=float) # Tebakan awal x = np.zeros(len(b)) # Parameter iterasi max_iter = 30 tolerance = 1e-6 print("Iterasi Jacobi:") for k in range(max_iter): x_new = np.zeros_like(x) # Update setiap variabel for i in range(len(A)): sigma = 0 for j in range(len(A)): if i != j: sigma += A[i][j] * x[j] x_new[i] = (b[i] - sigma) / A[i][i] # Tampilkan iterasi print(f"Iterasi {k+1}: {x_new}") # Cek konvergensi if np.linalg.norm(x_new - x, ord=np.inf) < tolerance: print("\nKonvergen!") break x = x_new print("\nSolusi akhir:", x)

Penjelasan Komputasi

1. Matriks A dan vektor b merepresentasikan sistem persamaan linear.

2. Solusi awal diinisialisasi dengan nol: x = [0, 0, 0].

3. Setiap iterasi menghitung nilai baru menggunakan:

x_i^(k+1) = (b_i - Σ a_ij x_j^(k)) / a_ii

4. Proses berhenti ketika perubahan antar iterasi sudah sangat kecil (norm error < tolerance).

Interpretasi Hasil

Metode Jacobi memberikan solusi pendekatan yang semakin akurat setiap iterasi. Hasil akhir mendekati solusi SPL sebenarnya setelah konvergensi tercapai.

Metode ini cocok untuk sistem yang besar karena setiap variabel dapat dihitung secara independen.
