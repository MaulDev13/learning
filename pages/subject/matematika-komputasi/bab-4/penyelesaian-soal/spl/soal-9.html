Penyelesaian Soal 9 (Metode Gauss-Seidel)

Metode Gauss-Seidel adalah metode iteratif untuk menyelesaikan Sistem Persamaan Linear (SPL). Berbeda dengan Jacobi, metode ini langsung menggunakan nilai terbaru pada setiap iterasi, sehingga biasanya lebih cepat konvergen.

Metode: Gauss-Seidel Iteration (NumPy)

Sistem Persamaan Linear

5x + y + z = 10 x + 4y + z = 12 2x + y + 6z = 20

Implementasi Python (Siap Run di VS Code)

import numpy as np # Matriks A A = np.array([ [5, 1, 1], [1, 4, 1], [2, 1, 6] ], dtype=float) # Vektor b b = np.array([10, 12, 20], dtype=float) # Tebakan awal x = np.zeros(len(b)) # Parameter iterasi max_iter = 30 tolerance = 1e-6 print("Iterasi Gauss-Seidel:") for k in range(max_iter): x_old = x.copy() # Update langsung (in-place update) for i in range(len(A)): sigma = 0 for j in range(len(A)): if j != i: sigma += A[i][j] * x[j] # menggunakan nilai terbaru x[i] = (b[i] - sigma) / A[i][i] print(f"Iterasi {k+1}: {x}") # Cek konvergensi if np.linalg.norm(x - x_old, ord=np.inf) < tolerance: print("\nKonvergen!") break print("\nSolusi akhir:", x)

Penjelasan Komputasi

1. Sistem SPL direpresentasikan dalam bentuk matriks A dan vektor b.

2. Solusi awal diinisialisasi dengan nol: x = [0, 0, 0].

3. Perbedaan utama dengan Jacobi adalah penggunaan nilai terbaru langsung:

x_i^(k+1) = (b_i - Σ a_ij x_j^(baru)) / a_ii

4. Setiap iterasi langsung memperbarui nilai x, sehingga mempercepat konvergensi.

Interpretasi Hasil

Metode Gauss-Seidel lebih efisien dibanding Jacobi karena menggunakan nilai terbaru pada setiap langkah iterasi.

Metode ini sangat cocok untuk sistem linear yang besar dan sparsely populated, terutama dalam aplikasi teknik dan simulasi numerik.
