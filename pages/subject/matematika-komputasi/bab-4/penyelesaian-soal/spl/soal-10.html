Penyelesaian Soal 10 (Metode Gauss-Seidel)

Metode Gauss-Seidel merupakan metode iteratif untuk menyelesaikan Sistem Persamaan Linear (SPL). Metode ini memperbarui nilai variabel secara langsung pada setiap iterasi, sehingga biasanya lebih cepat konvergen dibandingkan metode Jacobi.

Metode: Gauss-Seidel Iteration (NumPy)

Sistem Persamaan Linear

6x + y - z = 5 2x + 7y + z = 15 x - y + 5z = 10

Implementasi Python (Siap Run di VS Code)

import numpy as np # Matriks koefisien A A = np.array([ [6, 1, -1], [2, 7, 1], [1, -1, 5] ], dtype=float) # Vektor hasil b b = np.array([5, 15, 10], dtype=float) # Tebakan awal x = np.zeros(len(b)) # Parameter iterasi max_iter = 30 tolerance = 1e-6 print("Iterasi Gauss-Seidel:") for k in range(max_iter): x_old = x.copy() # Update variabel satu per satu for i in range(len(A)): sigma = 0 for j in range(len(A)): if j != i: sigma += A[i][j] * x[j] x[i] = (b[i] - sigma) / A[i][i] print(f"Iterasi {k+1}: {x}") # Cek konvergensi if np.linalg.norm(x - x_old, ord=np.inf) < tolerance: print("\nKonvergen!") break print("\nSolusi akhir:", x)

Penjelasan Komputasi

1. Sistem persamaan linear direpresentasikan dalam bentuk matriks A dan vektor b.

2. Solusi awal diinisialisasi dengan nol: x = [0, 0, 0].

3. Setiap variabel diperbarui langsung menggunakan nilai terbaru:

x_i^(k+1) = (b_i - Σ a_ij x_j^(baru)) / a_ii

4. Iterasi dihentikan ketika perubahan antar iterasi sangat kecil (konvergen).

Interpretasi Hasil

Metode Gauss-Seidel memberikan solusi pendekatan yang semakin akurat setiap iterasi. Dibandingkan Jacobi, metode ini lebih cepat konvergen karena menggunakan nilai terbaru pada setiap langkah perhitungan.

Metode ini sangat umum digunakan dalam simulasi numerik, teknik, dan pemodelan sistem besar.
