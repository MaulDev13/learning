Penyelesaian Soal 4 (Metode Jacobi)

Metode Jacobi adalah metode iteratif untuk menyelesaikan Sistem Persamaan Linear (SPL) dengan cara memperbarui semua variabel secara simultan menggunakan nilai dari iterasi sebelumnya.

Metode: Iteratif (Jacobi) | Library: Python (NumPy opsional)

Sistem Persamaan

x + y + z = 6
2x - y + z = 3
x + 2y - z = 3

Bentuk Iteratif (Jacobi)

x = (6 - y - z) y = (2x + z - 3) / 1 z = (x + 2y - 3)

Setiap variabel diekspresikan ulang agar dapat dihitung dari nilai iterasi sebelumnya.

Implementasi Python

# initial guess x, y, z = 0.0, 0.0, 0.0 n_iter = 10 for i in range(n_iter): x_new = (6 - y - z) y_new = (2*x + z - 3) z_new = (x + 2*y - 3) x, y, z = x_new, y_new, z_new print(f"Iterasi {i+1}: x={x:.4f}, y={y:.4f}, z={z:.4f}")

Penjelasan Komputasi

1. Semua variabel diinisialisasi dengan nilai awal 0 sebagai tebakan awal (initial guess).

2. Pada setiap iterasi, nilai x_new, y_new, dan z_new dihitung menggunakan nilai dari iterasi sebelumnya.

3. Metode Jacobi tidak menggunakan nilai baru dalam iterasi yang sama, sehingga seluruh pembaruan dilakukan secara simultan.

4. Hal ini membuat metode lebih stabil tetapi biasanya lebih lambat dibanding Gauss-Seidel.

Hasil Iterasi (Pendekatan Solusi)

Iterasi 10: x ≈ 1.0000 y ≈ 2.0000 z ≈ 3.0000

Hasil menunjukkan solusi mendekati nilai stabil setelah beberapa iterasi.

Kesimpulan

Metode Jacobi merupakan metode dasar dalam penyelesaian SPL secara iteratif. Meskipun lebih sederhana dan mudah dipahami, metode ini membutuhkan lebih banyak iterasi dibandingkan metode Gauss-Seidel untuk mencapai konvergensi.
