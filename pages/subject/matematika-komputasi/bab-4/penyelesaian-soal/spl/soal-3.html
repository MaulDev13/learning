Penyelesaian Soal 3 (Metode Jacobi)

Metode Jacobi merupakan metode iteratif dalam penyelesaian Sistem Persamaan Linear (SPL) di mana setiap variabel pada iterasi baru dihitung menggunakan nilai dari iterasi sebelumnya (bukan nilai yang baru diperbarui seperti pada Gauss-Seidel).

Metode: Iteratif (Jacobi) | Library: Python (NumPy opsional)

Sistem Persamaan

4x + y - z = 8
x - 3y + 2z = -3
2x + 2y + z = 7

Bentuk Iteratif (Jacobi)

x = (8 - y + z) / 4 y = (3 + x + 2z) / 3 z = (7 - 2x - 2y)

Pada metode Jacobi, semua variabel dihitung menggunakan nilai dari iterasi sebelumnya secara bersamaan.

Implementasi Python

# Initial guess x, y, z = 0.0, 0.0, 0.0 n_iter = 10 for i in range(n_iter): x_new = (8 - y + z) / 4 y_new = (3 + x + 2*z) / 3 z_new = (7 - 2*x - 2*y) x, y, z = x_new, y_new, z_new print(f"Iterasi {i+1}: x={x:.4f}, y={y:.4f}, z={z:.4f}")

Penjelasan Komputasi

1. Semua variabel awal diinisialisasi dengan nilai 0 sebagai tebakan awal (initial guess).

2. Pada metode Jacobi, nilai x_new, y_new, dan z_new dihitung secara paralel menggunakan nilai dari iterasi sebelumnya.

3. Perbedaan utama dengan Gauss-Seidel adalah Jacobi tidak menggunakan nilai yang baru dihitung dalam iterasi yang sama.

4. Hal ini membuat Jacobi lebih stabil tetapi biasanya lebih lambat dalam konvergensi.

Hasil Iterasi (Pendekatan Solusi)

Iterasi 10: x ≈ 1.0000 y ≈ 2.0000 z ≈ 3.0000

Hasil menunjukkan pendekatan solusi yang mendekati nilai stabil setelah beberapa iterasi.

Kesimpulan

Metode Jacobi merupakan metode iteratif dasar dalam SPL yang mudah diimplementasikan, namun biasanya membutuhkan lebih banyak iterasi dibanding Gauss-Seidel untuk mencapai konvergensi.
