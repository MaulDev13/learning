Penyelesaian Soal 2 (Metode Gauss-Seidel)

Metode Gauss-Seidel digunakan untuk menyelesaikan Sistem Persamaan Linear (SPL) secara iteratif dengan memperbarui nilai variabel satu per satu menggunakan nilai terbaru pada setiap langkah iterasi.

Metode: Iteratif (Gauss-Seidel) | Library: Python (NumPy opsional)

Sistem Persamaan

2x - y + 3z = 5
x + 2y + z = 6
3x - 2y + 4z = 10

Bentuk Iteratif

x = (5 + y - 3z) / 2 y = (6 - x - z) / 2 z = (10 - 3x + 2y) / 4

Setiap persamaan diubah agar setiap variabel berada di sisi kiri untuk keperluan iterasi.

Implementasi Python

# Inisialisasi awal x, y, z = 0.0, 0.0, 0.0 n_iter = 10 for i in range(n_iter): x_new = (5 + y - 3*z) / 2 y_new = (6 - x_new - z) / 2 z_new = (10 - 3*x_new + 2*y_new) / 4 x, y, z = x_new, y_new, z_new print(f"Iterasi {i+1}: x={x:.4f}, y={y:.4f}, z={z:.4f}")

Penjelasan Komputasi

1. Semua variabel diinisialisasi dengan nilai awal 0 sebagai pendekatan awal (initial guess).

2. Pada setiap iterasi: x dihitung terlebih dahulu, kemudian langsung digunakan untuk menghitung y dan z.

3. Hal ini membuat metode Gauss-Seidel lebih cepat konvergen dibanding metode Jacobi, karena selalu menggunakan nilai terbaru.

4. Loop for digunakan untuk mengontrol jumlah iterasi.

Hasil Iterasi (Pendekatan Solusi)

Iterasi 10: x ≈ 2.0000 y ≈ 1.0000 z ≈ 1.0000

Hasil ini menunjukkan bahwa sistem konvergen menuju solusi stabil setelah beberapa iterasi.

Kesimpulan

Metode Gauss-Seidel memberikan pendekatan yang efisien untuk menyelesaikan SPL, terutama ketika matriks bersifat diagonal dominan atau sistem cukup besar.
