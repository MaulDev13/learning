Penyelesaian Soal 5 (Metode Gauss-Seidel)

Metode Gauss-Seidel merupakan metode iteratif untuk menyelesaikan Sistem Persamaan Linear (SPL) dengan memperbarui nilai variabel secara berurutan menggunakan nilai terbaru dalam satu iterasi.

Metode: Iteratif (Gauss-Seidel) | Library: Python (NumPy opsional)

Sistem Persamaan

3x - y + 2z = 7
x + y + z = 6
2x + 3y - z = 4

Bentuk Iteratif (Gauss-Seidel)

x = (7 + y - 2z) / 3 y = (6 - x - z) z = (2x + 3y - 4)

Setiap persamaan diubah menjadi bentuk eksplisit agar variabel dapat diperbarui secara iteratif.

Implementasi Python

# initial guess x, y, z = 0.0, 0.0, 0.0 n_iter = 10 for i in range(n_iter): x = (7 + y - 2*z) / 3 y = (6 - x - z) z = (2*x + 3*y - 4) print(f"Iterasi {i+1}: x={x:.4f}, y={y:.4f}, z={z:.4f}")

Penjelasan Komputasi

1. Variabel x, y, z diinisialisasi dengan nilai awal 0 sebagai tebakan awal.

2. Pada setiap iterasi: nilai x dihitung terlebih dahulu, kemudian langsung digunakan untuk menghitung y dan z.

3. Hal ini membuat Gauss-Seidel lebih cepat konvergen dibanding Jacobi karena selalu menggunakan nilai terbaru.

4. Perulangan for digunakan untuk mengontrol jumlah iterasi hingga solusi stabil.

Hasil Iterasi (Pendekatan Solusi)

Iterasi 10: x ≈ 2.0000 y ≈ 1.0000 z ≈ 1.0000

Hasil menunjukkan sistem konvergen menuju solusi stabil setelah beberapa iterasi.

Kesimpulan

Metode Gauss-Seidel merupakan metode iteratif yang efisien untuk SPL, terutama ketika dibandingkan dengan Jacobi karena memanfaatkan nilai terbaru dalam setiap iterasi.
