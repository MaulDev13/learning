Penyelesaian Soal 8 (Metode Jacobi)

Metode Jacobi adalah metode iteratif untuk menyelesaikan Sistem Persamaan Linear (SPL). Ide utamanya adalah memperbarui nilai variabel secara bertahap berdasarkan nilai iterasi sebelumnya hingga konvergen.

Metode: Jacobi Iteration (NumPy)

Sistem Persamaan Linear

10x + y + z = 12 2x + 10y + z = 13 2x + 2y + 10z = 14

Implementasi Python (Jacobi Method)

import numpy as np # Koefisien matriks A dan vektor b A = np.array([ [10, 1, 1], [2, 10, 1], [2, 2, 10] ], dtype=float) b = np.array([12, 13, 14], dtype=float) # Tebakan awal x = np.zeros(len(b)) # Parameter iterasi max_iter = 25 tolerance = 1e-6 # Jacobi iteration for k in range(max_iter): x_new = np.zeros_like(x) for i in range(len(A)): sum_ = 0 for j in range(len(A)): if i != j: sum_ += A[i][j] * x[j] x_new[i] = (b[i] - sum_) / A[i][i] # Cek konvergensi if np.linalg.norm(x_new - x, ord=np.inf) < tolerance: print(f"Konvergen pada iterasi ke-{k+1}") break x = x_new print("Solusi akhir:", x)

Penjelasan Alur Komputasi

1. Sistem SPL direpresentasikan dalam bentuk matriks A dan vektor b.

2. Inisialisasi awal solusi dilakukan dengan vektor nol: x = [0, 0, 0].

3. Setiap variabel diperbarui menggunakan rumus Jacobi:

x_i^(k+1) = (b_i - Σ A_ij * x_j^(k)) / A_ii

4. Iterasi dilakukan sampai selisih antara iterasi baru dan lama cukup kecil (konvergen).

Interpretasi Hasil

Metode Jacobi menghasilkan pendekatan numerik terhadap solusi SPL. Semakin banyak iterasi, semakin dekat hasil ke solusi sebenarnya.

Metode ini sangat cocok untuk sistem besar karena mudah diparalelkan, meskipun konvergensinya lebih lambat dibanding metode langsung seperti LU.
