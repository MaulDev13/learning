Penyelesaian Soal 6 (Metode LU Decomposition)

Sistem persamaan linear diselesaikan menggunakan metode LU Decomposition, yaitu pemfaktoran matriks A menjadi matriks segitiga bawah (L) dan segitiga atas (U). Metode ini termasuk metode langsung (direct method) dalam komputasi numerik.

Metode: LU Decomposition (NumPy / SciPy)

Sistem Persamaan Linear

3x + 2y - z = 7 2x - 2y + 4z = 10 -x + 0.5y - z = -2

Bentuk Matriks A dan b

A = [[ 3 2 -1 ], [ 2 -2 4 ], [-1 0.5 -1 ]] b = [7, 10, -2]

Implementasi Python (LU Decomposition)

import numpy as np from scipy.linalg import lu A = np.array([ [3, 2, -1], [2, -2, 4], [-1, 0.5, -1] ]) b = np.array([7, 10, -2]) # LU decomposition P, L, U = lu(A) # forward substitution (Ly = Pb) Pb = np.dot(P, b) y = np.linalg.solve(L, Pb) # backward substitution (Ux = y) x = np.linalg.solve(U, y) print("Solusi x:", x)

Penjelasan Alur Komputasi

1. Matriks A didekomposisi menjadi P, L, dan U menggunakan fungsi lu().

2. Matriks permutasi P digunakan untuk mengatur ulang sistem agar stabil secara numerik.

3. Sistem pertama diselesaikan: Ly = Pb menggunakan forward substitution.

4. Sistem kedua diselesaikan: Ux = y menggunakan backward substitution.

Proses ini menggantikan eliminasi Gauss dengan struktur matriks yang lebih stabil untuk komputasi numerik.

Interpretasi Hasil

Solusi akhir diperoleh dari vektor x yang merepresentasikan nilai variabel (x, y, z) yang memenuhi seluruh sistem persamaan secara simultan.

Metode LU sangat efisien untuk sistem linear berukuran besar karena faktorisasi hanya dilakukan sekali, kemudian digunakan ulang untuk berbagai vektor b.
