Penyelesaian Soal 10 (Fungsi Piecewise)

Fungsi piecewise merupakan fungsi yang memiliki definisi berbeda pada interval domain yang berbeda. Dalam komputasi matematika, fungsi ini penting untuk memodelkan sistem yang memiliki perilaku berbeda berdasarkan kondisi tertentu.

Pada kasus ini, fungsi memiliki dua bentuk: kuadrat untuk x < 0 dan linier untuk x ≥ 0. Hal ini membutuhkan pendekatan logika kondisional dalam implementasi numerik.

Library: numpy, matplotlib

1. Implementasi Lengkap Python

import numpy as np import matplotlib.pyplot as plt # Domain numerik x = np.linspace(-5, 5, 200) # Definisi fungsi piecewise def f(x): return np.where(x < 0, x**2, x) # Evaluasi fungsi y = f(x) # Titik referensi x_ref = 2 y_ref = f(x_ref) print("Nilai referensi:") print("x =", x_ref) print("y =", y_ref) # Visualisasi plt.figure(figsize=(8, 5)) plt.plot(x, y, label="Piecewise Function") plt.scatter(x_ref, y_ref, color="red", label="Titik Referensi") plt.title("Grafik Fungsi Piecewise") plt.xlabel("x") plt.ylabel("f(x)") plt.grid(True) plt.legend() plt.show()

2. Alur Komputasi

Fungsi np.where digunakan untuk menerapkan logika percabangan pada seluruh elemen array secara vectorized.

Jika nilai x < 0, maka fungsi akan menghitung x², sedangkan untuk x ≥ 0 digunakan fungsi linear x.

Pendekatan ini menggantikan struktur if-else tradisional dengan operasi berbasis array yang lebih efisien.

Evaluasi dilakukan tanpa loop eksplisit sehingga lebih optimal untuk komputasi numerik.

3. Interpretasi Visual

Grafik menunjukkan dua karakteristik fungsi yang berbeda pada dua region domain.

Pada x < 0, grafik berbentuk parabola, sedangkan pada x ≥ 0 grafik berbentuk garis lurus.

Perubahan bentuk ini menunjukkan pentingnya logika kondisional dalam pemodelan sistem komputasi matematika.
