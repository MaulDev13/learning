Penyelesaian Soal 5 (Fungsi Trigonometri: Cosinus)

Fungsi cosinus merupakan fungsi trigonometri periodik yang sering digunakan dalam pemodelan fenomena osilasi, rotasi, dan sinyal periodik. Fungsi ini memiliki kemiripan struktur dengan sinus, namun berbeda fase awal.

Dalam komputasi matematika, fungsi cosinus dievaluasi secara numerik pada domain diskrit untuk membentuk representasi grafik yang dapat dianalisis secara visual.

Library: numpy, matplotlib

1. Implementasi Lengkap Python

import numpy as np import matplotlib.pyplot as plt # Definisi fungsi cosinus def f(x): return np.cos(x) # Domain numerik x = np.linspace(-2*np.pi, 2*np.pi, 200) # Evaluasi fungsi y = f(x) # Titik referensi (x = 0) x_ref = 0 y_ref = f(x_ref) print("Nilai referensi:") print("x =", x_ref) print("y =", y_ref) # Visualisasi plt.figure(figsize=(8, 5)) plt.plot(x, y, label="f(x) = cos(x)") plt.scatter(x_ref, y_ref, color="red", label="Titik Referensi") plt.title("Grafik Fungsi Cosinus") plt.xlabel("x") plt.ylabel("f(x)") plt.grid(True) plt.legend() plt.show()

2. Alur Komputasi

Fungsi np.cos(x) digunakan untuk menghitung nilai cosinus secara vektorisasi pada seluruh elemen array NumPy.

Domain ditentukan pada interval -2π hingga 2π agar beberapa siklus periodik fungsi dapat diamati dengan jelas.

Titik referensi x = 0 dipilih karena cos(0) = 1, yang merupakan nilai maksimum awal fungsi cosinus.

Seluruh proses evaluasi dilakukan menggunakan pendekatan vectorized computation untuk efisiensi komputasi.

3. Interpretasi Visual

Grafik cosinus menunjukkan pola osilasi periodik dengan amplitudo tetap antara -1 dan 1, serupa dengan fungsi sinus tetapi bergeser fase.

Perbedaan utama terlihat pada titik awal, di mana cosinus dimulai dari nilai maksimum.

Representasi ini penting dalam analisis sinyal dan sistem dinamis berbasis gelombang.
