Penyelesaian Soal 6 (Fungsi Rasional)

Fungsi rasional merupakan fungsi yang berbentuk perbandingan dua polinomial. Dalam komputasi matematika, fungsi ini penting untuk menganalisis perilaku asimtotik, diskontinuitas, dan singularitas pada domain tertentu.

Pada kasus ini, fungsi memiliki titik tidak terdefinisi pada x = 2, sehingga domain harus dibatasi untuk menghindari pembagian dengan nol.

Library: numpy, matplotlib

1. Implementasi Lengkap Python

import numpy as np import matplotlib.pyplot as plt # Definisi fungsi rasional def f(x): return 1 / (x - 2) # Domain numerik (menghindari x = 2) x = np.linspace(-2, 6, 400) # Menghindari singularitas x = x[x != 2] # Evaluasi fungsi y = f(x) # Titik referensi x_ref = 1 y_ref = f(x_ref) print("Nilai referensi:") print("x =", x_ref) print("y =", y_ref) # Visualisasi plt.figure(figsize=(8, 5)) plt.plot(x, y, label="f(x) = 1 / (x - 2)") plt.scatter(x_ref, y_ref, color="red", label="Titik Referensi") plt.title("Grafik Fungsi Rasional") plt.xlabel("x") plt.ylabel("f(x)") plt.grid(True) plt.legend() plt.ylim(-10, 10) plt.show()

2. Alur Komputasi

Fungsi f(x) = 1 / (x - 2) memiliki titik singularitas pada x = 2 karena menyebabkan pembagian dengan nol.

Oleh karena itu, domain dibentuk menggunakan np.linspace kemudian nilai x = 2 dihapus untuk menghindari error komputasi.

Evaluasi fungsi dilakukan secara vectorized, tetapi tetap mempertimbangkan batas domain agar tidak terjadi nilai tak hingga eksplisit dalam array.

Titik referensi dipilih pada x = 1 sebagai titik aman dalam domain.

3. Interpretasi Visual

Grafik menunjukkan dua cabang fungsi yang terpisah oleh asimtot vertikal pada x = 2.

Nilai fungsi meningkat secara tak terbatas ketika mendekati x = 2 dari kiri maupun kanan, menunjukkan perilaku singularitas.

Visualisasi ini penting dalam analisis kestabilan fungsi rasional dalam komputasi numerik.
