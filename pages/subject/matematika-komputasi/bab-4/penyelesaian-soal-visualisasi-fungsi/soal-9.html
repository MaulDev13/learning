Penyelesaian Soal 9 (Fungsi Osilasi: Sinus + Cosinus)

Fungsi osilasi merupakan kombinasi dari beberapa fungsi trigonometri yang menghasilkan pola gelombang kompleks. Dalam komputasi matematika, fungsi seperti ini digunakan untuk memodelkan sinyal, getaran, dan fenomena periodik gabungan.

Pada kasus ini, fungsi merupakan superposisi dari sinus dan cosinus yang menghasilkan pola interferensi sederhana.

Library: numpy, matplotlib

1. Implementasi Lengkap Python

import numpy as np import matplotlib.pyplot as plt # Definisi fungsi osilasi def f(x): return np.sin(x) + np.cos(x) # Domain numerik x = np.linspace(-2*np.pi, 2*np.pi, 200) # Evaluasi fungsi y = f(x) # Titik referensi x_ref = 0 y_ref = f(x_ref) print("Nilai referensi:") print("x =", x_ref) print("y =", y_ref) # Visualisasi plt.figure(figsize=(8, 5)) plt.plot(x, y, label="f(x) = sin(x) + cos(x)") plt.scatter(x_ref, y_ref, color="red", label="Titik Referensi") plt.title("Grafik Fungsi Osilasi") plt.xlabel("x") plt.ylabel("f(x)") plt.grid(True) plt.legend() plt.show()

2. Alur Komputasi

Fungsi np.sin dan np.cos digunakan secara bersamaan untuk menghasilkan superposisi dua gelombang.

Domain dipilih pada interval -2π hingga 2π agar pola periodik dapat diamati secara lengkap.

Evaluasi dilakukan secara vectorized sehingga kedua fungsi dihitung secara paralel pada seluruh elemen array.

Titik referensi x = 0 digunakan untuk menunjukkan nilai awal superposisi fungsi.

3. Interpretasi Visual

Grafik menunjukkan pola osilasi yang tidak simetris sempurna akibat penjumlahan dua fungsi trigonometri.

Terjadi perubahan amplitudo lokal akibat interferensi antara sinus dan cosinus.

Representasi ini sering digunakan dalam analisis sinyal dan sistem gelombang dalam komputasi numerik.
