Penyelesaian Soal 4 (Fungsi Trigonometri: Sinus)

Fungsi trigonometri sinus merupakan fungsi periodik yang sangat penting dalam komputasi matematika, khususnya pada pemodelan fenomena gelombang, osilasi, dan sistem periodik.

Dalam pendekatan komputasi, fungsi sinus dievaluasi pada domain numerik diskrit untuk menghasilkan representasi grafik yang menunjukkan sifat periodiknya secara visual.

Library: numpy, matplotlib

1. Implementasi Lengkap Python

import numpy as np import matplotlib.pyplot as plt # Definisi fungsi sinus def f(x): return np.sin(x) # Domain numerik x = np.linspace(-2*np.pi, 2*np.pi, 200) # Evaluasi fungsi y = f(x) # Titik referensi (x = 0) x_ref = 0 y_ref = f(x_ref) print("Nilai referensi:") print("x =", x_ref) print("y =", y_ref) # Visualisasi plt.figure(figsize=(8, 5)) plt.plot(x, y, label="f(x) = sin(x)") plt.scatter(x_ref, y_ref, color="red", label="Titik Referensi") plt.title("Grafik Fungsi Sinus") plt.xlabel("x") plt.ylabel("f(x)") plt.grid(True) plt.legend() plt.show()

2. Alur Komputasi

Fungsi np.sin(x) digunakan untuk menghitung nilai sinus secara vektorisasi pada seluruh elemen array NumPy.

Domain ditentukan sebagai interval -2π hingga 2π untuk menampilkan beberapa periode penuh fungsi sinus.

Titik referensi x = 0 dipilih karena sin(0) = 0, yang merupakan titik awal penting pada grafik sinus.

Semua perhitungan dilakukan secara vectorized sehingga lebih efisien dibandingkan iterasi manual.

3. Interpretasi Visual

Grafik menunjukkan pola osilasi periodik dengan amplitudo tetap antara -1 dan 1.

Fungsi sinus memiliki periode 2π, yang terlihat dari pola berulang pada interval domain.

Titik referensi membantu mengidentifikasi posisi awal gelombang pada sumbu koordinat.
