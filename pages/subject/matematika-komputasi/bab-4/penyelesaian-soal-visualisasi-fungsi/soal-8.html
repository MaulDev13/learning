Penyelesaian Soal 8 (Model Pertumbuhan Eksponensial)

Model pertumbuhan eksponensial digunakan untuk merepresentasikan fenomena perubahan yang meningkat secara proporsional terhadap nilai saat ini, seperti pertumbuhan populasi, penyebaran data, atau proses akumulasi.

Dalam komputasi matematika, model ini dihitung secara numerik pada domain waktu diskrit dan kemudian divisualisasikan untuk memahami laju pertumbuhannya.

Library: numpy, matplotlib

1. Implementasi Lengkap Python

import numpy as np import matplotlib.pyplot as plt # Definisi model pertumbuhan eksponensial def f(t): return 100 * np.exp(0.3 * t) # Domain waktu t = np.linspace(0, 10, 100) # Evaluasi fungsi y = f(t) # Titik referensi t_ref = 5 y_ref = f(t_ref) print("Nilai referensi:") print("t =", t_ref) print("y =", y_ref) # Visualisasi plt.figure(figsize=(8, 5)) plt.plot(t, y, label="f(t) = 100 · e^(0.3t)") plt.scatter(t_ref, y_ref, color="red", label="Titik Referensi") plt.title("Model Pertumbuhan Eksponensial") plt.xlabel("t") plt.ylabel("f(t)") plt.grid(True) plt.legend() plt.show()

2. Alur Komputasi

Fungsi np.exp digunakan untuk menghitung nilai eksponensial secara vektorisasi pada seluruh elemen array.

Variabel t merepresentasikan domain waktu diskrit yang digunakan untuk mendekati proses kontinu.

Koefisien 100 menunjukkan nilai awal (initial value), sedangkan 0.3 adalah laju pertumbuhan.

Evaluasi dilakukan tanpa loop eksplisit sehingga lebih efisien dalam komputasi numerik.

3. Interpretasi Visual

Grafik menunjukkan pertumbuhan yang semakin cepat seiring bertambahnya waktu.

Kurva eksponensial memiliki karakteristik meningkat tajam pada nilai t yang lebih besar, menunjukkan sifat non-linear yang kuat.

Model ini sering digunakan dalam simulasi sistem dinamis dan analisis pertumbuhan dalam komputasi matematika.
