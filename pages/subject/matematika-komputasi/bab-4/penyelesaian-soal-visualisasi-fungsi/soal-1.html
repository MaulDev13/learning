Penyelesaian Soal 1 (Fungsi Kuadrat)

Penyelesaian fungsi kuadrat dalam komputasi matematika dilakukan dengan pendekatan evaluasi numerik menggunakan array, kemudian hasilnya divisualisasikan untuk memahami bentuk grafik secara geometris.

Pada kasus ini, fungsi tidak hanya dihitung secara aljabar, tetapi juga dievaluasi pada banyak titik untuk membentuk representasi grafik yang kontinu secara numerik.

Library: numpy, matplotlib

1. Implementasi Lengkap Python

import numpy as np import matplotlib.pyplot as plt # Definisi fungsi matematika def f(x): return x**2 - 4*x + 3 # Domain diskrit (sampling numerik) x = np.linspace(-2, 6, 100) # Evaluasi fungsi secara vectorized y = f(x) # Titik minimum (analitik sederhana) x_min = 2 y_min = f(x_min) print("Titik minimum:") print("x =", x_min) print("y =", y_min) # Visualisasi grafik plt.figure(figsize=(8, 5)) plt.plot(x, y, label="f(x) = x² - 4x + 3") plt.scatter(x_min, y_min, color="red", label="Titik Minimum") plt.title("Grafik Fungsi Kuadrat") plt.xlabel("x") plt.ylabel("f(x)") plt.grid(True) plt.legend() plt.show()

2. Alur Komputasi

Pertama, fungsi didefinisikan menggunakan def agar dapat menerima input berupa array NumPy.

Selanjutnya, np.linspace digunakan untuk membentuk domain diskrit sebanyak 100 titik sebagai pendekatan dari domain kontinu.

Proses y = f(x) merupakan bentuk vectorized computation, yaitu operasi dilakukan langsung pada seluruh elemen array tanpa perulangan eksplisit.

Titik minimum ditentukan secara analitik untuk memberikan referensi validasi terhadap hasil grafik.

3. Interpretasi Visual

Grafik yang dihasilkan menunjukkan bentuk parabola terbuka ke atas. Titik merah pada grafik merepresentasikan titik minimum fungsi.

Visualisasi ini membantu menghubungkan representasi aljabar dengan interpretasi geometris, yang merupakan inti dari komputasi matematika.
