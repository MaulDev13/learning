Penyelesaian Soal 3 (Fungsi Logaritma)

Fungsi logaritma merupakan invers dari fungsi eksponensial dan banyak digunakan dalam komputasi matematika untuk memodelkan skala non-linear, termasuk pertumbuhan tereduksi dan transformasi data.

Dalam implementasi numerik, fungsi ini hanya terdefinisi untuk nilai input positif, sehingga domain harus dibatasi secara hati-hati untuk menghindari error komputasi.

Library: numpy, matplotlib

1. Implementasi Lengkap Python

import numpy as np import matplotlib.pyplot as plt # Definisi fungsi logaritma natural def f(x): return np.log(x) # Domain harus positif (x > 0) x = np.linspace(0.1, 5, 100) # Evaluasi fungsi y = f(x) # Titik referensi x_ref = 1 y_ref = f(x_ref) print("Nilai referensi:") print("x =", x_ref) print("y =", y_ref) # Visualisasi plt.figure(figsize=(8, 5)) plt.plot(x, y, label="f(x) = ln(x)") plt.scatter(x_ref, y_ref, color="red", label="Titik Referensi") plt.title("Grafik Fungsi Logaritma Natural") plt.xlabel("x") plt.ylabel("f(x)") plt.grid(True) plt.legend() plt.show()

2. Alur Komputasi

Fungsi np.log(x) digunakan untuk menghitung logaritma natural secara elemen-per-elemen pada array NumPy.

Domain dimulai dari 0.1 karena nilai logaritma tidak terdefinisi untuk x ≤ 0 dalam bilangan real.

Titik referensi x = 1 dipilih karena ln(1) = 0, yang menjadi titik penting dalam analisis fungsi logaritma.

Evaluasi dilakukan secara vectorized sehingga seluruh nilai dihitung tanpa perulangan eksplisit.

3. Interpretasi Visual

Grafik menunjukkan bahwa fungsi logaritma meningkat secara perlahan seiring bertambahnya nilai x.

Pada nilai x mendekati nol dari sisi positif, grafik turun tajam menuju negatif tak hingga, yang menunjukkan adanya asimtot vertikal.

Titik (1, 0) menjadi referensi utama dalam memahami posisi grafik terhadap sumbu-x.
