Penyelesaian Soal 7 (Fungsi Akar)

Fungsi akar merupakan fungsi non-linear yang hanya terdefinisi untuk domain tertentu pada bilangan real. Dalam komputasi matematika, fungsi ini sering digunakan untuk memodelkan hubungan pertumbuhan sublinear dan transformasi skala.

Pada kasus ini, fungsi hanya terdefinisi untuk x ≥ 0 sehingga domain harus dibatasi untuk menghindari nilai kompleks.

Library: numpy, matplotlib

1. Implementasi Lengkap Python

import numpy as np import matplotlib.pyplot as plt # Definisi fungsi akar def f(x): return np.sqrt(x) # Domain numerik (hanya nilai non-negatif) x = np.linspace(0, 10, 100) # Evaluasi fungsi y = f(x) # Titik referensi x_ref = 4 y_ref = f(x_ref) print("Nilai referensi:") print("x =", x_ref) print("y =", y_ref) # Visualisasi plt.figure(figsize=(8, 5)) plt.plot(x, y, label="f(x) = √x") plt.scatter(x_ref, y_ref, color="red", label="Titik Referensi") plt.title("Grafik Fungsi Akar") plt.xlabel("x") plt.ylabel("f(x)") plt.grid(True) plt.legend() plt.show()

2. Alur Komputasi

Fungsi np.sqrt(x) digunakan untuk menghitung akar kuadrat secara vektorisasi pada seluruh elemen array NumPy.

Domain dibatasi dari 0 hingga 10 karena fungsi akar tidak terdefinisi untuk nilai negatif dalam bilangan real.

Evaluasi dilakukan secara vectorized computation sehingga tidak diperlukan iterasi manual.

Titik referensi dipilih pada x = 4 untuk menunjukkan hubungan dasar bahwa √4 = 2.

3. Interpretasi Visual

Grafik menunjukkan pertumbuhan fungsi yang semakin lambat seiring meningkatnya nilai x.

Kurva berbentuk cekung ke bawah, menunjukkan sifat sublinear dari fungsi akar.

Visualisasi ini sering digunakan dalam analisis skala dan transformasi data dalam komputasi numerik.
