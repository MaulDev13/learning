Penyelesaian Soal 2 (Fungsi Eksponensial)

Fungsi eksponensial memiliki karakteristik pertumbuhan yang sangat penting dalam komputasi matematika, terutama pada model pertumbuhan, peluruhan, dan proses dinamis.

Dalam pendekatan komputasi, fungsi ini dievaluasi pada domain numerik diskrit kemudian divisualisasikan untuk memahami laju pertumbuhannya secara grafis.

Library: numpy, matplotlib

1. Implementasi Lengkap Python

import numpy as np import matplotlib.pyplot as plt # Definisi fungsi eksponensial def f(x): return np.exp(x) # Domain numerik x = np.linspace(-2, 3, 100) # Evaluasi fungsi y = f(x) # Titik referensi (x = 0) x_ref = 0 y_ref = f(x_ref) print("Nilai referensi:") print("x =", x_ref) print("y =", y_ref) # Visualisasi plt.figure(figsize=(8, 5)) plt.plot(x, y, label="f(x) = e^x") plt.scatter(x_ref, y_ref, color="red", label="Titik Referensi") plt.title("Grafik Fungsi Eksponensial") plt.xlabel("x") plt.ylabel("f(x)") plt.grid(True) plt.legend() plt.show()

2. Alur Komputasi

Fungsi np.exp(x) digunakan untuk menghitung nilai eksponensial secara vektorisasi pada seluruh elemen array.

Domain dibentuk menggunakan np.linspace untuk merepresentasikan interval kontinu dalam bentuk diskrit.

Titik referensi x = 0 dipilih karena e^0 = 1, yang menjadi dasar penting dalam analisis fungsi eksponensial.

Proses evaluasi dilakukan tanpa loop eksplisit, sehingga lebih efisien secara komputasi.

3. Interpretasi Visual

Grafik menunjukkan pertumbuhan eksponensial yang semakin cepat seiring meningkatnya nilai x.

Pada sisi negatif, fungsi mendekati nol namun tidak pernah menyentuh nol, yang menunjukkan sifat asimtotik dari fungsi eksponensial.

Titik referensi membantu mengkalibrasi pemahaman bahwa fungsi melewati titik (0, 1).
