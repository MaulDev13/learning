Sistem Persamaan Linear (SPL) dalam Komputasi Matematika

Pengantar

Sistem Persamaan Linear (SPL) merupakan salah satu model utama dalam komputasi matematika yang digunakan untuk merepresentasikan hubungan linear antar variabel.

Dalam implementasi Python, SPL dapat diselesaikan menggunakan metode numerik seperti eliminasi Gauss, dekomposisi LU, serta metode iteratif seperti Jacobi dan Gauss-Seidel menggunakan NumPy.

Bentuk Umum SPL

a11x1 + a12x2 + ... + a1nxn = b1 a21x1 + a22x2 + ... + a2nxn = b2 ... an1x1 + an2x2 + ... + annxn = bn

Dalam bentuk matriks:

A x = b

Metode Penyelesaian

Metode Jenis
Eliminasi Gauss Metode langsung
Dekomposisi LU Metode langsung
Jacobi Metode iteratif
Gauss-Seidel Metode iteratif

Implementasi Dasar Python

import numpy as np A = np.array([ [3, 1, 1], [1, 4, -2], [2, -1, 3] ]) b = np.array([9, 2, 7]) x = np.linalg.solve(A, b) print(x)

Fungsi np.linalg.solve digunakan untuk menyelesaikan SPL secara langsung dengan metode dekomposisi internal NumPy.
