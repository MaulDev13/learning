Masalah Awal: Bagaimana Komputer Mengambil Keputusan?

Jika diberikan sebuah bilangan n, bagaimana komputer dapat menentukan apakah bilangan tersebut genap atau ganjil secara otomatis?

Masalah sederhana ini sebenarnya mewakili inti dari seluruh sistem komputasi: bagaimana sebuah mesin mengikuti langkah logis untuk menghasilkan keputusan.

Untuk menyelesaikan masalah ini, kita perlu memahami konsep dasar yang disebut algoritma.

Eksplorasi Awal

Sebelum mendefinisikan algoritma, perhatikan bagaimana manusia menyelesaikan masalah ini secara manual.

Kita biasanya membagi bilangan dengan 2 dan melihat apakah sisanya nol.

Pertanyaannya: bagaimana proses ini diterjemahkan menjadi langkah yang dapat dilakukan komputer?

Konsep: Algoritma

Algoritma adalah urutan langkah logis dan terstruktur untuk menyelesaikan suatu masalah.

Dalam komputasi, algoritma berfungsi sebagai jembatan antara masalah matematika dan eksekusi komputer.

def check(n): if n % 2 == 0: return "Genap" else: return "Ganjil"

Bagaimana Komputer Berpikir?

Untuk menyelesaikan satu masalah sederhana, komputer tidak langsung "mengerti". Ia memecahnya menjadi langkah kecil.

Struktur Dasar Berpikir Komputer

1. Sekuensial: langkah berurutan

2. Seleksi: pengambilan keputusan

3. Iterasi: pengulangan proses

Aplikasi: Menjumlahkan Angka

Misalkan kita ingin menjumlahkan angka dari 1 sampai n.

total = 0 for i in range(1, n+1): total += i

Proses ini menunjukkan bagaimana iterasi digunakan untuk membangun solusi.

Koneksi dengan Matematika

Banyak algoritma berasal dari struktur matematika.

n! = n × (n-1) × ... × 1

Konsep ini dapat diterjemahkan menjadi proses iteratif dalam komputer.
