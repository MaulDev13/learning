Bab 1: Flowchart

Flowchart merupakan representasi grafis dari suatu algoritma yang digunakan untuk menggambarkan alur proses secara visual.

Dalam komputasi matematika, flowchart membantu dalam memahami struktur logika sebelum diimplementasikan ke dalam bahasa pemrograman seperti Python.

Definisi Flowchart

Flowchart adalah diagram yang menggunakan simbol-simbol standar untuk merepresentasikan langkah-langkah dalam suatu algoritma.

Setiap simbol dalam flowchart memiliki makna tertentu, seperti proses, keputusan, input/output, dan aliran kontrol.

Flowchart digunakan untuk mempermudah analisis, perancangan, dan dokumentasi sistem.

Tujuan Penggunaan Flowchart

Flowchart memiliki beberapa tujuan utama dalam proses pengembangan algoritma dan program.

1. Memvisualisasikan alur logika secara jelas dan sistematis.

2. Mempermudah komunikasi antar pengembang atau pembelajar.

3. Mengidentifikasi kesalahan logika sebelum implementasi.

4. Menjadi dokumentasi dari suatu proses komputasi.

Simbol-Simbol Dasar Flowchart

Flowchart menggunakan simbol standar untuk merepresentasikan jenis operasi yang berbeda.

1. Terminator (Mulai/Selesai): berbentuk oval.

2. Proses: berbentuk persegi panjang.

3. Input/Output: berbentuk jajar genjang.

4. Decision (Keputusan): berbentuk belah ketupat.

5. Flowline: panah yang menunjukkan arah aliran proses.

Aturan Penyusunan Flowchart

Penyusunan flowchart harus mengikuti aturan tertentu agar mudah dipahami dan tidak menimbulkan ambiguitas.

1. Flowchart dimulai dari simbol "Mulai" dan diakhiri dengan "Selesai".

2. Aliran proses ditunjukkan dengan panah yang jelas.

3. Setiap simbol hanya memiliki satu fungsi spesifik.

4. Hindari garis yang saling berpotongan untuk menjaga keterbacaan.

Contoh Flowchart Sekuensial

Masalah: Menghitung luas persegi panjang.

Langkah algoritmik:

1. Mulai

2. Input panjang (p)

3. Input lebar (l)

4. Hitung luas = p × l

5. Tampilkan luas

6. Selesai

Representasi flowchart mengikuti urutan langkah tersebut dengan simbol input/output dan proses.

Contoh Flowchart Seleksi

Masalah: Menentukan bilangan genap atau ganjil.

Langkah-langkah:

1. Mulai

2. Input n

3. Periksa apakah n mod 2 = 0

4. Jika benar → tampilkan "Genap"

5. Jika salah → tampilkan "Ganjil"

6. Selesai

Pada flowchart, langkah keputusan direpresentasikan dengan simbol belah ketupat.

Contoh Flowchart Perulangan

Masalah: Menjumlahkan bilangan dari 1 sampai n.

Langkah-langkah:

1. Mulai

2. Input n

3. Inisialisasi jumlah = 0 dan i = 1

4. Jika i ≤ n maka:

5. jumlah = jumlah + i

6. i = i + 1

7. Kembali ke langkah 4

8. Tampilkan jumlah

9. Selesai

Flowchart dalam Komputasi Matematika

Flowchart sangat berguna dalam merancang algoritma matematis sebelum diterjemahkan ke dalam kode program.

Dengan flowchart, struktur logika seperti iterasi dan seleksi dapat divisualisasikan dengan jelas.

Contoh Step-by-Step: Faktorial

Masalah: Menghitung faktorial dari n.

Langkah detail:

1. Mulai

2. Input n

3. Set faktorial = 1

4. Set i = 1

5. Periksa apakah i ≤ n

6. Jika ya:

7. faktorial = faktorial × i

8. i = i + 1

9. Kembali ke langkah 5

10. Jika tidak:

11. Tampilkan faktorial

12. Selesai

Hubungan Flowchart dengan Python

Flowchart berfungsi sebagai tahap perancangan sebelum implementasi dalam Python.

Setiap simbol dalam flowchart dapat diterjemahkan menjadi instruksi Python.

Misalnya, simbol keputusan akan direpresentasikan sebagai struktur if-else dalam Python.

Tujuan Pembelajaran

Setelah mempelajari bagian ini, pembaca diharapkan:

1. Memahami konsep flowchart.

2. Mampu membaca dan membuat flowchart.

3. Menghubungkan flowchart dengan implementasi program.

Kesimpulan

Flowchart merupakan alat bantu visual yang sangat penting dalam memahami dan merancang algoritma.

Dengan menggunakan flowchart, proses komputasi dapat digambarkan secara sistematis dan mudah dipahami.

Penggunaan flowchart membantu mengurangi kesalahan logika sebelum implementasi dalam bahasa pemrograman.
