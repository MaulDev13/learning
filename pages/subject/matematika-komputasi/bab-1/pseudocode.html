Bab 1: Pseudocode

Pseudocode merupakan representasi algoritma dalam bentuk deskripsi terstruktur yang menyerupai bahasa pemrograman, namun tidak terikat pada sintaks bahasa tertentu.

Dalam komputasi matematika, pseudocode digunakan sebagai jembatan antara perancangan algoritma dan implementasi dalam bahasa pemrograman seperti Python.

Definisi Pseudocode

Pseudocode adalah cara penulisan algoritma menggunakan bahasa yang menyerupai kode program, tetapi tetap mudah dipahami oleh manusia.

Pseudocode tidak memiliki aturan sintaks yang kaku, sehingga dapat disesuaikan dengan kebutuhan penulisan algoritma.

Tujuan utama pseudocode adalah untuk menggambarkan logika penyelesaian masalah secara jelas dan sistematis.

Karakteristik Pseudocode

Pseudocode memiliki beberapa karakteristik penting.

1. Bersifat fleksibel dan tidak terikat sintaks tertentu.

2. Menggunakan bahasa yang mudah dipahami.

3. Menekankan pada logika, bukan detail implementasi.

4. Mudah diterjemahkan ke dalam bahasa pemrograman.

Struktur Dasar Pseudocode

Struktur pseudocode mengikuti struktur dasar algoritma, yaitu sekuensial, seleksi, dan perulangan.

Beberapa kata kunci yang umum digunakan antara lain:

- START / BEGIN

- END / STOP

- INPUT / READ

- OUTPUT / PRINT

- IF, THEN, ELSE

- FOR, WHILE

Contoh Pseudocode Sekuensial

Masalah: Menghitung luas persegi panjang.

Pseudocode:

START

INPUT p

INPUT l

luas ← p × l

OUTPUT luas

END

Contoh Pseudocode Seleksi

Masalah: Menentukan bilangan genap atau ganjil.

Pseudocode:

START

INPUT n

IF n mod 2 = 0 THEN

OUTPUT "Genap"

ELSE

OUTPUT "Ganjil"

ENDIF

END

Contoh Pseudocode Perulangan

Masalah: Menjumlahkan bilangan dari 1 sampai n.

Pseudocode:

START

INPUT n

jumlah ← 0

FOR i ← 1 TO n DO

jumlah ← jumlah + i

ENDFOR

OUTPUT jumlah

END

Contoh Pseudocode Matematis (Faktorial)

Masalah: Menghitung faktorial n.

Pseudocode:

START

INPUT n

faktorial ← 1

FOR i ← 1 TO n DO

faktorial ← faktorial × i

ENDFOR

OUTPUT faktorial

END

Analisis Step-by-Step

Diberikan n = 4, hitung faktorial menggunakan pseudocode di atas.

Langkah 1: Inisialisasi

faktorial = 1

Langkah 2: Iterasi i = 1

faktorial = 1 × 1 = 1

Langkah 3: Iterasi i = 2

faktorial = 1 × 2 = 2

Langkah 4: Iterasi i = 3

faktorial = 2 × 3 = 6

Langkah 5: Iterasi i = 4

faktorial = 6 × 4 = 24

Langkah 6: Output

Hasil = 24

Hubungan Pseudocode dengan Python

Pseudocode dapat dengan mudah diterjemahkan ke dalam Python.

Contoh implementasi faktorial:

n = 4

faktorial = 1

for i in range(1, n+1):

faktorial = faktorial * i

print(faktorial)

Kelebihan Pseudocode

Pseudocode memiliki beberapa keunggulan dibandingkan representasi algoritma lainnya.

1. Mudah dipahami oleh manusia.

2. Tidak bergantung pada bahasa pemrograman tertentu.

3. Mempermudah proses perancangan algoritma.

Keterbatasan Pseudocode

Meskipun memiliki banyak kelebihan, pseudocode juga memiliki beberapa keterbatasan.

1. Tidak dapat langsung dieksekusi oleh komputer.

2. Tidak memiliki standar baku yang universal.

Tujuan Pembelajaran

Setelah mempelajari bagian ini, pembaca diharapkan:

1. Memahami konsep pseudocode.

2. Mampu menulis pseudocode secara sistematis.

3. Mengubah pseudocode menjadi program Python.

Kesimpulan

Pseudocode merupakan alat penting dalam perancangan algoritma yang menjembatani pemikiran logis dengan implementasi program.

Dengan pseudocode, proses penyusunan solusi menjadi lebih terstruktur dan mudah diterjemahkan ke dalam bahasa pemrograman.
