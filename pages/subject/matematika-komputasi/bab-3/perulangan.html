Perulangan dalam Python

Definisi Konsep

Perulangan dalam pemrograman merupakan struktur kontrol yang memungkinkan eksekusi suatu blok kode secara berulang selama kondisi tertentu terpenuhi. Dalam bahasa Python, perulangan digunakan untuk mengotomatisasi proses iteratif yang sering muncul dalam komputasi matematika.

Konsep perulangan sangat penting dalam implementasi algoritma numerik, terutama pada proses yang membutuhkan iterasi berulang hingga mencapai kondisi tertentu, seperti konvergensi atau batas iterasi.

Jenis Perulangan

Python menyediakan dua jenis utama struktur perulangan, yaitu for dan while. Selain itu, terdapat kontrol tambahan seperti break, continue, dan else yang dapat digunakan untuk mengatur alur perulangan.

Struktur Fungsi
for Melakukan iterasi berdasarkan urutan (sequence)
while Melakukan iterasi selama kondisi bernilai benar
break Menghentikan perulangan secara langsung
continue Melewati iterasi saat ini dan melanjutkan ke iterasi berikutnya
else Dijalankan setelah perulangan selesai tanpa interupsi

Kesimpulan

Perulangan merupakan komponen fundamental dalam pemrograman karena memungkinkan efisiensi dalam pengolahan data dan implementasi algoritma berulang. Pemahaman terhadap berbagai jenis perulangan akan mempermudah penyelesaian permasalahan komputasi matematika.

Perulangan for

Definisi

Struktur for digunakan untuk melakukan iterasi terhadap elemen dalam suatu urutan, seperti list, tuple, atau hasil dari fungsi range().

Penjelasan

Perulangan for akan mengambil setiap elemen dari urutan secara berurutan, kemudian mengeksekusi blok kode untuk setiap elemen tersebut.

Contoh Kode

for i in range(1, 6): print(i)

Penjelasan Contoh

Fungsi range(1, 6) menghasilkan bilangan dari 1 hingga 5. Setiap nilai akan dicetak secara berurutan melalui perulangan.

Perulangan while

Definisi

Struktur while digunakan untuk menjalankan blok kode selama suatu kondisi bernilai True.

Penjelasan

Perulangan ini akan terus berjalan selama kondisi terpenuhi. Oleh karena itu, perlu diperhatikan agar kondisi berhenti didefinisikan dengan benar untuk menghindari infinite loop.

Contoh Kode

i = 1 while i <= 5: print(i) i += 1

Penjelasan Contoh

Variabel i bertambah setiap iterasi. Ketika nilai melebihi 5, kondisi menjadi salah dan perulangan berhenti.

Kontrol break

Definisi

Pernyataan break digunakan untuk menghentikan perulangan sebelum kondisi normal terpenuhi.

Contoh Kode

for i in range(1, 10): if i == 5: break print(i)

Penjelasan

Ketika nilai i mencapai 5, perulangan dihentikan secara langsung.

Kontrol continue

Definisi

Pernyataan continue digunakan untuk melewati iterasi tertentu dan melanjutkan ke iterasi berikutnya.

Contoh Kode

for i in range(1, 6): if i == 3: continue print(i)

Penjelasan

Ketika i bernilai 3, perintah print dilewati dan program melanjutkan ke iterasi berikutnya.

Perulangan dengan else

Definisi

Pada Python, perulangan dapat memiliki blok else yang akan dijalankan jika perulangan selesai tanpa interupsi dari break.

Contoh Kode

for i in range(1, 4): print(i) else: print("Perulangan selesai")

Penjelasan

Setelah semua iterasi selesai secara normal, blok else dijalankan.
