Percabangan dalam Python

Definisi Konsep

Percabangan dalam pemrograman merupakan struktur kontrol yang memungkinkan program mengambil keputusan berdasarkan kondisi tertentu. Dalam bahasa Python, percabangan digunakan untuk menentukan alur eksekusi program sehingga dapat menyesuaikan dengan nilai atau kondisi yang diberikan.

Struktur percabangan termasuk dalam konsep control flow, yang mengatur urutan eksekusi instruksi dalam suatu program. Dengan percabangan, program tidak selalu berjalan secara linear, tetapi dapat memilih jalur yang berbeda.

Penjelasan

Percabangan bekerja dengan mengevaluasi suatu kondisi logika yang menghasilkan nilai True atau False. Jika kondisi bernilai benar, maka blok kode tertentu akan dijalankan. Sebaliknya, jika kondisi bernilai salah, program dapat menjalankan blok alternatif atau melewati instruksi tersebut.

Python menyediakan beberapa bentuk percabangan, yaitu if, if-else, if-elif-else, dan match-case.

Tabel Klasifikasi Percabangan

Struktur Fungsi
if Menjalankan blok kode jika kondisi benar
if-else Menentukan dua kemungkinan aksi
if-elif-else Menangani lebih dari dua kondisi
match-case Memilih aksi berdasarkan pola atau nilai tertentu

Kesimpulan

Percabangan merupakan komponen penting dalam pemrograman karena memungkinkan program bersifat dinamis dan adaptif terhadap berbagai kondisi. Pemahaman terhadap berbagai bentuk percabangan akan sangat membantu dalam implementasi algoritma komputasi matematika.

Percabangan if

Definisi

Struktur if digunakan untuk menjalankan suatu blok kode hanya jika kondisi tertentu bernilai benar.

Penjelasan

Jika kondisi pada if bernilai True, maka blok kode di dalamnya akan dieksekusi. Jika tidak, program akan melanjutkan ke baris berikutnya tanpa menjalankan blok tersebut.

Contoh Kode

nilai = 80 if nilai >= 75: print("Lulus")

Penjelasan Contoh

Pada contoh tersebut, kondisi nilai >= 75 dievaluasi. Karena bernilai benar, maka perintah print dijalankan.

Percabangan if-else

Definisi

Struktur if-else digunakan untuk menyediakan dua jalur eksekusi berdasarkan kondisi.

Penjelasan

Jika kondisi pada if bernilai benar, maka blok pertama dijalankan. Jika salah, maka blok else akan dijalankan.

Contoh Kode

nilai = 60 if nilai >= 75: print("Lulus") else: print("Tidak Lulus")

Penjelasan Contoh

Karena kondisi bernilai salah, maka program menjalankan blok else dan menampilkan "Tidak Lulus".

Percabangan if-elif-else

Definisi

Struktur if-elif-else digunakan untuk mengevaluasi beberapa kondisi secara berurutan.

Penjelasan

Program akan memeriksa kondisi satu per satu. Jika suatu kondisi bernilai benar, maka blok tersebut dijalankan dan pemeriksaan berhenti.

Contoh Kode

nilai = 85 if nilai >= 90: print("A") elif nilai >= 75: print("B") else: print("C")

Penjelasan Contoh

Nilai 85 memenuhi kondisi kedua, sehingga program mencetak "B" dan tidak memeriksa kondisi berikutnya.

Percabangan match-case

Definisi

Struktur match-case digunakan untuk mencocokkan nilai dengan beberapa pola yang telah ditentukan.

Penjelasan

Struktur ini mirip dengan switch-case pada bahasa lain. Program akan memilih blok case yang sesuai dengan nilai yang diuji.

Contoh Kode

hari = 3 match hari: case 1: print("Senin") case 2: print("Selasa") case 3: print("Rabu") case _: print("Tidak diketahui")

Penjelasan Contoh

Nilai 3 cocok dengan case 3, sehingga program menampilkan "Rabu".

Catatan

Penggunaan match-case tersedia mulai Python versi 3.10 dan digunakan untuk meningkatkan keterbacaan dalam percabangan kompleks.
