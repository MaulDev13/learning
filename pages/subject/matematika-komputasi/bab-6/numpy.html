NumPy untuk Komputasi Numerik

NumPy (Numerical Python) merupakan library fundamental pada Python yang digunakan untuk komputasi numerik dan operasi matematika ilmiah.

NumPy menyediakan struktur data array multidimensi, operasi vektor dan matriks, fungsi matematika, serta optimisasi performa komputasi numerik.

Dalam metode numerik, NumPy menjadi library utama karena sebagian besar algoritma melibatkan manipulasi array, matriks, iterasi numerik, dan operasi linear algebra.

Mengapa Menggunakan NumPy?

Python bawaan sebenarnya sudah memiliki struktur data seperti list, namun list tidak dirancang untuk komputasi numerik berskala besar.

Aspek Python List NumPy Array
Kecepatan Lebih lambat Lebih cepat
Operasi Matematis Terbatas Lengkap
Operasi Matriks Tidak efisien Sangat efisien
Memori Lebih besar Lebih hemat
Komputasi Vektor Tidak optimal Optimal

Oleh karena itu, hampir seluruh komputasi numerik modern pada Python menggunakan NumPy.

Instalasi NumPy

NumPy dapat dipasang menggunakan pip.

pip install numpy

Setelah instalasi selesai, library dapat digunakan menggunakan perintah import.

import numpy as np

Penulisan np merupakan alias standar yang umum digunakan dalam Python scientific computing.

Konsep Array pada NumPy

Struktur data utama pada NumPy adalah ndarray atau multidimensional array.

Array NumPy digunakan untuk menyimpan data numerik secara efisien dalam bentuk vektor maupun matriks.

import numpy as np A = np.array([1, 2, 3, 4]) print(A)

Berbeda dengan list biasa, array NumPy mendukung operasi matematis langsung terhadap seluruh elemen.

Array Satu Dimensi

Array satu dimensi digunakan untuk merepresentasikan vektor.

$$ \mathbf{x} = \begin{bmatrix} x_1 & x_2 & x_3 \end{bmatrix} $$
import numpy as np x = np.array([10, 20, 30]) print(x)

Setiap elemen array dapat diakses menggunakan indeks.

print(x[0]) print(x[1])

Array Dua Dimensi

Array dua dimensi digunakan untuk merepresentasikan matriks.

$$ A = \begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix} $$
import numpy as np A = np.array([ [1, 2], [3, 4] ]) print(A)

Matriks sangat penting dalam metode numerik, terutama pada penyelesaian sistem persamaan linear.

Informasi Dimensi Array

NumPy menyediakan atribut untuk mengetahui ukuran dan dimensi array.

import numpy as np A = np.array([ [1, 2], [3, 4] ]) print(A.shape) print(A.ndim) print(A.size)
Atribut Fungsi
shape Ukuran array
ndim Jumlah dimensi
size Total elemen

Operasi Dasar Array

NumPy mendukung operasi matematis langsung pada array.

import numpy as np A = np.array([1, 2, 3]) B = np.array([4, 5, 6]) print(A + B) print(A - B) print(A * B)

Operasi dilakukan secara elemen per elemen.

$$ A + B = \begin{bmatrix} a_1+b_1 & a_2+b_2 & a_3+b_3 \end{bmatrix} $$

Operasi Matriks

NumPy menyediakan operasi matriks untuk kebutuhan linear algebra.

import numpy as np A = np.array([ [1, 2], [3, 4] ]) B = np.array([ [5, 6], [7, 8] ]) C = np.dot(A, B) print(C)

Fungsi np.dot() digunakan untuk perkalian matriks.

$$ C = AB $$

Membuat Array Otomatis

NumPy menyediakan berbagai fungsi untuk membuat array secara otomatis.

import numpy as np print(np.zeros((2, 2))) print(np.ones((3, 3))) print(np.arange(0, 10, 2)) print(np.linspace(0, 1, 5))
Fungsi Kegunaan
zeros() Membuat array nol
ones() Membuat array satu
arange() Deret dengan interval tertentu
linspace() Deret dengan jumlah titik tertentu

NumPy pada Visualisasi Fungsi

NumPy sering digunakan bersama Matplotlib untuk membuat grafik fungsi matematika.

import numpy as np import matplotlib.pyplot as plt x = np.linspace(-5, 5, 100) y = x**2 - 4 plt.plot(x, y) plt.grid() plt.show()

Fungsi linspace() sangat berguna untuk menghasilkan titik domain grafik secara otomatis.

Operasi Statistik Dasar

NumPy juga menyediakan fungsi statistik sederhana.

import numpy as np data = np.array([1, 2, 3, 4, 5]) print(np.mean(data)) print(np.max(data)) print(np.min(data)) print(np.sum(data))
Fungsi Kegunaan
mean() Rata-rata
max() Nilai maksimum
min() Nilai minimum
sum() Jumlah seluruh elemen

Linear Algebra pada NumPy

Modul numpy.linalg menyediakan berbagai operasi linear algebra.

import numpy as np A = np.array([ [1, 2], [3, 4] ]) det = np.linalg.det(A) inv = np.linalg.inv(A) print(det) print(inv)

Fungsi tersebut sangat penting dalam implementasi metode numerik berbasis matriks.

Keunggulan NumPy dalam Metode Numerik

NumPy memiliki berbagai keunggulan untuk komputasi ilmiah dan numerik.

  • Operasi numerik sangat cepat
  • Efisien dalam penggunaan memori
  • Mendukung operasi matriks kompleks
  • Terintegrasi dengan library ilmiah lainnya

Oleh karena itu, NumPy menjadi fondasi utama dalam scientific computing menggunakan Python.
