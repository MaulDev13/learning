Metode Iteratif Gauss-Seidel

Metode Gauss-Seidel adalah metode iteratif untuk menyelesaikan Sistem Persamaan Linear (SPL) dengan memperbarui nilai variabel secara bertahap hingga mencapai konvergensi.

Metode ini merupakan pengembangan dari Jacobi, dengan perbedaan utama pada penggunaan nilai terbaru secara langsung dalam setiap iterasi.

Bentuk umum SPL: Ax = b

Ide Dasar Gauss-Seidel

Pada Gauss-Seidel, setiap nilai variabel yang sudah dihitung langsung digunakan untuk menghitung variabel berikutnya dalam iterasi yang sama.

Ini membuat proses konvergensi biasanya lebih cepat dibanding Jacobi.

Rumus Iterasi

xᵢ^(k+1) = (1 / aᵢᵢ) [ bᵢ − Σ (aᵢⱼ xⱼ) ]

Perbedaan penting:

• Jacobi → semua x menggunakan iterasi lama (k)
• Gauss-Seidel → sebagian sudah menggunakan nilai baru (k+1)

Langkah Penyelesaian

1. Tentukan tebakan awal
2. Hitung x₁, x₂, ..., xₙ secara berurutan
3. Gunakan nilai terbaru langsung saat tersedia
4. Ulangi hingga konvergen

Contoh Sistem Persamaan

8x₁ + x₂ − x₃ = 8
2x₁ + x₂ + 9x₃ = 12
x₁ − 7x₂ + 2x₃ = -4

Tebakan awal: x₁⁰ = 0, x₂⁰ = 0, x₃⁰ = 0

Iterasi 1 (Gauss-Seidel)

x₁¹ = (8 - x₂⁰ + x₃⁰) / 8 = 1

x₂¹ = (12 - 2x₁¹ - 9x₃⁰) / 1 = 10

x₃¹ = (-4 - x₁¹ + 7x₂¹) / 2 = 32.5

Perbedaan dengan Jacobi

• Jacobi: semua variabel dihitung dari nilai lama
• Gauss-Seidel: langsung memakai nilai terbaru

Dampaknya:

✔ Lebih cepat konvergen
✔ Lebih efisien secara iterasi
✔ Lebih stabil pada banyak kasus

Konvergensi

Metode Gauss-Seidel akan konvergen jika:

• Matriks memiliki diagonal dominan
• Sistem cukup stabil secara numerik

Jika tidak terpenuhi, metode bisa lambat atau divergen.

Kesimpulan

Gauss-Seidel adalah metode iteratif yang lebih efisien dibanding Jacobi karena menggunakan nilai terbaru dalam setiap langkah perhitungan.

Metode ini cocok untuk sistem besar yang membutuhkan pendekatan numerik cepat.
