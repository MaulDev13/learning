Operasi Matriks & Operasi Baris Elementer

Operasi matriks adalah aturan dasar untuk mengolah matriks menggunakan penjumlahan, pengurangan, dan perkalian skalar. Sedangkan operasi baris elementer digunakan untuk mengubah bentuk matriks dalam menyelesaikan sistem persamaan linear.

1. Penjumlahan & Pengurangan Matriks

Dua matriks hanya bisa dijumlahkan atau dikurangkan jika ukurannya sama. Operasi dilakukan elemen yang seposisi. 

A + B =
[ a11 a12 ] + [ b11 b12 ] = [ a11+b11  a12+b12 ]
[ a21 a22 ]   [ b21 b22 ]   [ a21+b21  a22+b22 ]

Contoh:
[1 2] + [3 4] = [4 6]

✔ Pengurangan juga dilakukan dengan cara yang sama:
[1 2] - [3 4] = [-2 -2]

2. Perkalian Skalar

Setiap elemen matriks dikalikan dengan sebuah angka. 

k · A =
k [ a b ] = [ ka kb ]
  [ c d ]   [ kc kd ]

Contoh:
2 × [1 2] = [2 4]

3. Perkalian Matriks

Perkalian matriks dilakukan dengan aturan baris × kolom. 

A × B =
[ a b ]   [ e f ]   [ a·e + b·g   a·f + b·h ]
[ c d ] × [ g h ] = [ c·e + d·g   c·f + d·h ]

Contoh Matriks 3×3

A × B =
[ a b c ]   [ j k l ]
[ d e f ] × [ m n o ]
[ g h i ]   [ p q r ]

=
[ a·j + b·m + c·p   a·k + b·n + c·q   a·l + b·o + c·r ]
[ d·j + e·m + f·p   d·k + e·n + f·q   d·l + e·o + f·r ]
[ g·j + h·m + i·p   g·k + h·n + i·q   g·l + h·o + i·r ]

✔ Pola tetap sama:
baris × kolom

4. Operasi Baris Elementer

Operasi baris elementer adalah cara untuk mengubah bentuk matriks tanpa mengubah solusi dari sistem persamaan yang diwakilinya.

Intinya: walaupun angka di dalam matriks berubah, hasil akhirnya (solusi sistem) tetap sama.

🔄 Apa itu Row Equivalence?

Dua matriks disebut row equivalent jika satu bisa diubah menjadi yang lain menggunakan operasi baris elementer.

  • Menukar baris ✔
  • Mengalikan baris ✔
  • Menjumlahkan / mengurangkan baris ✔

✔ Solusi tetap sama

🧠 Kenapa solusi tidak berubah?

x + y = 5
2x + 3y = 8

R2 = R2 - 2R1

(2x + 3y) - 2(x + y)
= y
                    
x + y = 5
y = 3

💡 Intuisi

  • Kalimat dipersingkat
  • Isi tetap sama
  • Bentuk berubah

⚖️ Balancer

row equivalence

🔥 Kesimpulan

  • Matriks boleh berubah ✔
  • Angka boleh berubah ✔
  • Solusi tetap ✔

4. Operasi Baris Elementer (Detail)

Operasi ini mengubah baris dengan aturan tertentu.

1. Menukar Baris

[1 2]
[3 4]

→ tukar

[3 4]
[1 2]

2. Mengalikan Baris

[1 2] → [2 4]

3. Mengurangi Baris

k = 3

R2 = R2 − 3R1
= (0 -2)

💡 Intuisi

  • Target: nolkan angka
  • k = bawah ÷ atas
  • Ri = Ri − kRj

🔥 Ringkasan

k = bawah ÷ atas
Ri = Ri − kRj
