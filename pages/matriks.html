Operasi Matriks & Operasi Baris Elementer

Dasar penting untuk Eliminasi Gauss & LU Decomposition

Operasi matriks adalah aturan dasar untuk mengolah matriks menggunakan penjumlahan, pengurangan, dan perkalian skalar. Sedangkan operasi baris elementer digunakan untuk mengubah bentuk matriks dalam menyelesaikan sistem persamaan linear.

1. Penjumlahan & Pengurangan Matriks

Dua matriks hanya bisa dijumlahkan atau dikurangkan jika ukurannya sama. Operasi dilakukan elemen yang seposisi. 

A + B =
[ a11 a12 ] + [ b11 b12 ] = [ a11+b11  a12+b12 ]
[ a21 a22 ]   [ b21 b22 ]   [ a21+b21  a22+b22 ]

Contoh:
[1 2] + [3 4] = [4 6]

✔ Pengurangan juga dilakukan dengan cara yang sama:
[1 2] - [3 4] = [-2 -2]

2. Perkalian Skalar

Setiap elemen matriks dikalikan dengan sebuah angka. 

k · A =
k [ a b ] = [ ka kb ]
  [ c d ]   [ kc kd ]

Contoh:
2 × [1 2] = [2 4]

3. Perkalian Matriks

Perkalian matriks dilakukan dengan aturan baris × kolom. 

A × B =
[ a b ]   [ e f ]   [ a·e + b·g   a·f + b·h ]
[ c d ] × [ g h ] = [ c·e + d·g   c·f + d·h ]

4. Operasi Baris Elementer

Digunakan dalam Eliminasi Gauss dan LU Decomposition untuk mengubah matriks menjadi bentuk segitiga atas.

Tiga operasi dasar:

  • 1. Menukar baris → Ri ↔ Rj
  • 2. Mengalikan baris → Ri = k × Ri
  • 3. Mengurangi baris → Ri = Ri − kRj

5. Penting: Peran Pembagian dalam Eliminasi

Dalam eliminasi Gauss, pembagian bukan operasi baris langsung. Pembagian hanya digunakan untuk mencari faktor pengali (m).

Rumus:
m = elemen bawah ÷ pivot

Setelah itu digunakan operasi utama:
Baris baru = Baris lama − m × Baris pivot

✔ Jadi:
• Pembagian → hanya untuk mencari m
• Perkalian → membuat kelipatan baris
• Pengurangan → menghilangkan elemen

6. Hubungan dengan LU Decomposition

- Hasil eliminasi Gauss → matriks U
- Nilai m (hasil pembagian) → disimpan di matriks L

Sehingga:
A = L × U
