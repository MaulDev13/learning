Dekomposisi LU (LU Decomposition)

Dekomposisi LU adalah metode untuk memecah matriks koefisien A menjadi dua matriks, yaitu matriks segitiga bawah (L) dan matriks segitiga atas (U).

Secara matematis: A = LU

Metode ini digunakan untuk menyelesaikan sistem persamaan linear: Ax = b dengan cara yang lebih efisien.

Bentuk Matriks

L = matriks segitiga bawah
U = matriks segitiga atas

Sehingga: A = LU

Contoh Soal

3x₁ + 4x₂ + 2x₃ = 16
x₁ + 3x₂ + x₃ = 8
2x₁ + 5x₂ + x₃ = 12

Langkah 1: Matriks A dan b

A:

[ 3  4  2 ]
[ 1  3  1 ]
[ 2  5  1 ]

b:

[ 16 ]
[ 8  ]
[ 12 ]

Langkah 2: Eliminasi (Membentuk U)

Kita lakukan eliminasi Gauss untuk mengubah A menjadi bentuk segitiga atas (U).

Eliminasi kolom pertama

Faktor eliminasi: l21 = 1/3 = 0.33
l31 = 2/3 = 0.67

Operasi: R2 = R2 - 0.33R1
R3 = R3 - 0.67R1 

[ 3   4     2 ]
[ 0   1.67  0.33 ]
[ 0   2.33 -0.33 ]

Langkah 2 (lanjutan): Eliminasi kolom kedua

Faktor: l32 = 2.33 / 1.67 ≈ 1.5

Operasi: R3 = R3 - 1.5R2 

[ 3  4     2 ]
[ 0  1.67  0.33 ]
[ 0  0    -1 ]

Ini adalah matriks U.

Langkah 3: Matriks L

L berisi semua faktor eliminasi yang digunakan: 

[ 1   0   0 ]
[ 0.33 1   0 ]
[ 0.67 1.5 1 ]

Langkah 4: Selesaikan Ly = b

Kita mulai dari atas:

y₁ = 16
0.33y₁ + y₂ = 8 → y₂ = 2.72
0.67y₁ + 1.5y₂ + y₃ = 12 → y₃ = -3

Langkah 5: Selesaikan Ux = y

-1x₃ = -3 → x₃ = 3
1.67x₂ + 0.33x₃ = 2.72 → x₂ = -1
3x₁ + 4x₂ + 2x₃ = 16 → x₁ = 2

Hasil Akhir

x₁ = 2
x₂ = -1
x₃ = 3

Kesimpulan

LU decomposition menyederhanakan sistem persamaan linear dengan memecah matriks A menjadi L dan U. Dengan cara ini, kita tidak perlu melakukan eliminasi berulang kali dan perhitungan menjadi lebih efisien.
