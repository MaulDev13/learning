Dekomposisi LU (LU Decomposition)

Dekomposisi LU adalah metode dalam Sistem Persamaan Linear yang digunakan untuk memecah sebuah matriks koefisien A menjadi hasil perkalian dua matriks, yaitu matriks segitiga bawah (L) dan matriks segitiga atas (U).

Secara matematis: A = LU

Metode ini digunakan untuk menyelesaikan sistem persamaan: Ax = b dengan cara yang lebih efisien.

Bentuk Matriks

L = matriks segitiga bawah (lower triangular)
U = matriks segitiga atas (upper triangular)

Sehingga:

A = LU

Langkah Penyelesaian LU

1. Faktorisasi A menjadi L dan U
2. Selesaikan Ly = b (substitusi maju)
3. Selesaikan Ux = y (substitusi mundur)

Contoh Soal

Diberikan sistem persamaan:

3x₁ + 4x₂ + 2x₃ = 16
x₁ + 3x₂ + x₃ = 8
2x₁ + 5x₂ + x₃ = 12

Langkah 1: Bentuk Matriks A

A =

[ 3 4 2 ]
[ 1 3 1 ]
[ 2 5 1 ]

b =

[ 16 ]
[ 8 ]
[ 12 ]

Langkah 2: Dekomposisi A = LU

L =

[ 1 0 0 ]
[ 0.33 1 0 ]
[ 0.67 1.5 1 ]

U =

[ 3 4 2 ]
[ 0 1.67 0.33 ]
[ 0 0 -1 ]

Langkah 3: Selesaikan Ly = b

y₁ = 16
0.33y₁ + y₂ = 8 → y₂ = 2.72
0.67y₁ + 1.5y₂ + y₃ = 12 → y₃ = -3

Langkah 4: Selesaikan Ux = y

-1x₃ = -3 → x₃ = 3
1.67x₂ + 0.33x₃ = 2.72 → x₂ = -1
3x₁ + 4x₂ + 2x₃ = 16 → x₁ = 2

Hasil Akhir

x₁ = 2
x₂ = -1
x₃ = 3

Kesimpulan

Dekomposisi LU mempermudah penyelesaian SPL dengan memecah matriks menjadi dua bagian yang lebih sederhana. Metode ini sangat efisien untuk sistem besar atau perhitungan berulang.
