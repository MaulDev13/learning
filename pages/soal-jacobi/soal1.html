Penyelesaian Soal 1 (Metode Jacobi)

Sistem persamaan linear ini diselesaikan menggunakan metode Jacobi, yaitu metode iteratif dalam numerik yang digunakan untuk mencari solusi pendekatan dari sistem persamaan linear Ax = b.

Metode ini bekerja dengan cara memperbarui nilai setiap variabel secara bertahap berdasarkan nilai iterasi sebelumnya, bukan langsung menyelesaikan secara eksak.

Dalam prosesnya, kita akan mengamati apakah nilai hasil iterasi menuju solusi tetap (konvergen) atau justru semakin tidak stabil (divergen).

Metode: Iterasi Jacobi

Tebakan awal: (0, 0, 0)

Sistem Persamaan

8x₁ + x₂ − x₃ = 8
2x₁ + x₂ + 9x₃ = 12
x₁ − 7x₂ + 2x₃ = -4

Sistem ini terdiri dari 3 persamaan dengan 3 variabel, sehingga dapat diselesaikan menggunakan metode iteratif.

Bentuk Iterasi Jacobi

Untuk menerapkan metode Jacobi, setiap persamaan diubah menjadi bentuk penyelesaian variabel tunggal:

x₁ = (8 − x₂ + x₃) / 8
x₂ = 12 − 2x₁ − 9x₃
x₃ = (−4 − x₁ + 7x₂) / 2

Setiap nilai x pada iterasi baru dihitung menggunakan nilai dari iterasi sebelumnya.

Proses Iterasi

Iterasi dimulai dari tebakan awal (0, 0, 0) dan dilanjutkan secara bertahap:

Iterasi 0: (0, 0, 0)
Iterasi 1: (1, 12, -2)
Iterasi 2: (-0.75, 28, 39.5)
Iterasi 3: (2.4375, -340.5, 97.375)

Setiap iterasi menggunakan hasil sebelumnya tanpa langsung memperbarui nilai dalam satu langkah.

Analisis Konvergensi

Dari hasil iterasi terlihat bahwa nilai semakin membesar dan tidak mendekati suatu nilai tetap.

Hal ini menunjukkan bahwa metode Jacobi tidak konvergen untuk sistem ini.

Penyebab utama adalah matriks koefisien tidak memenuhi syarat dominasi diagonal, yaitu kondisi di mana elemen diagonal harus lebih besar atau sama dengan jumlah elemen lainnya dalam baris tersebut.

Ketika syarat ini tidak terpenuhi, iterasi menjadi tidak stabil dan hasil tidak dapat dipercaya.

Kesimpulan

• Metode Jacobi tidak menghasilkan konvergensi pada soal ini
• Nilai iterasi menjadi semakin tidak stabil
• Sistem tidak memenuhi syarat dominasi diagonal
• Metode ini tidak cocok digunakan untuk sistem tersebut
