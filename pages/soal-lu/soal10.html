Penyelesaian Soal 10 (Metode LU)

Sistem ini diselesaikan menggunakan metode dekomposisi LU, dengan memfaktorkan matriks A menjadi matriks L dan U.

Metode: Dekomposisi LU

Sistem Persamaan

7x₁ + 2x₂ + x₃ = 17
2x₁ + 8x₂ + x₃ = 19
x₁ + x₂ + 9x₃ = 21

Matriks A

[ 7 2 1 ]
[ 2 8 1 ]
[ 1 1 9 ]

Dekomposisi LU

L = [ 1 0 0 ]
[ 0.286 1 0 ]
[ 0.143 0.086 1 ]

U = [ 7 2 1 ]
[ 0 7.429 0.714 ]
[ 0 0 8.786 ]

Penyelesaian Ly = b

y₁ = 17
y₂ = 19 - 0.286(17) ≈ 14.143
y₃ = 21 - 0.143(17) - 0.086(14.143) ≈ 17.35

Penyelesaian Ux = y

8.786x₃ = 17.35 → x₃ ≈ 1.975
7.429x₂ + 0.714x₃ = 14.143 → x₂ ≈ 1.716
7x₁ + 2x₂ + x₃ = 17 → x₁ ≈ 1.656

Hasil Akhir

x₁ ≈ 1.656
x₂ ≈ 1.716
x₃ ≈ 1.975

Metode LU memberikan solusi stabil dan akurat.
