Penyelesaian Soal 4 (Metode LU)

Sistem ini diselesaikan menggunakan metode dekomposisi LU, dengan memfaktorkan matriks A menjadi matriks L dan U.

Metode: Dekomposisi LU

Sistem Persamaan

12x₁ − x₂ + x₃ = 5
−x₁ + 9x₂ − x₃ = 10
x₁ − x₂ + 8x₃ = 7

Matriks A

[ 12 -1 1 ]
[ -1 9 -1 ]
[ 1 -1 8 ]

Dekomposisi LU

L = [ 1 0 0 ]
[ -0.083 1 0 ]
[ 0.083 -0.102 1 ]

U = [ 12 -1 1 ]
[ 0 8.917 -0.917 ]
[ 0 0 7.823 ]

Penyelesaian Ly = b

y₁ = 5
y₂ = 10 + 0.083(5) ≈ 10.417
y₃ = 7 - 0.083(5) + 0.102(10.417) ≈ 7.65

Penyelesaian Ux = y

7.823x₃ = 7.65 → x₃ ≈ 0.978
8.917x₂ − 0.917x₃ = 10.417 → x₂ ≈ 1.270
12x₁ − x₂ + x₃ = 5 → x₁ ≈ 0.441

Hasil Akhir

x₁ ≈ 0.441
x₂ ≈ 1.270
x₃ ≈ 0.978

Metode LU memberikan solusi stabil dan akurat.
