Penyelesaian Soal 7 (Metode LU)

Sistem ini diselesaikan menggunakan metode dekomposisi LU, dengan memfaktorkan matriks A menjadi matriks L dan U.

Metode: Dekomposisi LU

Sistem Persamaan

9x₁ + x₂ + x₃ = 20
x₁ + 8x₂ − x₃ = 18
2x₁ − x₂ + 7x₃ = 16

Matriks A

[ 9 1 1 ]
[ 1 8 -1 ]
[ 2 -1 7 ]

Dekomposisi LU

L = [ 1 0 0 ]
[ 0.111 1 0 ]
[ 0.222 -0.146 1 ]

U = [ 9 1 1 ]
[ 0 7.889 -1.111 ]
[ 0 0 6.616 ]

Penyelesaian Ly = b

y₁ = 20
y₂ = 18 - 0.111(20) ≈ 15.778
y₃ = 16 - 0.222(20) + 0.146(15.778) ≈ 13.889

Penyelesaian Ux = y

6.616x₃ = 13.889 → x₃ ≈ 2.099
7.889x₂ − 1.111x₃ = 15.778 → x₂ ≈ 2.277
9x₁ + x₂ + x₃ = 20 → x₁ ≈ 1.736

Hasil Akhir

x₁ ≈ 1.736
x₂ ≈ 2.277
x₃ ≈ 2.099

Metode LU memberikan solusi stabil dan akurat.
