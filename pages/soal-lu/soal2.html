Penyelesaian Soal 2 (Metode LU)

Sistem ini diselesaikan menggunakan metode dekomposisi LU, yaitu memfaktorkan matriks A menjadi matriks L dan U.

Metode: Dekomposisi LU

Sistem Persamaan

10x₁ − x₂ + 2x₃ = 6
−x₁ + 11x₂ − x₃ = 25
2x₁ − x₂ + 10x₃ = -11

Matriks A

[ 10 -1 2 ]
[ -1 11 -1 ]
[ 2 -1 10 ]

Dekomposisi LU

L = [ 1 0 0 ]
[ -0.1 1 0 ]
[ 0.2 -0.073 1 ]

U = [ 10 -1 2 ]
[ 0 10.9 -0.8 ]
[ 0 0 9.742 ]

Penyelesaian Ly = b

y₁ = 6
y₂ = 25 + 0.1(6) = 25.6
y₃ = -11 - 0.2(6) + 0.073(25.6) ≈ -10.33

Penyelesaian Ux = y

9.742x₃ = -10.33 → x₃ ≈ -1.060
10.9x₂ − 0.8x₃ = 25.6 → x₂ ≈ 2.281
10x₁ − x₂ + 2x₃ = 6 → x₁ ≈ 1.043

Hasil Akhir

x₁ ≈ 1.043
x₂ ≈ 2.281
x₃ ≈ -1.060

Metode LU memberikan solusi yang stabil dan akurat.
