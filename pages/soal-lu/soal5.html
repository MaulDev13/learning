Penyelesaian Soal 5 (Metode LU)

Sistem ini diselesaikan menggunakan metode dekomposisi LU, dengan memfaktorkan matriks A menjadi matriks L dan U.

Metode: Dekomposisi LU

Sistem Persamaan

5x₁ + 2x₂ + x₃ = 12
2x₁ + 6x₂ + x₃ = 15
x₁ + x₂ + 7x₃ = 20

Matriks A

[ 5 2 1 ]
[ 2 6 1 ]
[ 1 1 7 ]

Dekomposisi LU

L = [ 1 0 0 ]
[ 0.4 1 0 ]
[ 0.2 0.077 1 ]

U = [ 5 2 1 ]
[ 0 5.2 0.6 ]
[ 0 0 6.754 ]

Penyelesaian Ly = b

y₁ = 12
y₂ = 15 - 0.4(12) = 10.2
y₃ = 20 - 0.2(12) - 0.077(10.2) ≈ 16.81

Penyelesaian Ux = y

6.754x₃ = 16.81 → x₃ ≈ 2.489
5.2x₂ + 0.6x₃ = 10.2 → x₂ ≈ 1.677
5x₁ + 2x₂ + x₃ = 12 → x₁ ≈ 1.231

Hasil Akhir

x₁ ≈ 1.231
x₂ ≈ 1.677
x₃ ≈ 2.489

Metode LU memberikan solusi stabil dan akurat.
